<p>Portfolio is a customer related account allowing you to directly manage and analyse portfolios associated with the package.</p>

<p>On the left menu, select <span>Portfolio Management</span> to:</p>

<ul> <li>Manage a collection of SIMs</li> <li>Configure portfolios and accounts </li> </ul>

<p><img src="/images/resources/images/infinity/portfoliomgmt-icon.png" alt="Portfoliomgmt Icon" /> </p>

<p> <span>Portfolio Management</span> is divided into the following sections:</p>

<ul> <li><span>Your Portfolio</span>: For more information, see <a href="#YourPortfolio">Managing your master account</a></li> <li><span>User Accounts</span>: For more information, see <a href="#ManageUsers">Managing your users</a></li> </ul>

<h2 id="Managingyourmasteraccount">Managing your master account</h2>

<p><span>To view current user's portfolio:</span> </p>

<p>The <span>Your Portfolio</span> page lists the information associated with the portfolio assigned to the current user including:</p>

<ul> <li>Portfolio details</li> <li>Billing and shipping addresses</li> <li>Users assigned to the portfolio</li> </ul>

<p><img src="/images/resources/images/infinity/youraccountpage.png" alt="Youraccountpage" /> </p>

<h3>About <span>Your Portfolio</span> attributes</h3>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Portfolio name</span> </td> <td> The portfolio name associated with the package that contains the SIM. </td> </tr> <tr> <td><span>Company</span> </td> <td> The company that is legally responsible for this portfolio. </td> </tr> <tr> <td><span>Country</span> </td> <td> The country in which the company is located.  </td> </tr> <tr> <td><span>Billing Currency</span> </td> <td> A three letter code that indicates which currency to use for charges and payments to the portfolio. <br />Ensure that the currency is applicable to the country.<br /> </td> </tr> <tr> <td><span>Addresses</span> </td> <td> Displays a list of the configured addresses configured for the portfolio. Each address contains the following information:  <ul> <li><span>Type </span>– each address is used for either:<ul><li><span>Billing </span>– where to send invoices.</li><li><span>Shipping </span>– where to ship hardware, for example, SIMs.</li></ul></li> <li><span>Name </span> – the contact name at the address.</li> <li><span>Address </span> – the lines of the address, including country, region and postal codes.</li> <li><span>Email </span> – the email address of the primary contact for the address.</li> <li><span>Phone number</span> – the mobile phone number of the contact at the address.</li> </ul> </td> </tr> <tr> <td><span>Users table</span> </td> <td> The list of user accounts assigned to your portfolio. For more information, see <a href="#ManageUsers">Managing your users</a> </td> </tr> <tr> <td><span>Name</span> </td> <td> The first and last name of the user account associated with this portfolio. </td> </tr> <tr> <td><span>Status</span> </td> <td> The user account <span>Status</span>, either: <ul> <li><span>Active</span>: Users who can log in and access the portal.</li> <li><span>Inactive</span>: Users whose account is suspended because they have not interacted with the portal for a specified period of time.</li> <li><span>Verified</span>: Users who have authenticated their account via the referral code link.</li> <li><span>Unverified</span>: Users who have not yet authenticated their account via the referral code link. </li> </ul> </td> </tr> <tr> <td><span>Email</span> </td> <td> The email address of the user account. </td> </tr> </tbody> </table>

<h2 id="Managingyourusers">Managing your users</h2>

<p>Use the <span>Manage Users</span> page to create new user accounts and edit, delete, view existing user accounts. </p>

<p><img src="/images/resources/images/infinity/manageuserspage.png" alt="Manageuserspage" /> </p>

<p>To add a new user account:</p>

<ol> <li>On the left menu, select <span>Portfolio Management</span> &gt; <span>User Accounts</span>.</li> <li> On the <span>Manage Users</span> page, select <span>Actions</span> &gt; <span>Add New User</span>, which displays the <span>Add New User</span> dialog.<br /><img src="/images/resources/images/infinity/addnewuserdialog.png" alt="Addnewuserdialog" /><br />Specify the following information that is unique to the portfolio for the new user: <ul><li><span>Username</span></li><li><span>Email address</span></li><li><span>First name</span></li><li><span>Last name</span></li></ul><br />Select one or more infinity roles: <ul><li><span>Admin</span></li><li><span>Logistics</span></li><li><span>Support</span></li></ul></li> <li>Select <span>Save</span>.</li> </ol>

<Note>When you create a brand new portfolio account, the first user added to that portfolio will be automatically assigned the role <span>Portfolio Admin</span> by the Support.</Note>

<Tip>Similarly, you can create multiple portfolios.</Tip>

<p>To change the status of an user account:</p>

<ol> <li>On the left menu, select <span>Portfolios</span> &gt; <span>Manage Portfolios</span>.</li> <li>On the <span>Portfolio</span> page, select <span>Users</span> &gt; <span>Portfolio ID</span> of the relevant user. </li> </ol>

<p><img src="/images/resources/images/infinity/edituser.png" alt="Edituser" /> </p>

<ul> <li>In the <span>Edit User</span> dialog, select the infinity roles: <span>Admin</span>.</li> <li>In the <span>User Status</span> drop-down list, select <span>active</span> or <span>inactive</span>.</li> <li>Enable or disable <span>Multi-factor authentication</span>. </li> <li>Select <span>Save</span> for the changes to reflect.</li> </ul>

<Tip>You can assign the <span>Portfolio Admin</span> role to any of the added users.</Tip>

<p>To delete an user account:</p>

<ol> <li>On the left menu,, select <span>Portfolio Management</span> &gt; <span>Manage Portfolios</span>.</li> <li>In the <span>Portfolio</span> page, select <span>Users</span> &gt; <span>Portfolio ID</span> of the relevant user. </li> </ol>

<ul> <li>In the <span>Edit User</span> dialog, select the infinity roles: <span>Admin</span>.</li> <li>Select <span>Delete User</span>.</li> <li>Select <span>Save</span> for the changes to reflect.</li> </ul>
