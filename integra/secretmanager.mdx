<p>The Secret Manager plays a critical role in maintaining the security and integrity of sensitive transmitted data by:</p>

<ul> <li> Securely managing cryptographic keys and sensitive information.</li> <li>Facilitating secure authentication.</li> <li>Handling encrypted communication. </li> <li> Preventing unauthorized access and misuse of the stored secrets.  </li> </ul>

<p>To access secret manager:</p>

<ul> <li>On the left menu, select <span>Secret Manager</span>.</li> </ul>

<p> <img src="/images/resources/images/integra/secretmanager-icon.png" alt="Secretmanager Icon" /> </p>

<h3>Secrets handling</h3>

<p>You can store the following secrets:</p>

<ul> <li>A set of credentials, such as a username and password.</li> <li>An authorisation token.</li> <li>Any other secret information that requires encryption.</li> </ul>

<h2 id="Viewingsecretdetails">Viewing secret details</h2>

<p>To view details about your secret:</p>

<ol> <li>  On the left menu, select <span>Secret Manager</span> &gt; <span>Get</span> to display the <span>View a Secret page</span>. </li> <li> In the <span>Enter Secret Name</span> field, paste the copied <span>Secret Name</span> (from the above procedure). <br /><img src="/images/resources/images/integra/view-secret.png" alt="View Secret" />  <span><strong>Tip: </strong> This page automatically displays the <a href="#Portfoli">Portfolio ID</a> that you provided while creating your secret.</span> </li> <li> Select <span>Lookup</span> to display the <span>Secret String</span> of your secret. </li> </ol>

<h2 id="Creatingsecrets">Creating secrets</h2>

<p>To create a new secret:</p>

<ol> <li> On the left menu, select <span>Secret Manager</span> &gt; <span>Create</span>. </li> <li> In the <span>Create a New Secret page</span>, provide values for the <a href="#About" target="_blank">Secret attributes</a>. <br /><img src="/images/resources/images/integra/create-secret.png" alt="Create Secret" />  <h4>About <span>Secret</span> attributes</h4> <table> <col /> <col /> <thead> <tr> <th> </th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Secret Name</span> </td> <td> A unique name provided for the secret. It can contain ASCII letters, numbers, and the following characters: /&#95;+=.@-. </td> </tr> <tr> <td><span>Portfolio ID</span> </td> <td> The unique portfolio identifier. The portfolio is the overall customer account that contains user, package, network and SIM information, and includes details such as linked addresses, contact phone numbers, tax information and billing currency information. </td> </tr> <tr> <td><span>Description</span> </td> <td>Enter a brief note about the secret.</td> </tr> <tr> <td><span>Secret String</span> </td> <td> The text to encrypt and store in the secret. You can input multiple strings as a comma-separated list.  <span><strong>Tip: </strong>We recommend that you use a JSON structure of key/ value pairs for your secret string value.</span> </td> </tr> </tbody> </table> </li> <li> Select <span>Save</span> to store the created secret.  <br /><img src="/images/resources/images/integra/copy-secret key.png" alt="Copy Secret Key" />  <span><strong>Tip: </strong>The <span>Secret Name</span> is displayed only once. Use the copy icon to copy the secret name and store it securely.</span> </li> </ol>

<h2 id="Updatingsecretdetails">Updating secret details</h2>

<p>To edit your secret details:</p>

<ol> <li>  In the <span>View a Secret page</span>, select <span>Edit</span>. </li> <li> In the <span>Secret String</span> field, make the preferred changes. <br /><img src="/images/resources/images/integra/edit-secret.png" alt="Edit Secret" />  </li> <li> Select <span>Save</span> to apply the changes. <span><strong>Tip: </strong>You will receive a green <span>Secret Updated</span> notification to confirm this action.</span> <h2 id="Deletingsecrets">Deleting secrets</h2> </li> </ol>

<p>To delete your secrets:</p>

<ol> <li>  In the <span>View a Secret page</span>, select <span>Delete</span>. </li> <li> Select <span>Save</span> to apply the changes. <span><strong>Tip: </strong>You will receive a green <span>Secret Deleted</span> notification to confirm this action.</span> </li> </ol>
