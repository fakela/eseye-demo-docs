<p>Integra is a web interface that provides the tools to manage your IoT portal and customers. This includes managing customer accounts, creating packages, provisioning SIMs, checking billing records and troubleshooting customer issues.</p>

<Note>Integra is only available to selected customers. The functionality is also limited to searching for a SIM and displaying all information related to that SIM. For information about accessing Integra or upcoming features, contact Support.</Note>

<p> <img src="/images/resources/images/integra/overview.svg" alt="Overview" /> </p>

<p>The Integra web interface enables users to search for all information related to a SIM, including:</p>

<ul> <li>SIM-related information</li> <li>SIM connectivity and IMSI information</li> <li>Contract information</li> <li>Last connection information</li> <li>Billing information</li> </ul>

<Note>For more information, see <a href="/managing-my-estate/viewing-sim-details">Viewing and managing details for a SIM</a>.</Note>

<h2 id="AccessingtheIntegrawebinterface">Accessing the Integra web interface</h2>

<p>To request access to the Integra web interface:</p>

<ol> <li> Request user credentials by contacting Support. <br />Eseye will validate your request and email you (from no&#95;reply@anynetsim.com) your new username and a temporary password.  </li> <li> After you receive the email, navigate to the Integra web interface and log in as described below. <span><strong>Note: </strong>Eseye strongly recommends <a href="/integra/configuring-profile-settings#Change">changing your temporary password</a> as soon as possible. </span> </li> </ol>

<h3>Logging into Integra</h3>

<p>To log in to the Integra web interface:</p>

<ol> <li> Navigate to the following URL: <span><a href="https://integra.anynetiot.com/" target="_blank">https://integra.anynetiot.com</a></span> </li> <li>Enter your username and password.</li> <li>Select <span>Log In</span>.</li> </ol>

<h3>Logging out of Integra</h3>

<p>To log out of the Integra web interface:</p>

<ul> <li>Select <span>Logout</span> in the top right corner of the user interface.</li> </ul>

<h3>Resetting password in Integra</h3>

<p>To reset your password from the login screen:</p>

<ol> <li> Navigate to the following URL: <span><a href="https://integra.anynetiot.com/" target="_blank">https://integra.anynetiot.com</a></span> </li> <li>On the <span>Login</span> page, select <span>Forgotten password</span>.</li> <li> Enter your username and select <span>Send Code</span>.  <br />This displays the <span>Reset Password</span> page. <br /><img src="/images/resources/images/integra/resetpassword.png" alt="Resetpassword" />  <span><strong>Tip: </strong>Wait for an email (from no&#95;reply@anynetsim.com) which contains a six digit verification code.</span> </li> <li> When you receive the email, on the <span>Reset Password</span> page, enter the <span>verification code</span>, provide a <span>new password</span> and select <span>Update</span>. </li> <li>On the confirmation screen, select <span>Login</span> to return to the login page where you can now log in using your new password.</li> </ol>
