<p>Constrained Application Protocol. Designed for low memory and low power devices, this transport layer protocol is based on UDP but also works with TCP and SMS. CoAP interfaces with other IP protocols through simple proxies, using binary coded headers to exploit bandwidth with higher efficiency. All resources are represented on a unique address on a server called the URI, and requests for data are sent to this address. </p>
