<p>An interconnect integrates Eseye and a partner MNO, using the following components:</p>

<ul> <li>Eseye-distributed SIM profile or IMSI</li> <li>APN connection into an Eseye PoP, via a packet gateway</li> <li>MNO-supplied APIs for provisioning, SMS, and so on</li> </ul>
