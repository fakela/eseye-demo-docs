<p>The following estate reports help you administer your current SIM card estate, giving you a breakdown of individual SIM card usage and details, as well as contract terms, network changes, and SIM card status, authentication, and location information:</p>

<ul> <li><a href="#Account">Account snapshot report</a> </li> <li><a href="#Estate">Estate report</a> </li> <li><a href="#Group">Group usage report</a> </li> <li><a href="#NoData">No data used report</a> </li> <li><a href="#SimSettings">SIM settings report</a> </li> <li><a href="#SimLocation">SIM location report</a> </li> </ul>

<h2 id="AccountSnapshotReportPDF">Account Snapshot Report (PDF)</h2>

<p>Lists portfolio details, including active users, current packages, total data usage. This report gives you a broad understanding of the data your SIM card estate consumes, and who is accessing the estate.</p>

<p><img src="/images/resources/images/management/reports/accountsnapshot.png" alt="Accountsnapshot" /> </p>

<p>To generate the report in Infinity Classic:</p>

<ul> <li>See <a href="/analytics/managing-reports#CreatingReportProfiles">Creating report profiles</a>.</li> </ul>

<p>To find the report in Infinity Classic:</p>

<ol> <li>From the menu, select <span>Activity &gt; Reports</span> to display the <a href="/infinity-classic/activity#Reports">Reports page</a>. </li> <li>Alongside <span>Account Snapshot Report</span>, select <img src="/images/resources/images/infinityclassic/iconcsvblack_24x24.png" alt="Iconcsvblack 24X24" />.</li> </ol>

<h2 id="EstateReportCSV">Estate Report (CSV)</h2>

<p>Reports details about each SIM card in the portfolio estate, including current status, data usage in the last month, last authenticated, country, last IMEI.</p>

<p>This report is helpful for tracking and managing individual SIM cards. Use the CSV file to analyse your data usage and examine trends.</p>

<p><img src="/images/resources/images/management/reports/estatereport.png" alt="Estatereport" /> </p>

<p>To generate the report in Infinity Classic:</p>

<ul> <li>See <a href="/analytics/managing-reports#CreatingReportProfiles">Creating report profiles</a>.</li> </ul>

<p>To find the report in Infinity Classic:</p>

<ol> <li>From the menu, select <span>Activity &gt; Reports</span> to display the <a href="/infinity-classic/activity#Reports">Reports page</a>. </li> <li>Alongside <span>Estate Report</span>, select <img src="/images/resources/images/infinityclassic/iconcsvblack_24x24.png" alt="Iconcsvblack 24X24" />.</li> </ol>

<h2 id="GroupUsageReportPDF">Group Usage Report (PDF)</h2>

<p>Graph displaying daily data usage across all SIM cards, showing devices using the least data and devices using the most data.</p>

<p>This graph can help you analyse your data and troubleshoot changes in data usage.</p>

<p><img src="/images/resources/images/management/reports/groupusagereport.png" alt="Groupusagereport" /> </p>

<p>Reports are generated for all SIMs (above) or for SIMs on specific packages (below).</p>

<p><img src="/images/resources/images/management/reports/groupusagereport2.png" alt="Groupusagereport2" /> </p>

<p>To generate the report in Infinity Classic:</p>

<ul> <li>See <a href="/analytics/managing-reports#CreatingReportProfiles">Creating report profiles</a>.</li> </ul>

<p>To find the report in Infinity Classic:</p>

<ol> <li>From the menu, select <span>Activity &gt; Reports</span> to display the <a href="/infinity-classic/activity#Reports">Reports page</a>. </li> <li>Alongside <span>Group Usage Report</span>, select <img src="/images/resources/images/infinityclassic/iconpdfblack_24x24.png" alt="Iconpdfblack 24X24" />.</li> </ol>

<h2 id="NoDataUsedReportCSV">No Data Used Report (CSV)</h2>

<p>Details the ICCID and MSIDSN of active devices that have not used any data in the past month, including SIM status and where the device was last seen. This report helps you to troubleshoot device issues, or understand device usage, which can inform your decisions about the device.</p>

<p><img src="/images/resources/images/management/reports/nodataused.png" alt="Nodataused" /> </p>

<p>To generate the report in Infinity Classic:</p>

<ul> <li>See <a href="/analytics/managing-reports#CreatingReportProfiles">Creating report profiles</a>.</li> </ul>

<p>To find the report in Infinity Classic:</p>

<ol> <li>From the menu, select <span>Activity &gt; Reports</span> to display the <a href="/infinity-classic/activity#Reports">Reports page</a>. </li> <li>Alongside <span>No Data Used Report</span>, select <img src="/images/resources/images/infinityclassic/iconcsvblack_24x24.png" alt="Iconcsvblack 24X24" />.</li> </ol>

<h2 id="SIMSettingsReportPDFCSV">SIM Settings Report (PDF, CSV)</h2>

<p>Contains general information about an individual SIM card, including friendly name, users, IMEI, Alert Profile and Report Profile. This can feed into the administration and change management of the individual SIM cards.</p>

<p><img src="/images/resources/images/management/reports/simsettings.png" alt="Simsettings" /> </p>

<p>To find and generate the report in Infinity Classic:</p>

<ol> <li> From the menu, select <span>SIMs</span> &gt; <span>SIM List</span>. </li> <li>Alongside a SIM card, select <img src="/images/resources/images/infinityclassic/simactioniconsettings_24x24.png" alt="Simactioniconsettings 24X24" /> to display the <a href="/infinity-classic/sim-actions#Settings">Settings page</a>.</li> <li>  To download the information in a particular format, select either: <img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" /> or <img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />. </li> </ol>

<h2 id="SIMLocationReportPDF">SIM Location Report (PDF)</h2>

<p>World graphic displaying number of devices per country for all SIM cards in your estate. This visual representation helps you to understand the geographic spread of your SIM card estate, which may help you plan for expansion into other countries, or within a specific country.</p>

<p><img src="/images/resources/images/management/reports/simlocation.png" alt="Simlocation" /> </p>

<p>To generate the report in Infinity Classic:</p>

<ul> <li>See <a href="/analytics/managing-reports#CreatingReportProfiles">Creating report profiles</a>.</li> </ul>

<p>To find the report in Infinity Classic:</p>

<ol> <li>From the menu, select <span>Activity &gt; Reports</span> to display the <a href="/infinity-classic/activity#Reports">Reports page</a>. </li> <li>Alongside <span>SIM Location Report</span>, select <img src="/images/resources/images/infinityclassic/iconpdfblack_24x24.png" alt="Iconpdfblack 24X24" />.</li> </ol>
