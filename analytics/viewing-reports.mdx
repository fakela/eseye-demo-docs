<p>We offer a number of preconfigured reports. You can: </p>

<ul> <li><a href="#Viewing">View</a> reports.</li> <li><a href="#Download">Download</a> reports.</li> <li><a href="#Generati">Generate</a> on-demand reports.</li> </ul>

<p>For information about generating regular reports, see Scheduling Infinity reports.</p>

<p>To access Infinity reports:</p>

<ol> <li> On the left menu, select <span>Reports</span>. <br /> <img src="/images/resources/images/infinity/reportsicon_80x78.png" alt="Reportsicon 80X78" />  </li> <li> Select <span>My Reports</span>. <br /><img src="/images/resources/images/infinity/reportsview.png" alt="Reportsview" />  </li> </ol>

<h3>Filtering the displayed reports</h3>

<p>Refine the displayed <span>My Reports</span> list using the supplied filters.</p>

<p>To filter the My Reports list:</p>

<ol> <li>On the <span>My Reports</span> page, select a <span>Category</span> to constrain the available reports.</li> <li> Select the <span>Report</span> type that you want to view. <span><strong>Note: </strong>You will only see those reports that you have permission to view. For information about permissions, see &lt;xref&gt;.</span> </li> <li>Select a <span>Status</span> to filter by report status.</li> <li>Select <span>Filter</span> to apply your constraints to the returned <span>My Reports</span> list.</li> <li>Alternatively, select <span>Reset</span> to clear the filter constraints.</li> </ol>

<Tip>Also see <a href="/analytics/sorting-navigating-results">Sorting and navigating records</a>.</Tip>

<h2 id="Downloadingreports">Downloading reports</h2>

<p>To download a report:</p>

<ul> <li> On the <span>My Reports</span> page, in the <span>Actions</span> column, select <span>Download</span> alongside the report you want to download.  <br />The report instantly downloads to your local Downloads folder. </li> </ul>

<h2 id="Generatingondemandreports">Generating on-demand reports</h2>

<p>You can instantly generate a report.</p>

<p>To generate an on-demand report in Infinity:</p>

<ol> <li> On the <span>My Reports</span> page, select the <span>Actions</span> button (to the right of <span>Filter</span>) &gt; <span>Run an On-Demand report</span>. <br /><img src="/images/resources/images/infinity/run-on-demand.png" alt="Run On Demand" />  </li> <li> Select a <span>Category</span> to constrain the available reports. </li> <li> Select the <span>Report</span> that you want to generate. <span><strong>Note: </strong>You will only see those reports that you have permission to view. </span> </li> <li> Define a <span>Report Period</span> parameter to restrict how much information the report includes. <span><strong>Note: </strong>Reports display information based on the previous reporting period, which may not contain up-to-date or accurate information for the current point in time. </span> </li> <li> Select <span>Save</span> to generate the report. <br />If the report generates successfully, a green notification appears: "The report has been queued for creation and will be processed as soon as possible". <br />If you encounter errors, contact Support. </li> </ol>
