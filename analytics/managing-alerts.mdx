<p>In this section, you will learn how to set up Alert Profiles for your Infinity Classic account. </p>

<Tip> <p>Optionally, you can set up SIM alert notifications within a calendar month. You can also set up alerts for <a href="/infinity-classic/suspending-sims#top" target="_blank">Suspending SIMs</a>. You can receive SIM alert notifications via Infinity and via email.</p> </Tip>

<p>You can receive SIM alert notifications via Infinity Classic and via email.</p>

<h2 id="Viewingalerts">Viewing alerts</h2>

<p>To view alerts in Infinity Classic:</p>

<p>Go to: <span>ACTIVITY</span> &gt; <span>Alerts</span></p>

<p>OR</p>

<p colspan="2">Go to: <span>SIMS</span> &gt; <span>SIM List</span></p>

<p>If a SIM has breached data limits defined in an alert, these are displayed in the SIM List <span>Alert</span> column:</p>

<p><img src="/images/resources/images/infinityclassic/simalerts.png" alt="Simalerts" /> </p>

<h2 id="Usingalertprofiles">Using alert profiles</h2>

<p>To create an alert profile in Infinity Classic:</p>

<p>Go to: <span>SETTINGS</span> &gt; <span>Alert Profiles</span></p>

<ol> <li>Select <span>Add New Alert Profile</span></li> <li>Type a Profile name. </li> <li>Select the <span>Default Profile</span> check box if you want this profile to become the default profile for newly activated SIMs.</li> <li> Set up the financial, data, messaging and device alerts for this profile.  </li> <li> Select <span>Save</span>. <br />The profile saves and closes. </li> </ol>

<p>To assign an alert profile to an individual SIM in Infinity Classic:</p>

<ol> <li>Go to: <span>SIMS</span> &gt; <span>SIMs List</span> &gt; <span>Action</span> <img src="/images/resources/images/icons/siamsimsettings_ic_24x24.png" alt="Siamsimsettings Ic 24X24" /></li> <li>On the <span>SIM Settings</span> form, select <span>Edit</span>.</li> <li>Using the <span>Alert Profile</span> drop-down menu, select the Alert Profile for the SIM.</li> <li>Select <span>Save</span> to save your changes.</li> <li>Close <span>SIM Settings</span>.</li> </ol>

<p>To assign an alert profile to multiple SIMs in Infinity Classic:</p>

<ol> <li> Using Infinity Classic, go to: <span>SIMS</span> &gt; <span>SIM List</span> </li> <li> Select the checkbox alongside each SIM you want to assign an Alert Profile to. <span><strong>Tip: </strong>Select the column header checkbox to select all SIMs in the list.</span> </li> <li>At the bottom of the SIM list, in the drop-down box, select <span>Change Settings</span>.</li> <li> Select <span>Perform Bulk Action</span>. <br /><img src="/images/resources/images/infinityclassic/bulkassignalerts.png" alt="Bulkassignalerts" />  <br />The Bulk change SIM Settings form appears. </li> <li>Using the <span>Alert Profile</span> drop-down menu, select the Alert Profile for the chosen SIMs.</li> <li>Select <span>Save</span> to save your changes.</li> <li>Close <span>Bulk change SIM Settings</span>.</li> </ol>

<p>To view or edit an alert profile in Infinity Classic:</p>

<ol> <li>Go to: <span>SETTINGS</span> &gt; <span>Alert Profiles</span>.</li> <li>Alongside the Alert Profile you want to edit, select <img src="/images/resources/images/billing/edit_ic.png" alt="Edit Ic" />.</li> </ol>

<h2 id="Enablingemailalerts">Enabling email alerts</h2>

<p>To enable SIM alert emails in Infinity Classic:</p>

<Note>If an account has multiple users, then each user must set up this feature if they want to receive alerts via email.</Note>

<ol> <li colspan="2">Go to: <span>&lt;Username&gt;</span> &gt; <span>Notifications </span></li> <li colspan="2">In the <span>Alert Profile Notifications</span> box, select the <span>Enable Alert Notification</span> check box.</li> <li colspan="2">Select <span>Update</span> to save your changes.</li> </ol>

<Note> <p colspan="2">Alert notification emails are sent to the Account holder's email address. </p> <p colspan="2">To view or edit your email address, go to: <span>&lt;Username&gt;</span> &gt; <span>Details</span></p> </Note>
