<h2 id="Sortingreturnedresults">Sorting returned results</h2>

<p>To sort the search results view by column name:</p>

<ul> <li>Select a column name to sort the results in ascending <img src="/images/resources/images/infinity/ascending_order_24x24.png" alt="Ascending Order 24X24" /> (a-z; 1-9) or descending <img src="/images/resources/images/infinity/descending_order_24x24.png" alt="Descending Order 24X24" /> (z-a; 9-1) order.</li> </ul>

<h2 id="Navigatingmultipleresultspages">Navigating multiple results pages</h2>

<p>This feature is only available on some pages, when there is a large number of returned results.</p>

<h5>Viewing more results</h5>

<p>Some pages return results in a continuous list. Select <span> View More</span> at the end of the list to see more items.</p>
