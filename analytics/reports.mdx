<p>Infinity reports offer structured insights into your SIM estate, helping you to monitor SIM and network performance and compliance, analyse financial data, and track connectivity. Use the reports to identify usage trends, resolve issues, and make informed decisions to optimise your resources effectively.</p>

<p>You can view, download and generate on-demand reports. For more information, see <a href="/analytics/viewing-reports#Viewing">Viewing Infinity reports</a></p>

<p>You can configure schedules to generate and distribute reports at set intervals. For more information, see Scheduling Infinity reports.</p>

<h2 id="Reportingservicesdeveloperdocumentation">Reporting services developer documentation</h2>

<p>We offer report APIs and object reference tables so that you can build your own reports. For more information, see <a href="/developer-docs/getting-started-for-developers">Getting started for developers</a>.</p>
