<p>The Infinity dashboard displays high-level key metrics, such as:</p>

<ul> <li>How many <span>Active</span>, <span>Available</span>, <span>Suspended</span> and <span>Total</span> SIMs you currently have.</li> <li>Business performance snapshot, using SIM data.</li> </ul>

<h2 id="ViewingtheInfinitydashboard">Viewing the Infinity dashboard</h2>

<p>To view the Infinity dashboard:</p>

<ul> <li> On the left menu, select <span>Dashboard</span>. <br /> <img src="/images/resources/images/infinity/db-icon_80x78.png" alt="Db Icon 80X78" />  </li> </ul>

<h2 id="Viewinghelpinformation">Viewing help information</h2>

<p>To view tooltips: </p>

<ul> <li> Select <img src="/images/resources/images/infinity/help-icon_24x24.png" alt="Help Icon 24X24" /> at the top right of a gadget to view helpful information.  <br /><img src="/images/resources/images/infinity/dashboard-user-page.png" alt="Dashboard User Page" />  </li> </ul>
