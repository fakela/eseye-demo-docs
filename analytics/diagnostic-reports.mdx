<p>The following diagnostic reports help you to detect network and SIM card issues, and to track data usage to identify connectivity problems:</p>

<ul> <li><a href="#Summary">SIM summary report</a> </li> <li><a href="#Radius">SIM RADIUS activity report</a> </li> <li><a href="#Netflow">SIM NetFlow activity report</a> </li> <li><a href="#Network">SIM network settings report</a> </li> <li><a href="#Messaging">SIM messaging activity report</a> </li> </ul>

<h2 id="SIMsummaryPDFCSV">SIM summary (PDF, CSV)</h2>

<p>Gives details about the SIM card, including the latest connection and location details, IMEI, last data session datetime stamp and data usage, cumulative data usage for the month, and last voice and messaging events.</p>

<p><img src="/images/resources/images/management/reports/simsummary.png" alt="Simsummary" /> </p>

<p>To find and generate the report in Infinity Classic:</p>

<ol> <li> From the menu, select <span>SIMs</span> &gt; <span>SIM List</span>. </li> <li>Alongside a SIM card, select <img src="/images/resources/images/infinityclassic/simactioniconsummary_24x24.png" alt="Simactioniconsummary 24X24" /> to display the <a href="/infinity-classic/sim-actions#Summary">Summary of activity page</a>.</li> <li>  To download the information in a particular format, select either: <img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" /> or <img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />. </li> </ol>

<h2 id="SIMRADIUSActivityPDFCSV">SIM RADIUS Activity (PDF, CSV)</h2>

<p>Shows SIM card data sessions, including start and stop times, data usage, IMSI and network operator.</p>

<p>Use this report to identify trends in your network usage and identify networking issues. For example, poor signal strength is often indicated where a device is constantly switching networks.</p>

<p><img src="/images/resources/images/management/reports/simradiusactivity.png" alt="Simradiusactivity" /> </p>

<p>To find and generate the report in Infinity Classic:</p>

<ol> <li> From the menu, select <span>SIMs</span> &gt; <span>SIM List</span>. </li> <li>Alongside a SIM card, select <img src="/images/resources/images/infinityclassic/simactioniconradius_24x24.png" alt="Simactioniconradius 24X24" /> to display the <a href="/infinity-classic/sim-actions#Radius">RADIUS page</a>.</li> <li>  To download the information in a particular format, select either: <img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" /> or <img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />. </li> </ol>

<h2 id="SIMNetFlowActivityPDFCSVPNG">SIM NetFlow Activity (PDF, CSV, PNG)</h2>

<p>Graphic reporting daily data flows in kilobytes to a SIM card (MT) and from the SIM card (MO) either:</p>

<ul> <li>Per day for a selected month</li> <li>Per hour for a selected day</li> <li>Per minute for a selected hour</li> </ul>

<p>Use this report to understand data usage and changes in SIM card behavior, which may be a result of network issues.</p>

<p><img src="/images/resources/images/management/reports/simnetflowactivity.png" alt="Simnetflowactivity" /> </p>

<p>To find and generate the report in Infinity Classic:</p>

<ol> <li> From the menu, select <span>SIMs</span> &gt; <span>SIM List</span>. </li> <li>Alongside a SIM card, select <img src="/images/resources/images/infinityclassic/simactioniconnetflow_24x24.png" alt="Simactioniconnetflow 24X24" /> to display the <a href="/infinity-classic/sim-actions#NetFlow">NetFlow page</a>.</li> <li>  To download the information in a particular format, select either: <img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />, <img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" /> or <img src="/images/resources/images/infinityclassic/iconpng_24x24.png" alt="Iconpng 24X24" />. </li> </ol>

<h2 id="SIMNetworkSettingsPDFCSV">SIM Network Settings (PDF, CSV)</h2>

<p>Displays the current status and network information (IMSI, MSISDN, and if relevant, IP address) for an individual SIM. Use this report to understand your device connectivity. This report can feed into other reports to check if network changes have occurred that may affect the device.</p>

<p><img src="/images/resources/images/management/reports/simnetworksettings.png" alt="Simnetworksettings" /> </p>

<p>To find and generate the report in Infinity Classic:</p>

<ol> <li> From the menu, select <span>SIMs</span> &gt; <span>SIM List</span>. </li> <li>Alongside a SIM card, select <img src="/images/resources/images/infinityclassic/simactioniconnetwork_24x24.png" alt="Simactioniconnetwork 24X24" /> to display the <a href="/infinity-classic/sim-actions#Network">Network settings page</a>.</li> <li>  To download the information in a particular format, select either: <img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" /> or <img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />. </li> </ol>

<h2 id="SIMMessagingActivityPDFCSV">SIM Messaging Activity (PDF, CSV)</h2>

<p>Contains details about every SMS sent and received per MSISDN, as well as message status (such as Delivered) and Text details. Use this report to track SMS usage per SIM card.</p>

<p><img src="/images/resources/images/management/reports/simmessagingactivity.png" alt="Simmessagingactivity" /> </p>

<p>To find and generate the report in Infinity Classic:</p>

<ol> <li> From the menu, select <span>SIMs</span> &gt; <span>SIM List</span>. </li> <li>Alongside a SIM card, select <img src="/images/resources/images/infinityclassic/simactioniconmessaging_24x24.png" alt="Simactioniconmessaging 24X24" /> to display the <a href="/infinity-classic/sim-actions#Messaging">Messaging page</a>.</li> <li>  To download the information in a particular format, select either: <img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" /> or <img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />. </li> </ol>
