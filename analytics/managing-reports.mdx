<p>Eseye generate daily, weekly and monthly reports to help you to better understand your devices and portfolio. These include:</p>

<ul> <li><a href="/analytics/audit-reports">Templates</a> – provide account usage and audit trails to keep staff accountable.</li> <li><a href="/analytics/diagnostic-reports">Diagnostic Reports</a> – provide information about SIM and network issues.</li> <li><a href="/analytics/estate-reports">Estate Reports</a> – provide information to help you administer your SIM estate.</li> <li><a href="/analytics/finance-reports">Finance Reports</a> – provide a range of information about the SIMs, such as data usage and costs involved.</li> <li><a href="/analytics/status-and-connectivity-reports">Status and Connectivity Reports</a> – provide information about network connectivity and SIM statuses.</li> </ul>

<p>You can access all these reports using Infinity Classic. You need permission to see specific reports. </p>

<p>For information about Eseye's available reports and how to generate them, see <a href="https://docs.eseye.com/Content/Resources/Files/8680-Available-Reports-Overview.pdf" target="_blank">8680 Available Reports Overview (PDF)</a>.</p>

<h2 id="Creatingreportprofiles">Creating report profiles</h2>

<p>To create a report profile in Infinity Classic:</p>

<ol> <li> Go to: <span>Settings</span> &gt; <span>Reporting Profile</span> </li> <li>Select <span>Add New Report Profile</span></li> <li>Type a <span>Profile Name</span>. </li> <li>Select the <span>Default Profile</span> check box if you want this profile to become the default profile for newly activated SIMs.</li> <li>Select the <span>Enable</span> checkbox for each report you want to generate.</li> <li>Select a report frequency using the <span>Interval</span> drop-down menu.</li> <li> Select <span>Save</span>. <br />The profile saves and closes. </li> <li> If you edit the profile, the <span>Run Now</span> action is available to generate the report immediately. </li> </ol>

<h2 id="Managingreportprofiles">Managing report profiles</h2>

<p>To view a report profile:</p>

<ol> <li>Go to: <span>Activity</span> &gt; <span>Reports</span></li> <li> Alongside the report you want to see, select a format to instantly download.  <br />The following formats may be available, depending on the report: <img src="/images/resources/images/billing/csv_ic.png" alt="Csv Ic" /><img src="/images/resources/images/billing/pdf_ic.png" alt="Pdf Ic" /><img src="/images/resources/images/billing/xml_ic.png" alt="Xml Ic" /> </li> </ol>

<p>To edit a report profile:</p>

<ol> <li>Go to: <span>Settings</span> &gt; <span>Reporting Profile</span></li> <li>Alongside the Report Profile you want to edit, select <img src="/images/resources/images/billing/edit_ic.png" alt="Edit Ic" />.</li> </ol>

<p>To enable report notification emails:</p>

<Note>If an account has multiple users, then each user must set up this feature if they want to receive email notifications that a report has become available in Infinity Classic.</Note>

<ol> <li colspan="2"> Go to: <span>&lt;Username&gt;</span> &gt; <span>Notifications</span> </li> <li> In the <span>Report Profile Notifications</span> box, select the checkbox alongside the notifications you want to receive for: <ul> <li><span>SIM Admin Reports</span> – reports describing what was done to the SIM</li> <li> <span>SIM Usage Reports</span> – reports describing service used by the SIM </li> </ul> </li> <li colspan="2">Select <span>Update</span> to save your changes.</li> </ol>

<Note> <p>Report notification emails are sent to the Account holder's email address. </p> <p>To view or edit your email address, go to: <span>&lt;Username&gt;</span> &gt; <span>Details</span></p> </Note>

<h2 id="AssigningreportprofilestoSIMs">Assigning report profiles to SIMs</h2>

<p>To assign a report profile to an individual SIM:</p>

<ol> <li>Go to: <span>SIMS</span> &gt; <span>SIMs List</span> &gt; <span>Action</span> <img src="/images/resources/images/icons/siamsimsettings_ic_24x24.png" alt="Siamsimsettings Ic 24X24" /></li> <li>On the <span>SIM Settings</span> form, select <span>Edit</span>.</li> <li>Using the <span>Report Profile</span> drop-down menu, select the Report Profile for the SIM.</li> <li>Select <span>Save</span> to save your changes.</li> <li>Close <span>SIM Settings</span>.</li> </ol>

<p>To assign a report profile to multiple SIMs:</p>

<ol> <li> Using Infinity Classic, go to: <span>SIMs</span> &gt; <span>SIM List</span> </li> <li> Select the checkbox alongside each SIM you want to assign the Report Profile to. <span><strong>Tip: </strong>Select the column header checkbox to select all SIMs in the list.</span> </li> <li>At the bottom of the SIM list, in the drop-down box, select <span>Change Settings</span>.</li> <li> Select <span>Perform Bulk Action</span>. <br /><img src="/images/resources/images/infinityclassic/bulkassignalerts.png" alt="Bulkassignalerts" />  <br />The Bulk change SIM Settings form appears. </li> <li>Using the <span>Report Profile</span> drop-down menu, select the Report Profile for the chosen SIMs.</li> <li>Select <span>Save</span> to save your changes.</li> <li>Close <span>Bulk change SIM Settings</span>.</li> </ol>
