<p>The AnyNet Federation is a partnership of mobile network operators (MNOs) that have agreed to accept network traffic from each other using Eseye's Connectivity Management Platform (CMP). </p>

<p>For example, Eseye can steer a device to use an MTN IMSI in South Africa, which will also give the device access to the Vodacom roaming profile. </p>

<p>Some AnyNet Federation partners offer full roaming on their network, whilst others offer access to a local profile using localisation, which enables devices to meet data sovereignty and regulatory requirements. For more information, see <a href="/getting-started/localisation" target="_blank">Understanding localisation</a>.</p>

<p>Eseye typically uses the AnyNet Federation partners where there are commercial or regulatory issues for permanent roaming.</p>

<p>For information about which MNOs are part of the AnyNet Federation, see <a href="/connectivity/mnos-by-pop" target="_blank">Mobile Network Operators by PoP</a>.</p>
