<h2 id="LTEMLTECatM">LTE-M / LTE Cat-M</h2>

<p>LTE-M is an IoT-specific radio access technology that utilises existing LTE infrastructure to support IoT devices and their specific use cases, including:</p>

<ul> <li>Low power consumption – supporting the <a href="/lpwan/ps-mande-drx">eDRX and PSM</a> power-saving features</li> <li>Long range – up to 10-15km</li> <li>Low data rates – maximum of 1Mbps (for CAT-M1 devices)</li> </ul>

<p>Although LTE-M doesn’t provide the same battery life length as NB-IoT, it offers a higher bandwidth which can support higher data rates.</p>

<h2 id="NBIoT">NB-IoT</h2>

<p>Narrowband Internet of Things (NB-IoT) is a radio access technology that utilises narrow frequency ranges (180kHz) in existing LTE radio bands to provide features and services that are optimised for IoT devices. This includes:</p>

<ul> <li>Lower power requirements (utilising the <a href="/lpwan/ps-mande-drx">eDRX and PSM</a> power-saving features)</li> <li>Lower data rates</li> </ul>

<p>For more information, see 3GPP <a href="https://www.3gpp.org/news-events/3gpp-news/1785-nb_iot_complete" target="_blank">release 13 specifications</a>.</p>
