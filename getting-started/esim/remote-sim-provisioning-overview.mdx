<p>Traditional SIM cards comprise of a single SIM profile that contains operator and subscriber data. In order to change which mobile network you are using, you need to physically replace the SIM.</p>

<p><a href="/getting-started/esim/euicc">eUICC</a> SIMs are single SIMs that can host multiple network operator profiles, which are installed and managed remotely. </p>

<p><img src="/images/resources/images/esim/rsp-a.png" alt="Rsp A" /> </p>

<p>When a device connects to a mobile network, the operator uses a Remote SIM Provisioning (RSP) system to securely download the profile for their network onto the SIM. </p>

<p><img src="/images/resources/images/esim/rsp-b.png" alt="Rsp B" /><img src="/images/resources/images/esim/rsp-c.png" alt="Rsp C" /> </p>

<p>When the profile is installed and activated, the device can connect to that operator's network. If the SIM contains multiple profiles, then you can change which network the device is using by remotely switching its current SIM profile. </p>

<p><img src="/images/resources/images/esim/rsp-d.png" alt="Rsp D" /> </p>

<Note>SIMs can only connect to one network at a time.</Note>

<p>For information about which modules support RSP, see <a href="/getting-started/esim/modules-supporting-rsp">Modules supporting Remote SIM Provisioning (RSP)</a>.</p>

<h2 id="BenefitsofusingremoteSIMprovisioning">Benefits of using remote SIM provisioning</h2>

<p>Loading an operational profile onto an eUICC SIM offers several benefits, including:</p>

<ul> <li> <span>Single hardware with multiple network profiles:</span> eUICC SIMs can host multiple profiles, eliminating the need for physical SIM swaps when changing network providers. </li> <li> <span>Remote provisioning:</span> Operational profiles allow for remote management, activation, and configuration of subscription services without requiring physical access to devices. This is particularly useful where IoT devices are geographically dispersed or difficult to reach. </li> <li> <span>Dynamic network selection:</span> Devices with eUICC SIMs can switch between mobile networks based on predefined criteria, such as signal strength, network availability, or cost considerations. </li> <li> <span>Efficient subscription management:</span> Centralised control and administration of your SIM estate. </li> <li> <span>Continuous flexible connectivity:</span> Operational profiles enable profile switching based on factors like location and network availability, ensuring your device has near 100% connectivity. </li> <li> <span>Customised services:</span> Operational profiles can store additional information beyond connectivity services, such as storing messages and customised settings. </li> <li> <span>Future-Proofing:</span> You can prepare for future services and technologies by remotely downloading updated profiles onto the SIM, allowing for new features such as improved security protocols or emerging connectivity standards. </li> </ul>
