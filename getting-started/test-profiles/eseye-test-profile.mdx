<p>Eseye supplies a test profile for downloading onto a v5 or v6 multi-IMSI bootstrap for testing network connectivity or equipment. </p>

<p>For an IoT device to register on the test equipment, the authentication parameters on the SIM must match those loaded onto the test network. </p>

<p>Eseye can encrypt the test profile, and load it onto a SIM at manufacture, through a SIM reader interface, or over-the-air (OTA).</p>

<p>We use the following test profile information:</p>

<ul> <li><span>imsi:</span> – 001001000000001</li> <li><span>op</span> – FFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF</li> <li><span>amf</span> – 8000</li> <li><span>ki</span> – 5C95978B5E89488CB7DB44381E237809</li> <li><span>algorithm</span> – milenage </li> </ul>
