<p>The contract information in a package outlines the details and terms of agreement between Eseye Ltd and the customer, including service deliverables, service duration, pricing, and so on. This information serves as a reference point for understanding the rights, responsibilities, and expectations of each party throughout the contract's term.</p>

<h2 id="Viewingpackagecontractinformation">Viewing package contract information</h2>

<p>To view the package contract information:</p>

<ol> <li> Using Integra, on the left menu, select <span>Packages</span> to view the <span>Packages</span> page. <br /> <img src="/images/resources/images/integra/packages-icon.png" alt="Packages Icon" />  </li> <li> Select the package <span>Id</span> to view a specific package. </li> <li> Select the <span>Contract</span> tab to view contract information relating to that package. <br /><img src="/images/resources/images/integra/contract-package.png" alt="Contract Package" />  </li> </ol>

<h2 id="Changingpackagestatus">Changing package status</h2>

<p>To change the status of your package:</p>

<ol> <li> On the package <span>Contract</span> page, alongside <span>Package Status</span>, select <img src="/images/resources/images/infinity/iconedit_24x24.png" alt="Iconedit 24X24" />. <br />The package status dialog appears. <br /><img src="/images/resources/images/integra/edit-package status.png" alt="Edit Package Status" />  </li> <li>Select the new <span>Package Status</span> from the drop-down list.</li> <li> Select <span>Update</span> to commit the new status. </li> </ol>
