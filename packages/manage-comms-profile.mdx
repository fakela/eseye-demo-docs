<p>A communication profile is a set of parameters and configurations that defines how a SIM or device communicates within a network. This profile typically includes details such as subnet range type, ICC types, MSISDN ranges, and other necessary configurations to ensure proper connectivity and functionality within the network. It essentially determines how the SIM or device interacts with the network and accesses various Eseye services.</p>

<h2 id="Viewingthecommunicationsprofileforaspecificpackage">Viewing the communications profile for a specific package</h2>

<p>To view the comms profile for a specific package:</p>

<ol> <li> Using Integra, on the left menu, select <span>Packages</span>. <br /> <img src="/images/resources/images/integra/packages-icon_80x78.png" alt="Packages Icon 80X78" />  </li> <li> On the <span>Packages</span> page, in the Packages list, select the package <span>ID</span> where you want to view or update the communications profile. </li> <li> Select the <span>Comms Profile</span> tab. <br /><img src="/images/resources/images/integra/commsprofile.png" alt="Commsprofile" />  <br />The Comms Profile displays the following fields: <table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Subnet Range Type</span> </td> <td> The network routing type used by the package. </td> </tr> <tr> <td><span>ICC Types</span> </td> <td> The SIM types that can use the current package.  </td> </tr> <tr> <td><span>MSISDN Ranges</span> </td> <td> The MSISDNs allocated to the current package. You can input multiple MSISDNs as a comma-separated list. </td> </tr> </tbody> </table> </li> </ol>

<h2 id="Updatingthecommunicationsprofile">Updating the communications profile</h2>

<p>To update the comms profile:</p>

<ol> <li>View the package <span>Comms Profile</span> tab.</li> <li> Alongside the <span>Comms Profile</span>, select <span>Edit</span>.</li> <li>Make your changes.</li> <li>Select <span>Save</span> to commit your changes to the package.</li> </ol>
