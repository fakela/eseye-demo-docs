<p>Users are assigned roles that grant access to different parts of the portal, as well as give permission to perform the following actions:</p>

<ul> <li><span>View</span> – enables the user to view the records</li> <li><span>Create</span> – enables the user to create a new record</li> <li><span>Update</span> – enables the user to edit an existing record</li> <li><span>Approve</span> – enables the user to approve the record for use</li> </ul>

<Note>Support can <a href="#Creating">create</a> new roles. These roles are published to the respective portals or portfolios. </Note>

<p>You can also create, update, and delete users. For more information, see <a href="/portfolios/manage-users">Managing users</a>.</p>
