<p>The Hera is designed for wall-mounting.</p>

<h2 id="Choosingalocation">Choosing a location</h2>

<p>Read all <a href="/hardware-products/hera/600-series/warnings604v5" target="_blank">warnings</a> before choosing a location for the Hera.</p>

<p>Locate the Hera in a weather-protected, stable, vibration-free environment.</p>

<ul> <li><span>Operating temperature:</span> -20°C to +55°C (-4°F to 131°F)</li> <li><span>Relative humidity:</span> 5-95%, non-condensing</li> </ul>

<Tip>Consider cellular signal strength, WiFi strength and signal interference from other equipment when choosing an installation location.</Tip>

<Tip> Consider if you need to easily see and access any antenna connectors, cable connection points, and LED indicators on the top, front, and bottom panels.</Tip>

<h2 id="WallmountingtheHera">Wall-mounting the Hera</h2>

<div> <h3>Before you begin</h3> <p>You will need:</p> <table> <col /> <col /> <tbody> <tr> <td> <ul> <li>To install the security plate – Security Torx screwdriver (T10 tip)</li> <li>Tape measure</li> <li>Spirit level</li> <li>Pencil</li> <li>Masking tape</li> <li>Cross head Pozidriv/Philips screwdriver</li> </ul> </td> <td> <ul> <li> Drill –  <br />Metric Ø 5mm <br /> Imperial Ø ³⁄₁₆" </li> <li> 4 x Plastic wall plugs/anchors –  <br />Metric Ø 5mm, L30mm  <br />Imperial Ø ³⁄₁₆", L1¼" </li> <li> 4 x pan head wood screws –  <br />Metric Ø 3.5mm, L30mm <br />Imperial Ø #6, L1¼" </li> </ul> </td> </tr> </tbody> </table> </div>

<ol> <li> If a SIM card is supplied, ensure it is fully inserted in the Hera.  <div> <img src="/images/resources/images/hera/insertsims.png" alt="Insertsims" />  </div> <span><strong>Tip: </strong>The socket is a push-to-insert, push-to-release type socket. When the SIM is properly inserted, it clicks into place and sits flush with the edge of the SIM slot. </span> </li> <li>If required, fit the SIM security plate (see below).</li> <li> Print the following drill template: <br /> <img src="/images/resources/images/hera/heradrilltemplate_650x526.png" alt="Heradrilltemplate 650X526" />  </li> <li>Measure the template to ensure it is accurate, as the printing process may distort it. </li> <li>If the template is accurate, fix the page to the wall using the spirit level and masking tape. Alternatively, use the Hera to mark the drill holes.</li> <li>Drill holes and prepare them with the wall plugs/anchors.</li> <li>Attach the cellular cables (Primary and RX Div)  and WiFi antennas to the Hera.</li> <li>Mount the Hera on the wall using the recommended fixings.</li> <li> Position all antennas for optimal performance. </li> <li>Insert the power connector.</li> <li>Plug the mains adaptor into a dedicated socket, then switch on.</li> <li> Ensure all cables and antennas are securely anchored to avoid intermittent connection problems.</li> <li> Configure the Hera using the supplied web user interface. <br />For more information, see <a href="/hardware-products/hera/600-series/connections604v5">How to connect to the Hera 604v5</a>. </li> </ol>
