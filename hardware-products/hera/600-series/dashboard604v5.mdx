<div> <p><span>Model:</span> Hera 604 <span>Version:</span> v5</p> </div>

<p>The Hera series router welcome page is divided into two sections:</p>

<p><img src="/images/resources/images/hera/herauiwelcomepage.png" alt="Herauiwelcomepage" /> </p>

<ol> <li> Lists different menus for configuring the Hera: <ul> <li> <span>System status</span> – for read-only information detailing the router settings, LAN, mobile network, and connected devices. For more information, see <a href="/hardware-products/hera/600-series/system-status604v5">Viewing the System Status</a>. </li> <li> <span>Setup wizards</span> – to help you configure cellular and Ethernet connections, and the Health Monitor to maintain these connections. For more information, see <a href="/hardware-products/hera/600-series/setup-wizards604v5">Using the Setup Wizards</a>. </li> <li> <span>Basic Settings</span> – to configure router, LAN, wireless network, mobile network, Ethernet WAN, routing and security settings. For more information, see <a href="/hardware-products/hera/600-series/basic-settings604v5">Configuring Basic Settings</a>. </li> <li> <span>Advanced Settings</span> – for lower-level configuration access to the router. </li> <li> <span>Hosted Applications</span> – to enable and configure supplied applications for connecting to the cloud, Serial to IP, Modbus to MQTT, or configuring and managing your own applications. </li> <li> <span>Diagnostics &amp; Maintenance</span> – for configuring the Health Monitor, creating backups and uploading software, configuring the Network Time Protocol (NTP), changing the password, resetting or restarting the router, and data logging. For more information, see <a href="/hardware-products/hera/600-series/health-monitor604v5">Configuring the Health Monitor</a>. </li> </ul> </li> <li>Lists real-time information regarding the specific Hera you’re administering, including the current cellular connectivity status. For more information, see <a href="/hardware-products/hera/600-series/system-status604v5">Viewing the System Status</a>.</li> </ol>
