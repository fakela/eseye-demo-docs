<p>Connect to the Hera with an Ethernet cable or over WiFi.</p>

<Tip>Eseye may include a unique <span>Configuration Information Sheet</span> with each Hera Series Router. Use this information to connect, log into, and configure the Hera. If the sheet is not included, your service provider or reseller will advise you of the contents.</Tip>

<Accordion title="Connect using an Ethernet (wired) connection">
 <ul> <li>Use an Ethernet cable to connect from the Hera Ethernet port <span>Link</span><span>2</span> to a PC or laptop, either directly or via an Ethernet switch.</li> </ul> 
</Accordion>

<Accordion title="Connect using WiFi">
 <span><strong>Note: </strong>If there is no SSID information supplied, use a wired connection to the Hera Series Router.</span> <ol> <li>Ensure the Hera WiFi LED is illuminated.</li> <li>Search wireless networks on your WiFi-enabled device. Connect to the network that matches the SSID.</li> <li>At the prompt, type the WiFi username and passphrase exactly as supplied.</li> </ol> 
</Accordion>

<Accordion title="Logging in to the Hera Series Router">
 <ol> <li> Using a browser on your device, type the supplied <b>Web Interface IP</b> into the address bar. For example:  <br /><span>http://192.168.0.1</span>  </li> <li> Press return. </li> <li> At the prompt, enter the supplied credentials, then select <span>OK</span>. <br />The Hera Series Router web interface appears. </li> </ol> 
</Accordion>

<h2 id="Configuringthecellularconnection">Configuring the cellular connection</h2>

<p>The Hera is preconfigured to connect to a cellular network using an Eseye AnyNet SIM.</p>
