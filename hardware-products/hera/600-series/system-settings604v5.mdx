<h2 id="ConfiguringtheNetworktimeprotocolNTP">Configuring the Network time protocol (NTP)</h2>

<p>The Hera enables clock synchronisation between systems to Coordinated Universal Time (UTC). You can use the default tier 3 service (0.openwrt.pool.ntp.org), or add your own. For each additional server, the tier increases by 1. For example, a tier 3 service will become a tier 4 service if you add another server.</p>

<div> <h3>Before you begin</h3> <ul> <li>Install and configure an NTP server on your network.</li> </ul> </div>

Select <span>Save</span> to save your changes. <br />Select <span>-</span> alongside an existing server to delete it. </td> </tr> <tr> <td><span>Use ipv4 for NTP server name resolution</span> </td> <td> Select: <br /><span>Yes</span> – enables the DNS server to resolve the host name to an IP address <br /><span>No</span> – prevents the DNS server resolving the host name to an IP address </td> </tr> <tr> <td><span>Minimum poll time</span> </td> <td>Minimum time between NTP server poll requests. Select the required value from the drop-down list.</td> </tr> <tr> <td><span>Maximum poll time</span> </td> <td>Maximum time between NTP server poll requests.
