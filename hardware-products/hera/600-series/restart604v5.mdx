---
title: "Restarting the Hera"
description: "Model: Hera 604 Version: v5"
---

<div> <p><span>Model:</span> Hera 604 <span>Version:</span> v5</p> </div>

<p>Configure a restart here:</p>

<p><span>Diagnostics &amp; Maintenance</span> &gt; <span>Restart</span></p>

<p>To perform a reboot:</p>

<ol> <li> Using the Hera user interface, navigate to <span>Diagnostics &amp; Maintenance</span> &gt; <span>Restart</span> &gt; <span>Reboot</span>. <br />The Reboot page appears. </li> <li> Select <span>Reboot</span>. <br />A confirmation dialog box appears. </li> <li> Select <span>Yes</span> to reboot the Hera, or <span>No</span> to cancel the action. </li> </ol>

<p>To perform a factory reset:</p>

<p>This action will delete any configuration changes you have made to the Hera and restore the settings back to the original Eseye configuration.</p>

<ol> <li> Using the Hera user interface, navigate to <span>Diagnostics &amp; Maintenance</span> &gt; <span>Restart</span> &gt; <span>Factory reset</span>. <br />The Factory reset page appears. </li> <li> Select <span>Reset</span>. <br />A confirmation dialog box appears. </li> <li> Select <span>Yes</span> to reset the Hera to the original configuration, or <span>No</span> to cancel the action. </li> </ol>
