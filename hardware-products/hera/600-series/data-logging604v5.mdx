---
title: "Using data logging to track events"
description: "Model: Hera 604 Version: v5"
---

<div> <p><span>Model:</span> Hera 604 <span>Version:</span> v5</p> </div>

<p> </p>

<p>Data logging is enabled by default to track Hera events. You can set the logging frequency and disable logging if required. </p>

<Tip>For information about viewing the log files, see <a href="/hardware-products/hera/600-series/performance604v5">Viewing Hera data performance</a>.</Tip>

<p>To configure data logging:</p>

<ol> <li> Using the Hera user interface, navigate to <span>Diagnostics &amp; Maintenance</span> &gt; <span>Data Logging</span> &gt; <span>Settings</span>. <br />The Settings page appears. </li> <li> Alongside <span>Logging</span>, select the toggle to enable or disable the data log. </li> <li> Alongside <span>Log period</span>, type the frequency (in seconds) that you want to log data changes. </li> <li> Select <span>Save</span> to save your changes. </li> </ol>
