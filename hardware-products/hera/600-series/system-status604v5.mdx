<div> <p><span>Model:</span> Hera 604 <span>Version:</span> v5</p> </div>

<p>You can view the current system status settings for the Hera. These properties are read-only.</p>

<h2 id="Router">Router</h2>

<p>Describes general router information. </p>

<p> </p>

<p>You can edit some basic router settings here:</p>

<p><span>Basic Settings</span> &gt; <span>Router</span> &gt; <span>Name and location</span></p>

<table> <col /> <col /> <thead> <tr> <th>Property</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Model</span> </td> <td>The Hera Router series.</td> </tr> <tr> <td><span>Serial number</span> </td> <td>Unique identifier for the current Hera.</td> </tr> <tr> <td><span>Software version</span> </td> <td>The current software release version installed.</td> </tr> <tr> <td><span>Hostname</span> </td> <td>Assigned network hostname.</td> </tr> <tr> <td><span>Firmware version</span> </td> <td>The currently installed firmware release version.</td> </tr> <tr> <td><span>Kernel version</span> </td> <td>The currently installed kernel version.</td> </tr> <tr> <td><span>Local time</span> </td> <td>The current time, local to the Hera.</td> </tr> <tr> <td><span>Last restart</span> </td> <td>Date and time stamp of the last boot up session</td> </tr> <tr> <td><span>Uptime</span> </td> <td>Duration of the current Hera session.</td> </tr> <tr> <td><span>Load average</span> </td> <td>Router performance trends over the past minute, five minutes, and fifteen minutes.</td> </tr> <tr> <td><span>GPS</span> </td> <td>The current GPS location, if the device hardware supports this function, and if GPS is enabled.</td> </tr> <tr> <td><span>Site name</span> </td> <td>A user-defined setting to describe which Hera you are remotely logged into.</td> </tr> <tr> <td><span>Site location</span> </td> <td>A user-defined setting to describe where the current Hera exists.</td> </tr> <tr> <td><span>Notes</span> </td> <td>A user-defined setting containing any notes about the current Hera. For example, the file name of the factory configuration file stored in subversion.</td> </tr> </tbody> </table>

<div> <h2 id="LocalareanetworkLAN">Local area network (LAN)</h2> <p>Describes the current network settings. You can edit the LAN settings here:</p> <p><span>Basic Settings</span> &gt; <span>Local Area Network (LAN)</span></p> <table> <col /> <col /> <thead> <tr> <th>Property</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>LAN IP address</span> </td> <td>The assigned Local Area Network IPv4 address.</td> </tr> <tr> <td><span>Net mask</span> </td> <td>The IPv4 subnet mask.</td> </tr> <tr> <td><span>Address assignment</span> </td> <td> How the address was assigned:  <ul> <li><span>STATIC</span> – manual configuration</li> <li><span>DHCP</span> – automatic configuration from a DHCP server on the LAN</li> </ul> </td> </tr> </tbody> </table> </div>

<h2 id="Mobilenetwork">Mobile network</h2>

<p>Describes the current cellular network connection. You can edit Mobile Network settings here:</p>

<p><span>Basic Settings</span> &gt; <span>Mobile Network</span></p>

<table> <col /> <col /> <thead> <tr> <th>Property</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Mobile connection</span> </td> <td> Select the slider to <span>Enable</span> or <span>Disable</span> the cellular connection. </td> </tr> <tr> <td><span>IMEI</span> </td> <td> International Mobile Equipment Identity. The unique 15-digit International Mobile Equipment Identity serial number that identifies the device on the cellular network. </td> </tr> <tr> <td><span>Active SIM</span> </td> <td>Which SIM is currently in use. The Hera can have up to three SIMs installed. For more information, see<a href="/hardware-products/hera/600-series/basic-settings604v5#Configur3" target="_blank">Configuring the Hera to use a SIM</a>.</td> </tr> <tr> <td><span>IMSI of active SIM</span> </td> <td>The current<a href="/glossary/imsi">IMSI</a> in use on the connected SIM. For information about multi-IMSI SIMs, see <a href="/getting-started/multi-imsi">Understanding multi-IMSI functionality</a>.</td> </tr> <tr> <td><span>ICCID of active SIM</span> </td> <td>The current <a href="/glossary/imsi">ICCID</a> in use on the connected SIM. For more information, see<a href="/getting-started/esim/euicc#IccidAndImsi" target="_blank">ICCID and IMSI</a>.</td> </tr> <tr> <td><span>Mobile network provider</span> </td> <td> The network operator for the network where the current SIM is subscribed.</td> </tr> <tr> <td><span>Mobile network IP address</span> </td> <td>The Hera's current network IPv4 address.</td> </tr> <tr> <td><span>Mobile network signal</span> </td> <td> Mouse over the signal strength indicator to see the current strength in decibels. Signal strength ranges from approximately -30 dBm (very strong) to -110 dBm (very weak). </td> </tr> <tr> <td><span>APN</span> </td> <td>The current Access Point Name used to connect the Hera to the network. For more information, see<a href="/connectivity/access-point-name" target="_blank">About Access Point Names (APNs)</a></td> </tr> </tbody> </table>

<h2 id="Connecteddevices">Connected devices</h2>

<p>Lists any devices that are locally connected via a TCP/IP network to the Hera. You can view the following attributes:</p>

<table> <col /> <col /> <thead> <tr> <th>Property</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>IP address</span> </td> <td>The assigned IPv4 address for the connected device.</td> </tr> <tr> <td><span>MAC address</span> </td> <td>The connection Media Access Control address.</td> </tr> <tr> <td><span>Device</span> </td> <td>The internal device name through which the connection is provided.</td> </tr> </tbody> </table>
