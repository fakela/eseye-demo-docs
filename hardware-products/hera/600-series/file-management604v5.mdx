<h2 id="Creatingabackupconfigurationfile">Creating a backup configuration file</h2>

<p>You can backup the current configuration used by the router.</p>

<p>To create a backup:</p>

<ol> <li> Using the Hera user interface, navigate to <span>Diagnostics &amp; Maintenance</span> &gt; <span>File Management</span> &gt; <span>Configuration backup</span>. <br />The Configuration backup page appears. </li> <li> Select <span>Backup</span>. <br />The backup-Hera600-&lt;<var>DownloadDate</var>&gt;.tar.gz file downloads to the default Downloads folder, where &lt;<var>DownloadDate</var>&gt; is the current date. </li> </ol>

<h2 id="Uploadinganewconfiguration">Uploading a new configuration</h2>

<p>You can upload a configuration or backup copy of the configuration.</p>

<p>To upload a configuration:</p>

<ol> <li> Using the Hera user interface, navigate to <span>Diagnostics &amp; Maintenance</span> &gt; <span>File Management</span> &gt; <span>Configuration upload</span>. <br />The Configuration upload page appears. </li> <li> Select <span>Upload</span>. <br />The Configuration upload dialog box appears. </li> <li> Select <span>Choose file</span> to select a &#42;.tar.gz configuration file to upload. <br />The Open window appears. </li> <li> Select the relevant file, then select <span>Open</span> to update the dialog box with the file. </li> <li> Select <span>OK</span> to upload and restore the configuration file. <br />A confirmation box appears. </li> <li> Reboot the router to apply the changes. </li> </ol>

<h2 id="UploadinganewHerasoftwareversion">Uploading a new Hera software version</h2>

<p>You can upload a new version of the Hera interface software.</p>

<p>To upload Hera interface software:</p>

<ol> <li> Using the Hera user interface, navigate to <span>Diagnostics &amp; Maintenance</span> &gt; <span>File Management</span> &gt; <span>Software upload</span>. <br />The Software upload page appears. </li> <li> If you want to keep your existing configuration, select <span>Upload – keeping existing configuration</span>. </li> <li> If you want to use the supplied default configuration, select <span>Upload</span>. </li> <li> In the relevant <span>Software Upload</span> box, select <span>Choose file</span> to select the supplied.tar.gz software file to upload. <br />The Open window appears. </li> <li> Select the relevant file, then select <span>Open</span> to update the dialog box with the file. </li> <li> Select <span>OK</span> to upload the software file. <br />A confirmation box appears. </li> <li> Reboot the router to apply the changes. </li> </ol>
