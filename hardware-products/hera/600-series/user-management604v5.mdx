<div> <p><span>Model:</span> Hera 604 <span>Version:</span> v5</p> </div>

<p>Configure the user settings here:</p>

<p><span>Diagnostics &amp; Maintenance</span> &gt; <span>User Management</span></p>

<h2 id="Changingtheadministratorpassword">Changing the administrator password</h2>

<p>To change the currently logged in administrator's password:</p>

<ol> <li> Using the Hera user interface, navigate to <span>Diagnostics &amp; Maintenance</span> &gt; <span>User Management</span> &gt; <span>Administrator password</span>. </li> <li> In the <span>New password</span> box, type the new password. <br />Valid passwords are 4 to 64 digits long. </li> <li>In the <span>Confirm new password</span> box, retype the new password.</li> <li>At the top right, select <span>Save</span>.</li> </ol>
