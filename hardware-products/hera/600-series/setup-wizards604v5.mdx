<div> <p><span>Model:</span> Hera 604 <span>Version:</span> v5</p> </div>

<p>The <span>Network connection Setup Wizard</span> helps you to configure your network settings in one place.</p>

<p>You can access the wizard here: <span>Setup Wizards</span> &gt; <span>Network connection</span></p>

<p>Use the wizard to configure any of the following connection types.</p>

<h2 id="Configuringacellularconnection">Configuring a cellular connection</h2>

<Accordion title="To set up a cellular connection using preconfigured settings:">
 <ol> <li>Select <span>Cellular connection</span>, then select <span>OK</span>.</li> <li> On the <span>Set up the connection</span> page, select <span>Keep the existing user profiles</span> to keep the listed preconfigured SIM profiles. <br />Eseye preconfigures your SIM profiles for you, unless you request otherwise. </li> <li> Select <span>OK</span>. </li> <li> Configure the Health Monitor, selecting <span>OK</span> when you have completed each page. <span><strong>Note: </strong>All Health Monitor features ensure connectivity is maintained where possible.</span> <span><strong>Note: the Health Monitor</a>, alternatively select the relevant links below to view information for a specific wizard page: <table> <col /> <col /> <thead> <tr> <th>Wizard page</th> <th>Health Monitor section</th> </tr> </thead> <tbody> <tr> <td><span>Set up health monitoring – 1 of 3</span> </td> <td> <a href="/hardware-products/hera/600-series/health-monitor604v5#General" target="HM">Configuring
