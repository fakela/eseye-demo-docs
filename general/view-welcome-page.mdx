<p>You may need to return to the AnyNet IRIS Welcome page to view your Customer ID, or to complete downloading the AnyNet IRIS executable.</p>

<p>To find the Welcome page:</p>

<ol> <li>Navigate to AWS Marketplace: <a href="https://aws.amazon.com/marketplace">https://aws.amazon.com/marketplace</a>.</li> <li>Sign into the account you used to subscribe to <span>AnyNet Cellular Connectivity for AWS IoT</span>.</li> <li>In the top right corner, select your logged-in identity.</li> <li>Select <span>Your Marketplace Software</span> from the drop-down menu.</li> <li>Select the <span>AnyNet Cellular Connectivity for AWS IoT</span> subscription.</li> <li>Select the <span>Read more on AWS Marketplace</span> link.</li> <li>Select <span>Continue to Subscribe</span>.</li> <li> In the <span>Having issues signing up for your product?</span> box, select <span>click here</span> to view the Welcome page. <br />Make a note of the CustomerID to help you configure AnyNet IRIS.  </li> <li> If required, select the relevant download. <br />For more information, see <a href="/pls62-w/aws-integration/pls62-w-getting-started">Getting Started: preparing the cloud</a><a href="/els61/aws-integration/els61-getting-started">Getting Started: preparing the cloud</a><i>Getting Started: preparing the cloud </i> on page 1. </li> </ol>
