---

<p>Before you can send and receive data between your thing and the cloud, you need to: </p>

<ol> <li> Prepare the cloud.  <br />For more information, see<a href="/iris/getting-started">Getting Started: preparing the cloud</a>. </li> <li> Install AnyNet IRIS. <br />For more information, see<a href="/iris/installing-iris">Installing AnyNet IRIS</a>. </li> <li> Create your thing in AWS IoT Console.  <br />For more information, see <a href="/iris/creating-thing-in-aws#_Toc25599552">Creating a thing in AWS IoT Core</a>. </li> <li>  Connect the Cinterion® PLS62‑W Wireless Module to a cellular network.  <br />For more information, see <a href="/pls62-w/establishing-cellular-connection">Establishing a cellular connection on the Intelligent Cloud Connect</a> . </li> <li> Provision the Cinterion® PLS62‑W Wireless Module. <br />For more information, see <a href="/pls62-w/provisioning#Provisio">Where to next?</a> . </li> <li> Use a terminal emulator to send commands to the Cinterion® PLS62‑W Wireless Module.  <br />For information about setting up the terminal emulator, see <a href="/pls62-w/rs232-serial-connection">Attaching to the Intelligent Cloud Connect</a>. </li> <li> Test that you can send data from your thing to the cloud.  <br />For more information, see <a href="/pls62-w/publish-data-to-cloud">Sending data from your thing to the cloud</a> . </li> <li> Test that you can send data from the cloud to your thing. <br />For more information, see <a href="/pls62-w/subscribe-data-from-cloud">Sending data from the cloud to your thing</a> . </li> <li> Your system is now ready. Use AT commands to configure how data is sent and received between the cloud and the modem. <br />For more information, see <a href="/pls62-w/at-commands/at-commands">General AT Commands for the Cinterion® PLS62‑W Wireless Module </a> . </li> </ol>
