---

<p>Displays information about the current ETM application and device. </p>

<table> <col /> <col /> <col /> <thead> <tr> <th>Type</th> <th>Syntax</th> <th>Response</th> </tr> </thead> <tbody> <tr> <td>Test</td> <td> <span>AT+ETMINFO=?</span>  </td> <td>  +ETMINFO:(ci,iccid,imei,imsi,mcc,mnc,<br />rssi,service,version)  </td> </tr> <tr> <td>Read</td> <td> <span>AT+ETMINFO?</span>  </td> <td> OK <br />or <br />ERROR </td> </tr> <tr> <td> Write </td> <td> <span>AT+ETMINFO=</span><span>&lt;<var>id</var>&gt;</span>  <br />where <span>&lt;<var>id</var>&gt;</span> is either: <ul> <li><span>ci</span> – the unique cell identity</li> <li><span>iccid</span> – the SIM unique identifier</li> <li><span>imei</span> –the device unique identifier</li> <li><span>imsi</span> – the cellular network user unique identifier</li> <li><span>mcc</span> – the mobile country code</li> <li><span>mnc</span> – the mobile network code</li> <li><span>rssi</span> – the received signal strength indicator</li> <li><span>service</span> – shows whether the modem is registered to a network</li> <li><span>version</span> – the current version of Eseye software on the selected modem</li> </ul> </td> <td> +ETMINFO: &lt;value&gt; <br />  <br />OK <br />where <span>value</span> is the requested value, such as the SIM ICCID. <br />For <span>service</span>, returns the registration status in decimal format:  <ul> <li><span>0</span> – not registered</li> <li><span>1</span> – registered</li> </ul> <br />or <br /><span>ERROR</span> – the command failed </td> </tr> </tbody> </table>

<h4>Example</h4>

<p>AT+ETMINFO="iccid"</p>

<p>8944538523020412345</p>

<p>OK</p>
