<p>Checks the current state of the Intelligent Cloud Connect.</p>

<table> <col /> <col /> <col /> <thead> <tr> <th>Type</th> <th>Syntax</th> <th>Response</th> </tr> </thead> <tbody> <tr> <td>Test</td> <td> <span>AT+ETMSTATE=?</span>  </td> <td> OK <br />  <br />+ETMSTATE: 0 disables URCs, 1 enables URCs </td> </tr> <tr> <td>Read</td> <td> <span>AT+ETMSTATE?</span>  </td> <td> +ETMSTATE: <var>&lt;State&gt;</var> <br /> <br />  <br />OK <br />or <br />ERROR </td> </tr> <tr> <td> Write </td> <td> <span>AT+ETMSTATE=&lt;<var>cmd</var>&gt;</span>  <br />where <span>&lt;<var>cmd</var>&gt;</span> is the command, either: <br /><span>0</span> – disable state change URC <br /><span>1</span> – enable state change URC </td> <td> AT+ETMSTATE=&lt;CurrentState&gt; <br />  <br />OK </td> </tr> </tbody> </table>

<p>AT+ETMSTATE?</p>
