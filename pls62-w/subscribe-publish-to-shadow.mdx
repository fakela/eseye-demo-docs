<h2>Interacting with the AWS IoT shadow</h2>

<p>You can configure the Cinterion® PLS62‑W Wireless Module to provide a persistent representation of your thing in the cloud, for use by applications or devices. You can publish current and updated <span>state</span> information to a shadow, and your thing can synchronize its state when it connects. </p>

<p>For information about enabling shadow use, see <a href="/pls62-w/at-commands/emqpersist">EMQPERSIST – report a set value to the device shadow</a>.</p>

<p>For detailed AWS information about using shadows, see <a href="https://docs.aws.amazon.com/iot/latest/developerguide/using-device-shadows.html">Simulating Device Shadow service communications</a>.</p>
