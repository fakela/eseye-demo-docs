title: "Integrating with AWS Marketplace using AnyNet IRIS"
description: "AnyNet IRIS enables you to easily and securely deploy required IAM roles and policies to selected AWS regions for integration with AWS Marketplace."
---

<p>AnyNet IRIS enables you to easily and securely deploy required IAM roles and policies to selected AWS regions for integration with AWS Marketplace.</p>

<Note> <p>AnyNet IRIS is supported on the following operating systems:</p> <ul> <li><span>Windows 10</span> version 1903 and above</li> <li><span>macOS 10.14</span> and above</li> </ul> </Note>

<Tip>For information about the IAM managed policies you need to use the <span>AnyNet Cellular Connectivity for AWS IoT SaaS</span> product, see Required IAM Managed Policies.</Tip>

<Tip>For information about the IAM permissions you need to use the <span>AnyNet Cellular Connectivity for AWS IoT SaaS</span> product, see Required IAM permissions.</Tip>

<p>After deploying resources, you can use AnyNet IRIS to review the status of your thing s in each region in a specified time period, view the deployed Cloud Formation template contents, as well as manage configuration and updates.</p>

<Tip>For more information, see <a href="/iris/review-overview#Deployed" target="_blank">Reviewing thing connectivity using AnyNet IRIS</a>.</Tip>
