<p>Eseye may apply SMS charges for:</p>

<ul> <li> Cellular network usage</li> <li>SMS API usage (subscription only)</li> </ul>

<p>The amount charged depends on your SIM contract and (if relevant) SMS API contract.</p>

<h2 id="SMSmessagechargesontheSIMcontract">SMS message charges on the SIM contract</h2>

<p>Depending on your SIM contract, charges may apply for AnyNet SIM cellular network usage, which includes SMS messages and delivery receipts. </p>

<p>The SIM contract may have an agreed number of in-bundle SMS messages. You are not charged extra for these unless you use more than the agreed amount. For information about bundles, see <a href="/billing/billing-bundled-usage">Understanding bundled usage</a>.</p>

<p>On the CSV invoice, the SMS charges for the SIM contract are shown against the relevant SIM number. </p>

<h2 id="SMSmessagechargesontheSMSAPIcontract">SMS message charges on the SMS API contract</h2>

<p> If you subscribe to the SMS API service, the usage charges are agreed in the SMS API contract. </p>

<Note>Your SMS API contract terms may differ from those on your SIM contract. </Note>

<p>SMS API charges include:</p>

<ul> <li> Incoming internet use of the AnyNet Messaging Service from the customer server. </li> <li>Sending or receiving SMS messages between the AnyNet Messaging Service and third party SIMs.</li> </ul>

<p>On the CSV invoice, the SMS API contract charges are totalled per package according to transmission type. </p>
