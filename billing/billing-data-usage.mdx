<p>Use the CSV Invoice to understand how much data each SIM uses.</p>

<p>To view SIM data usage in the CSV Invoice:</p>

<ol> <li> Select the row containing the <span>ItemRef</span><span>Data</span> (Column A). </li> <li> Navigate to <span>Description</span> (column H) to view a description of Data. </li> <li> Total data usage in bytes = <span>Quantity</span> + <span>InBundleQuantity</span>, where: <br /><span>Quantity</span> (column K) is the number of excess (overage) bytes used <br /><span>InBundleQuantity</span> (column S) is the number of bytes used inside the bundled amount </li> <li> For kilobytes, divide the total by 1024. </li> <li> For megabytes, divide the total by 1048576. </li> </ol>
