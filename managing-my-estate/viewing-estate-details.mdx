<p>Use the <span>SIMs and Devices</span> page to:</p>

<ul> <li> Find and display information about SIMs, IoT devices, packages, packages associated with SIMs and portfolios. </li> <li>Save searches for future use, and save the results as CO and GR.</li> </ul>

<p>To access the Infinity SIMs and Devices:</p>

<ol> <li> On the left menu, select <span>Connect &amp; Manage</span>. <br /> <img src="/images/resources/images/infinity/connectandmanageicon_80x78.png" alt="Connectandmanageicon 80X78" />  </li> <li> Select <span>SIM and Device Search</span> to view the list of SIMs and devices.  <br /><img src="/images/resources/images/infinity/simanddevicesearch.png" alt="Simanddevicesearch" />  </li> </ol>

<h3>Filtering the displayed SIM details list</h3>

<p>The <span>Search results</span> list displays the search results of the search performed using the <span>search filters</span>. Scroll down to display more results. You can filter SIMs and devices using both quick filters and search filters.</p>

<Note>The number of results displayed depends on the screen size. </Note>

<p><img src="/images/resources/images/infinity/resultstable.png" alt="Resultstable" /> </p>

<p>To sort the search results view by column name:</p>

<ul> <li>Select a column name to sort the results in ascending (a-z; 1-9) or descending (z-a; 9-1) order.</li> </ul>

<h4>Applying quick filters</h4>

<p><span>QUICK FILTERS</span> is the filter engine to facilitate searching using the default parameters such as <span>Active Packages</span>, <span>Active SIMs</span>, <span>All Devices</span> and <span>This Portfolio</span>.</p>

<p><img src="/images/resources/images/infinity/quickfilters.png" alt="Quickfilters" /> </p>

<p>To apply a quick filter:</p>

<p> Select a value for one of the following fields:</p>

<ul> <li>Under <span>Devices</span>, select the <a href="#Device">Device Status</a> to use as a search filter.</li> <li>Under <span>SIMs</span>, select the <a href="#SIM%C2%A0Stat">SIM Status</a> to use as a search filter.</li> <li>Under <span>Packages</span>, select the <a href="#Package">Package Status</a> to use as a search filter.</li> <li>Under <span>Portfolios</span>, select to filter packages and SIMs for:<ul><li><span>All portfolios</span>: Lists <span>Home</span> and <span>Child</span> portfolios.</li><li><span>Child portfolios</span>: Lists the particular site/department/territory of the Home country or direct descendants of the Home portfolios.</li><li><span>This portfolio</span>: Identifies the portfolio that you want to view.</li></ul></li> </ul>

<Note>Not all values for the SIM status and package status parameters are available as quick filters.</Note>

<p>Changing the value in the quick filter applies the value to the relevant field on the search filter tab. Changing the value on the search filters tab also updates the relevant quick filter.</p>

<Tip>Each time you refresh the page, the quick filter displays the default view.</Tip>

<h4>Applying search filters</h4>

<p>Search filters refine search results to match your preferences using selected parameters.</p>

<p>To find the specific package/ SIM/ device/ portfolio/ collection:</p>

<ol> <li>Select <span>Connect and Manage</span> &gt; <span>SIM and Device Search</span>.</li> <li>Filter by the respective:</li> </ol>

<ul> <li><a href="#Package2">Package columns</a> </li> <li><a href="#SIM2">SIM columns</a> </li> <li><a href="/managing-my-estate/manage-tagging-sims#To" target="_blank">SIM group</a> </li> <li><a href="#Device2">Device columns</a> </li> <li><a href="#Portfoli">Portfolio columns</a> </li> <li><a href="#Collecti">Finding your SIMs and devices</a> </li> <li>Saved searches</li> </ul>

<p>Select for the filtered search results to display in the <span>Search results</span> list.</p>

<p><img src="/images/resources/images/infinity/searchfilterssim.png" alt="Searchfilterssim" /> </p>

<p>The <span>Search results</span> list displays information regarding the selected parameters in <span>SIM details</span> rows.</p>

<Note>The numeric value on the filter tab displays the number of applied search filters on that tab.</Note>

<h2 id="Savingactivesearches">Saving active searches</h2>

<p>The tabs in the search filters and actions pane enable you to filter the search results by device, SIM, SIM group, package, portfolio and SIM collection information.</p>

<p><img src="/images/resources/images/infinity/searchfilters.png" alt="Searchfilters" /> </p>

<p>To save the active search:</p>

<ol> <li>In the <span>Search results</span> list, select <img src="/images/resources/images/infinity/iconsavedsearches_24x24.png" alt="Iconsavedsearches 24X24" /> to display the <span>Saved Searches</span> dialog.</li> <li> Under <span>Save the current search</span>, type an unique name for the saved search. <br /><img src="/images/resources/images/infinity/savecurrentsearch.png" alt="Savecurrentsearch" />  </li> <li> Select <span>Save</span> to display the name in the <span>Search results</span> list. </li> </ol>

<p>To load a previously saved search:</p>

<ol> <li>In the <span>Search results</span> list, select <img src="/images/resources/images/infinity/iconsavedsearches_24x24.png" alt="Iconsavedsearches 24X24" /> to display the <span>Saved Searches</span> dialog.</li> <li> Under <span>Load a previously saved search</span>, drop-down list to select the name you had provided for a previously saved search view. <br /><img src="/images/resources/images/infinity/savecurrentsearch.png" alt="Savecurrentsearch" />  </li> <li> Select <span>Load</span> for your previous search view to display in the <span>Search results</span> list. </li> </ol>

<h2 id="ViewingSIMmessaginginformation">Viewing SIM messaging information</h2>

<p>To display SIM messaging information:</p>

<ol> <li> In the <span>Search results</span> list, select the checkbox alongside the specific <span>SIM details</span> row. </li> <li> At the top right, select <span>Actions</span> &gt; <span>SIM Info</span> &gt; <span>Messaging</span> to display the <span>SIM Messaging</span> dialog.  <br /><img src="/images/resources/images/infinity/simmessagingdialog.png" alt="Simmessagingdialog" />  </li> <li> Specify the below parameters when filtering the messaging information: </li> </ol>

<ul> <li> <span>Start date</span> and <span>End date</span> – displays the messaging information within the chosen time range. </li> <li> <span>Message type</span> – filters messages on SMS direction: <span>SMS MO</span> (mobile originated), <span>SMS MT</span> (mobile terminated) or Select all (for both directions). </li> <li>Select <img src="/images/resources/images/infinity/search-icon_24x24.png" alt="Search Icon 24X24" /> to list the SMS messages sent or received by the selected SIM.</li> </ul>

<h2 id="Linkingdevices">Linking devices</h2>

<p>To link IMEI with device model:</p>

<ol> <li> In the <span>Search results</span> list, select the checkbox alongside the specific <span>SIM details</span> row. </li> <li> At the top right, select <span>Actions</span> &gt; <span>Device Settings</span> to display the <span>Device Settings</span> dialog.  <br /><img src="/images/resources/images/infinity/devicesettingsdialog.png" alt="Devicesettingsdialog" />  </li> <li> Type new inputs for the <span>IMEI</span> and choose from the <span>Model</span> drop-down list. For more information, see<a href="/managing-my-estate/about-lists-and-searches#Managing" target="_blank">Viewing device types</a>.  <span><strong>Tip: </strong>You can enter IMEI for a new device also known as <a href="/managing-my-estate/viewing-sim-details#Custom" target="_blank">Custom IMEI</a>.</span> </li> <li> Select <span>Save</span> to update the values for all selected SIMs. </li> </ol>

<p> </p>

<p>To reset the active search:</p>

<ol> <li> In the search filters pane, select <span>Actions</span>. </li> <li> Select <span>Reset Active Search</span> to remove all values from the search filter parameters on every tab. </li> </ol>

<Note>The reset action, clears all the filter values except for the <a href="#SIM%C2%A0Stat">SIM Status</a> filter (value resets to <span>Available</span>) and the <a href="#Package">Package Status</a> (value resets to <span>Active</span>).</Note>

<h2 id="Managingsearchviews">Managing search views</h2>

<p>To save, load and modify the search result view:</p>

<ol> <li> In the <span>Search results</span> list, select <img src="/images/resources/images/infinity/filter-icon_24x24.png" alt="Filter Icon 24X24" /> to display the <span>Search Results View</span> dialog that displays the column names. <br /><img src="/images/resources/images/infinity/showcolumns.png" alt="Showcolumns" />  </li> <li> Perform one of the following: <br />To save a search view: <ul> <li> Under <span>Save the current selection</span>, either select an existing saved result view to overwrite, or type an unique name for a new custom result view. </li> <li> Select <span>Save</span> for the specific output columns to display in the <span>Search results</span> view. </li> </ul> <br />To load an existing search view: <ul> <li>Use the <span>Load a saved set of columns</span> list to select a previously saved result view.</li> <li>By default, the interface displays all the column names for <span>Device</span>, <span>SIM</span>, <span>Package</span> and <span>Portfolio</span>.</li> <li>Select <span>Load</span> to update the selected output columns.</li> </ul> <br />To modify the current search view: <ul> <li> Under <span>Output Columns</span>, select/ clear the check boxes next to the column names you wish to display/ hide. </li> </ul> </li> <li> Select <span>OK</span> for the changes to reflect in the <span>Search results</span> list. </li> </ol>
