title: "Managing email notifications"
description: "You can receive emails when alerts are triggered, and when the latest SIM usage report and SIM admin report become available."
---

<p>You can receive emails when alerts are triggered, and when the latest <span>SIM usage</span> report and <span>SIM admin</span> report become available.</p>

<h2 id="Enablingemailnotifications">Enabling email notifications</h2>

<p>To receive email notifications:</p>

<ol> <li>Sign in to the <span>Infinity</span> web interface.</li> <li>At the top-right, select your <span>username</span> &gt; <span>Notifications</span> to display the <span>Notifications</span><br />dialog box.</li> <li> Under <span>Alerts</span>, select the <span>Alert Triggered</span> checkbox to receive an email every time the system generates an alert. </li> <li> Under <span>Reports</span>, select the <span>New SIM Usage Report</span> checkbox to receive an email when the latest report becomes available.  <br />This report tracks the usage cost per SIM.  <span><strong>Tip: </strong>You can set how often the report generates. For more information, see <a href="/analytics/reports#Device" target="_blank">Managing Infinity reports</a>.</span> <br /><img src="/images/resources/images/infinity/enable-notifications.png" alt="Enable Notifications" />  <br />  </li> <li> Select <span>Save</span> to commit your selection. </li> </ol>

<h2 id="Disablingemailnotifications">Disabling email notifications</h2>

<p>To disable email notifications:</p>

<ol> <li>Sign in to the <span>Infinity</span> web interface.</li> <li>At the top-right, select your <span>username</span> &gt; <span>Notifications</span> to display the <span>Notifications</span><br />dialog box.</li> <li> Under <span>Alerts</span>, clear the <span>Alert Triggered</span> checkbox to stop receiving an email every time the system generates an alert. </li> <li> Under <span>Reports</span>, clear the <span>New SIM Usage Report</span> checkbox to stop receiving an email when the latest report becomes available.  </li> <li> Select <span>Save</span> to commit your selection. </li> </ol>
