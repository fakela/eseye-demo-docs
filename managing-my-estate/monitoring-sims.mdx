<p>Infinity has an alerting system that is designed to keep users informed about critical changes and SIM usage patterns, through a combination of alert profiles, alerts, and notifications.</p>

<ul> <li> <span>Alert Profiles</span> are customisable sets of thresholds that determine when an alert triggers. These thresholds are based on various criteria, such as financial limits, data consumption levels, the number of SMS messages sent, maximum call usage, or when the SIM changes device. By setting these thresholds, users can closely monitor their SIMs and ensure that usage stays within expected limits. <br />For more information, see <a href="/managing-my-estate/alertprofile">About Infinity alert profiles</a>. </li> <li> <span>Alerts</span> are in-platform warnings that users receive when a defined alert profile threshold is breached. Alerts provide immediate feedback and details about the exceeded threshold. This real-time information allows users to respond quickly to potential issues or overages. <br />For more information, see <a href="/managing-my-estate/alerts">About Infinity alerts</a> </li> <li> <span>Notifications</span> complement alerts, ensuring interested parties also receive an email when a threshold is breached. This ensures that users remain informed even when they are not logged into Infinity, providing an additional layer of awareness and the ability to take action swiftly. <br />For more information, see <a href="/infinity/manage-user-profile#Notifica" target="_blank">Managing your Infinity user profile</a> </li> </ul>

<p>Together, these features help users proactively manage their SIMs and ensure that usage stays within acceptable limits, minimizing unexpected costs and optimising resource allocation.</p>
