<p>Collections organise SIMs based on specific criteria or attributes, either as static (manually updated) or dynamic (automatically updated) groups.</p>

<p>Use SIM collections to:</p>

<ul> <li> Generate reports or monitor a group of SIMs.  <br />For example, high-usage or country-specific SIMs. </li> <li> Apply bulk actions to a group of SIMs, saving time.  <br />For example, activate, deactivate, or update multiple SIMs.  </li> <li> Track and manage a group of SIMs by lifecycle stage.  <br />For example, newly activated or suspended SIMs. </li> </ul>

<h2 id="ViewingSIMcollections">Viewing SIM collections</h2>

<p>To view your SIM collections:</p>

<ol> <li> On the left menu, select <span>My lists &amp; searches</span>. <br /> <img src="/images/resources/images/infinity/mylistsicon2_80x78.png" alt="Mylistsicon2 80X78" />  </li> <li> The <span>My Lists and Searches</span> page appears. <br /><img src="/images/resources/images/infinity/mylists&searches.png" alt="Mylists&Searches" />  </li> <li> Select <span>SIM Collections</span>. <br /><img src="/images/resources/images/infinity/simcollections.png" alt="Simcollections" />  <br />The <span>SIM Collections</span> list contains the following fields: <table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Name</span> </td> <td> The SIM collection name. </td> </tr> <tr> <td><span>Portfolio</span> </td> <td> Displays the following: <br /><span>Portfolio Name</span> – A friendly name for identifying the portfolio, usually the company name followed by a geographic region or project. <br /><span>Portfolio ID</span> – The unique portfolio identifier. The portfolio is the overall customer account that contains user, package, network and SIM information, and includes details such as linked addresses, contact phone numbers, tax information and billing currency information. </td> </tr> <tr> <td><span>SIMs</span> </td> <td>The total number of SIMs in the SIM collection.</td> </tr> <tr> <td><span>Last Updated By</span> </td> <td>The username of the user who most recently made changes to the SIM collection.</td> </tr> <tr> <td><span>Updated Date</span> </td> <td>The date and timestamp when the SIM collection was most recently changed.</td> </tr> </tbody> </table> </li> </ol>

<h2 id="UpdatingaSIMcollection">Updating a SIM collection</h2>

<p>You can change the SIM collection name.</p>

<p>To update the SIM collection:</p>

<ol> <li>On the <span>My lists and searches</span> page, on the <span>SIM collections</span> tab, select <img src="/images/resources/images/infinity/iconedit_24x24.png" alt="Iconedit 24X24" />alongside the collection you want to update.</li> <li>Type a new name for the SIM collection.</li> <li>Select <span>Save</span> to commit your change.</li> </ol>

<h2 id="DeletingaSIMcollection">Deleting a SIM collection</h2>

<p>To delete the SIM collection:</p>

<ol> <li> On the <span>My lists and searches</span> page, on the <span>SIM collections</span> tab, select <img src="/images/resources/images/infinity/icondelete_24x24.png" alt="Icondelete 24X24" />alongside the collection you want to delete. <br />A confirmation box appears. </li> <li>Select <span>Confirm</span> to remove the SIM collection.</li> </ol>

<h2 id="ViewingSIMsbySIMcollection">Viewing SIMs by SIM collection</h2>

<p>To view SIMs by SIM collection:</p>

<ol> <li> On the left menu, select <span>Connect &amp; Manage</span>. <br /> <img src="/images/resources/images/infinity/connectandmanageicon_80x78.png" alt="Connectandmanageicon 80X78" />  </li> <li> Select <span>SIM and Device Search</span> to view the search lists. <br /><img src="/images/resources/images/infinity/simanddevicesearch.png" alt="Simanddevicesearch" />  </li> <li> On the <span>SIM and Device Search</span> page, select the <span>Collections</span> tab to view SIMs by SIM collection. <br />The <span>SIMs by SIM Collections</span> list contains the following fields: <table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Group</span> </td> <td> The tag or label for a set of SIMs, to help identify and categorise the SIMs. </td> </tr> <tr> <td><span>ICCID</span> </td> <td> The unique Integrated Circuit Card Identifier for the SIM or eUICC SIM profile.  </td> </tr> <tr> <td><span>Last IMSI</span> </td> <td> The most recent international mobile subscriber identity number (<a href="/glossary/imsi">IMSI</a>) used to authenticate the SIM on the network. </td> </tr> <tr> <td><span>MCC</span> </td> <td> The unique three digit ISO 3166-1 numeric mobile country code, used to identify the country where the mobile network exists. </td> </tr> <tr> <td><span>MSISDN</span> </td> <td>  The Mobile Station International Subscriber Directory Number (MSISDN) is a unique identifier for a mobile phone number in telecommunications. It includes the country code, mobile network code and subscriber number, allowing for routing of calls and messages to the correct device. The MSISDN contains between 8 and 18 numeric digits and can include the '+' country code identifier, for example: +441234567890.  </td> </tr> <tr> <td><span>IP Address</span> </td> <td> One or more IPv4 addresses associated with the IMSI.  </td> </tr> <tr> <td><span>SIM ID</span> </td> <td> The unique Eseye SIM reference number, printed on the physical SIM card. </td> </tr> <tr> <td><span>SIM Name</span> </td> <td> A unique customer-defined user-friendly tag or label for the SIM, to help identify and organise the SIM. </td> </tr> <tr> <td> <span>SIM Status</span>  </td> <td> The current SIM status, either: <ul> <li><span>available</span> – the SIM or IMSI (for multi-IMSI SIMs) exists on a customer account and is assigned a package. No charges are incurred. The customer must request to activate the SIM for use.</li> <li><span>active</span> – the SIM or IMSI (for multi-IMSI SIMs) is fully active on the network and ready for use. The SIM or IMSI is charged at standard service charge rates, and for any usage.</li> </ul> <br />For more information, see <a href="/billing/billing-sim-lifecycle" target="_blank">Understanding the SIM lifecycle</a>. </td> </tr> </tbody> </table> </li> </ol>

<h3>Filtering the displayed <span>SIMs by SIM Collections</span> list</h3>

<p>To filter the displayed <span>SIMs by SIM collection</span> list:</p>

<ul> <li> On the <span>SIM and Device Search</span> page, on the <span>Collections</span> tab, use the <span>SIM Collection</span> drop-down list to select the relevant collection. </li> <li> <img src="/images/resources/images/infinity/simcollection.png" alt="Simcollection" />  <span><strong>Tip: </strong>To search all collections, select <span>No selected collection</span>.</span> <br />The list filters immediately. </li> </ul>

<h2 id="CreatingaSIMcollectionforagroupofSIMs">Creating a SIM collection for a group of SIMs</h2>

<p>You can assign one or more SIMs to a collection to easily access their details.</p>

<p> </p>

<p>To create a SIM collection for a group of SIMs:</p>

<ol> <li> On the left menu, select <span>Connect &amp; Manage</span>. <br /> <img src="/images/resources/images/infinity/connectandmanageicon_80x78.png" alt="Connectandmanageicon 80X78" />  </li> <li> Select <span>SIM and Device Search</span> to view the search lists. <br /><img src="/images/resources/images/infinity/simanddevicesearch.png" alt="Simanddevicesearch" />  </li> <li> On the <span>SIM and Device Search</span> page, select the <span>Collections</span> tab to view the SIMs by SIM collection. </li> <li> Select the checkbox alongside one or more displayed SIMs that you want to add to the new SIM collection. <br /><img src="/images/resources/images/infinity/selectsimcollection.png" alt="Selectsimcollection" />  </li> <li> Select <span>Actions</span> &gt; <span>Save Selected SIMs as a Collection</span> (top-right) to view the <span>Save a SIM Collection</span> dialog. </li> <li>In the <span>Collection Name</span> box, type a unique collection name.</li> <li> Alternatively, use an existing collection name from the drop-down list. <span><strong>Note: </strong>This action will replace the existing collection with the newly selected SIMs.</span> </li> <li>Select <span>Save</span> to apply the new collection to the selected SIMs.</li> </ol>

<h2 id="TaggingtheSIMsinacollection">Tagging the SIMs in a collection</h2>

<p>When you have created a collection, you can apply up to five user-defined tags, as well as a Group tag, to that collection, or to a partial collection.</p>

<p>To collectively tag SIMs in a collection:</p>

<ol> <li> On the <span>SIM and Device search</span> <span>Collections</span> tab, use the <span>SIM Collection</span> drop-down list to select the SIM collection you want to tag.  <br />For more information, see <a href="#Filterin">Filtering the displayed SIMs by SIM Collections list</a>. </li> <li> If you want to tag all the SIMs in the displayed collection, select the checkbox in the header row to select all SIMs. </li> <li> If you want to tag a partial collection, select one or more checkboxes alongside the relevant SIMs only. <br /><img src="/images/resources/images/infinity/group-collection.png" alt="Group Collection" />  </li> <li> Select <span>Actions</span> &gt; <span>SIM Settings</span>. </li> <li> If required, type in values for up to five <span>User fields</span>. </li> <li> If required, type a <span>Group</span> name. <br /><img src="/images/resources/images/infinity/selected collections.png" alt="Selected Collections" />  </li> <li> Select <span>Update</span> to collectively tag the selected SIMs. </li> </ol>
