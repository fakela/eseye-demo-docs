<p>You can tag SIMs with a group name, to help identify and categorise them. For example, you may want to categorise the SIMs according to geographic location, or product type, and so on.</p>

<p>To access SIM groups using SIM and Device Search:</p>

<ol> <li> On the left menu, select <span>Connect &amp; Manage</span>. <br /> <img src="/images/resources/images/infinity/connectandmanageicon_80x78.png" alt="Connectandmanageicon 80X78" />  </li> <li> Select <span>SIM and Device Search</span> to view the <span>Search results</span> list. <br />The SIM and Device search page appears. <br /><img src="/images/resources/images/infinity/simanddevicesearch1.png" alt="Simanddevicesearch1" />  </li> </ol>

<h2 id="ViewingSIMGroups">Viewing SIM Groups</h2>

<p>To view the Infinity SIM Groups list:</p>

<ul> <li> On the <span>SIM and Device Search</span> page, select the <span>SIM Group</span> tab to view SIMs by group. <br /><img src="/images/resources/images/infinity/simanddevicesearch.png" alt="Simanddevicesearch" />  <br />The <span>SIM Group</span> search results list contains the following fields: <table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Group</span> </td> <td> The tag or label for a set of SIMs, to help identify and categorise the SIMs. </td> </tr> <tr> <td><span>ICCID</span> </td> <td> The unique Integrated Circuit Card Identifier for the SIM or eUICC SIM profile.  </td> </tr> <tr> <td><span>Last IMSI</span> </td> <td> The most recent international mobile subscriber identity number (<a href="/glossary/imsi">IMSI</a>) used to authenticate the SIM on the network. </td> </tr> <tr> <td><span>MCC</span> </td> <td> The unique three digit ISO 3166-1 numeric mobile country code, used to identify the country where the mobile network exists. </td> </tr> <tr> <td><span>MSISDN</span> </td> <td>  The Mobile Station International Subscriber Directory Number (MSISDN) is a unique identifier for a mobile phone number in telecommunications. It includes the country code, mobile network code and subscriber number, allowing for routing of calls and messages to the correct device. The MSISDN contains between 8 and 18 numeric digits and can include the '+' country code identifier, for example: +441234567890.  </td> </tr> <tr> <td><span>IP Address</span> </td> <td> One or more IPv4 addresses associated with the IMSI.  </td> </tr> <tr> <td><span>SIM ID</span> </td> <td> The unique Eseye SIM reference number, printed on the physical SIM card. </td> </tr> <tr> <td><span>SIM Name</span> </td> <td> A unique customer-defined user-friendly tag or label for the SIM, to help identify and organise the SIM. </td> </tr> <tr> <td> <span>SIM Status</span>  </td> <td> The current SIM status, either: <ul> <li><span>available</span> – the SIM or IMSI (for multi-IMSI SIMs) exists on a customer account and is assigned a package. No charges are incurred. The customer must request to activate the SIM for use.</li> <li><span>active</span> – the SIM or IMSI (for multi-IMSI SIMs) is fully active on the network and ready for use. The SIM or IMSI is charged at standard service charge rates, and for any usage.</li> </ul> <br />For more information, see <a href="/billing/billing-sim-lifecycle" target="_blank">Understanding the SIM lifecycle</a>. </td> </tr> </tbody> </table> </li> </ul>

<h3>Filtering the displayed SIM groups</h3>

<p>To filter the search results list by SIM group:</p>

<ul> <li> On the <span>SIM Group</span> tab, type the group name in the <span>Group</span> search filter. <br /><img src="/images/resources/images/infinity/group-sims.png" alt="Group Sims" />  <br />The list filters immediately. </li> </ul>

<h2 id="Creatinggroups">Creating groups</h2>

<p>To create a SIM group on a SIM:</p>

<ol> <li>On the <span>SIM and Device Search</span> page, select the checkbox alongside one or more displayed <span>SIM details</span> rows that you want to tag with a group name.</li> <li> At the top right, select <span>Actions</span> &gt; <span>SIM</span> <span>Actions</span> &gt; <span>SIM Settings</span> to display the <span>SIM Settings</span> dialog. <br /><img src="/images/resources/images/infinity/group-simsettings.png" alt="Group Simsettings" />  </li> <li> Enter values for each of the fields and type a <span>Group</span> name. <br /><img src="/images/resources/images/infinity/group-name-sims.png" alt="Group Name Sims" />  <span><strong>Note: </strong>Optionally, you can select from the <span>Alert Profile</span> drop-down list to assign this alert profile to this group of SIMs. For more information, see <a href="/managing-my-estate/alertprofile#Creating">Creating alert profiles</a>.</span> </li> <li> Select <span>Update</span> to apply the name to the selected SIMs. <span><strong>Note: </strong>Consequently, the SIM group is displayed in the <span>Search results</span> list. </span> </li> </ol>

<h2 id="UpdatingSIMgroups">Updating SIM groups</h2>

<p>You can change the group name on selected SIMs.</p>

<p>To update SIM groups:</p>

<ol> <li>On the <span>SIM Group</span> tab, filter the search list by the <span>Group</span> name.</li> <li>Select the checkbox alongside one or more SIMs that you want to update to a new group name.</li> <li> Select <span>Actions</span> &gt; <span>SIM Settings</span> (top-right) to display the <span>SIM Settings</span> dialog. <span><strong>Note: </strong>You can modify the appropriate field in the <span>Selected SIMs' Settings</span> dialog.</span> </li> <li> Type a new <span>Group</span> name.</li> <li>Select <span>Update</span> to commit the new group name to the selected SIMs.</li> </ol>
