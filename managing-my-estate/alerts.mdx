<p>Alerts display the SIMs where the usage has exceeded alert thresholds. These are generated by the alerting system and used to initiate orchestrations. </p>

<p>You can <a href="#Viewing">view</a> and <a href="#Filterin">filter</a> alerts. </p>

<Note>Alerts can also trigger a notification. You must subscribe to receive notifications. For more information, see[CDATA[ ]]<a href="/infinity/manage-user-profile#top">Managing your Infinity user profile</a>.</Note>

<p>To access the Infinity alerts:</p>

<ol> <li> On the left menu, select <span>Connect &amp; Manage</span>. <br /> <img src="/images/resources/images/infinity/connectandmanageicon_80x78.png" alt="Connectandmanageicon 80X78" />  </li> <li> Select <span>Alerts</span> to view the alerts list.  <br /><img src="/images/resources/images/infinity/alertsview.png" alt="Alertsview" />  </li> </ol>

<h2 id="Viewingalerts">Viewing alerts </h2>

<p>The <span>Alerts</span> list contain the following fields:</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Date</span> </td> <td> The date and time of the change, formatted as dd/mm/yyyy hh:mm:ss. </td> </tr> <tr> <td><span>Type</span> </td> <td>Displays either <span>Finance</span>, <span>Data</span>, <span>Messaging</span>, <span>IMEI</span>, or <span>Calls</span> alerts. For more information, see <a href="/managing-my-estate/alertprofile#Creating" target="_blank">Creating alert profiles</a>.</td> </tr> <tr> <td><span>ICCID</span> </td> <td>  Displays the alerts triggered for a specific <a href="/glossary/iccid">ICCID</a>. </td> </tr> <tr> <td> <span>Subject</span>  </td> <td> A short summary about the change made to the alert record, for auditing purposes. </td> </tr> </tbody> </table>

<p>To view Infinity alerts:</p>

<ul> <li> On the <span>Alerts</span> page, select <span>View</span> to display the alerts list.  </li> </ul>

<h3>Filtering the displayed alerts</h3>

<p>Refine the displayed <span>Alerts</span> list by using any of the available search filters:</p>

<table> <col /> <col /> <col /> <thead> <tr> <th>Search filter</th> <th>Description</th> <th>Example</th> </tr> </thead> <tbody> <tr> <td> <span>Start Date</span>  </td> <td> Display alerts triggered on or after the chosen date. </td> <td> "21/03/2024" corresponds to alerts generated on or after March 21, 2024. </td> </tr> <tr> <td> <span>End Date</span>  </td> <td>Display alerts triggered up to and including the chosen date.</td> <td> "21/03/2024" corresponds to alerts generated on or before March 21, 2024.</td> </tr> <tr> <td> <span>Type</span>  </td> <td> Display alerts where the following thresholds are breached: <span>Finance</span>, <span>Data</span>, <span>Messaging</span>, <span>IMEI</span>, or <span>Calls</span>.  </td> <td>"IMEI" retrieves the alerts that have triggered because the SIM has moved to another device.</td> </tr> <tr> <td> <span>ICCID</span>  </td> <td>  Display the alerts triggered for a specific ICCID. </td> <td>"89445385230249074" retrieves the alerts associated with that specific ICCID.</td> </tr> </tbody> </table>

<p>To filter alerts:</p>

<ol> <li>On the <span>Alerts</span> page, on the <span>View</span> tab, select the <span>Start Date</span> and <span>End Date</span> to constrain the displayed list to a particular time period.</li> <li>Select <span>Type</span> to return alerts that have breached a particular threshold.</li> <li>Select <span>ICCID</span> to constrain the results to a specific SIM.</li> <li>Select <span>Filter</span> to return matching results.</li> <li>Select <span>Reset</span> to clear all the applied filters.</li> </ol>
