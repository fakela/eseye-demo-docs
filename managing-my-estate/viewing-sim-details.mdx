<p>On the left menu, select <span>SIMs</span> to search for a SIM and display related information.</p>

<p><img src="/images/resources/images/integra/simsmenu.png" alt="Simsmenu" /> </p>

<Tip>When you <a href="/managing-my-estate/viewing-estate-details#SIM">search for a SIM</a>, Infinity displays the following information associated with the SIM.</Tip>

<p>SIMs is divided into the following sections:</p>

<ul> <li><span>SIM Details</span>: For more information, see <a href="#SimDetails">Viewing general information for a relevant SIM</a>.</li> <li> <span>SIM</span>: For more information, see <a href="#IMSIs">Viewing connectivity status information</a>.</li> <li><span>Contract Information</span>: For more information, see <a href="#ContractInfo">Viewing contract information</a>.</li> <li><span>Last Connection details</span>: For more information, see <a href="#LastConnectionDetails">Viewing last recorded network connection</a>.</li> <li><span>Billing usage</span>: For more information, see <a href="#BillingUsage">Viewing billing usage</a>.</li> </ul>

<h2 id="Aboutsearchfields">About search fields</h2>

<p>The search fields enable you to search for a SIM.</p>

<p><img src="/images/resources/images/infinity/simssearchfields.png" alt="Simssearchfields" /> </p>

<p>To search for a SIM or group of SIMs:</p>

<ol> <li>On the left menu, select <span>SIMs</span>.</li> <li>Select the <span>Search type</span> drop-down list.</li> <li>Provide value in the <span>Enter an ICCID, SIMID, EID or IMSI:</span>.</li> <li> Select <span>Search</span> to view the results in the <span>Search results table</span>. <span><strong>Note: </strong>Up to 10 saved searches are stored in your search history of your current user session.</span> </li> </ol>

<p>To display a previously saved search:</p>

<ol> <li>On the left menu, select <span>SIMs</span>.</li> <li>In the <span>Select a previous search</span> drop-down list, select <span>Search</span> to display the below tabs for your relevant SIM.</li> </ol>

<h3>About <span>Search fields</span> attributes</h3>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Search type</span> </td> <td> The identifier type to use in the search string field: <ul> <li> <span><a href="/glossary/iccid#top">ICCID</a></span>  </li> <li> <span><a href="/glossary/simid#top">SIM ID</a></span>  </li> <li> <span><a href="/glossary/eid#top">EID</a></span>  </li> <li> <span><a href="/glossary/imsi#top">IMSI</a></span>  </li> </ul> </td> </tr> <tr> <td><span>Search string</span> </td> <td> The search string (of the type defined above) used to identify the SIM. <span><strong>Note: </strong>You must specify the full string for the selected search type (ICCID, SIM ID, EID, etc). The search does not return values for partial matches.</span> </td> </tr> <tr> <td><span>Previous search</span> </td> <td> Select from your search history to display information for a previously searched SIM.  </td> </tr> </tbody> </table>

<h2 id="ViewinggeneralinformationforarelevantSIM">Viewing general information for a relevant SIM</h2>

<p>To display general information about your specific SIM:</p>

<ul> <li>In the <span>Search results</span> table, select the checkbox alongside the relevant <span>SIM details</span> row.</li> <li>Select <span>SIM Details</span> tab.</li> </ul>

<p><img src="/images/resources/images/infinity/simdetailstab.png" alt="Simdetailstab" /> </p>

<p>At the top right, select <span>Actions &gt; Copy to Clipboard</span> to copy the content from this tab to the clipboard. Paste the information in a.CSV file, notepad, word, email or any other text editor.</p>

<Note>On this tab, if data is unavailable for any field, the information for that particular field is not copied.</Note>
