<p>These transitions typically happen based on user actions, billing issues, or administrative decisions within the Infinity portal. For more information, see <a href="/billing/billing-sim-lifecycle" target="_blank">Understanding the SIM lifecycle</a>.</p>

<p>To access SIM States using SIM and Device Search:</p>

<ol> <li> On the left menu, select <span>Connect &amp; Manage</span>. <br /> <img src="/images/resources/images/infinity/connectandmanageicon_80x78.png" alt="Connectandmanageicon 80X78" />  </li> <li> Select <span>SIM and Device Search</span> to view the <span>Search results</span> list. <br />The SIM and Device search page appears. <br /><img src="/images/resources/images/infinity/simanddevicesearch1.png" alt="Simanddevicesearch1" />  </li> </ol>

<h2 id="ChangingSIMstates">Changing SIM states</h2>

<p>To change the state of a single SIM:</p>

<ol> <li>On the <span>SIM and Device search</span> page, in the <span>Search results</span> list, select the checkbox alongside the specific SIM that requires a state change.</li> <li> At the top right, select <span>Actions</span> &gt; <span>SIM State Change</span> to display a list of states.  <span><strong>Note: </strong>The available states depend on the current SIM state. For more information, see<a href="/billing/billing-sim-lifecycle" target="_blank">Understanding the SIM lifecycle</a>.</span> </li> <li> Select the new state. For example: <br /><img src="/images/resources/images/infinity/actionssimstatechange.png" alt="Actionssimstatechange" />  <br />If the SIM state changes successfully, a green notification appears: "The state has been updated". The SIM appears in the list with the updated state. <br />If processing is not instant, the SIM status is listed as <span>InProgress</span>. <br />If you encounter errors, contact Support. </li> </ol>

<h2 id="ActivatingaSIMonadifferentpackage">Activating a SIM on a different package</h2>

<Tip>You can activate a SIM on a different package to its current associated package. You can only change package if the SIM is in an <span>Available</span> state.</Tip>

<p>Activating a SIM on a different package means that the usage costs will change according to the agreements defined on that package.</p>

<p>To activate a SIM on a different package:</p>

<ol> <li> Change SIM state to <span>Active</span>. <br />For more information, see <a href="#Changing">Changing SIM states</a>. <br />The <span>Activate SIM</span> dialog appears. </li> <li> Select the specific package, then select <span>Activate</span>. <br /><img src="/images/resources/images/infinity/simpackage.png" alt="Simpackage" />  <br />The SIM updates with an <span>Active</span> state and new package. <ul> <li>If the SIM state changes successfully, a green notification appears: "The state has been updated". The SIM appears in the list with the updated state and package.</li> <li>If processing is not instant, the SIM status is listed as <span>InProgress</span>.</li> </ul> <span><strong>Note: </strong>If you encounter errors, contact Support.</span> </li> </ol>

<h2 id="ActivatingmultipleSIMs">Activating multiple SIMs </h2>

<p>To activate multiple SIMs on their associated packages:</p>

<Tip>You can activate multiple SIMs on their respective packages,, provided the SIMs are in the <span>Available</span> state.</Tip>

<ol> <li> On the <span>SIM and Device search</span> page, in the <span>Search results</span> list, select the checkboxes alongside the preferred SIMs that require activation. <br /><img src="/images/resources/images/infinity/selectsims.png" alt="Selectsims" />  </li> <li> At the top right, select <span>Actions</span> &gt; <span>Activate</span>.  </li> <li> Select <span>OK</span> to activate the selected SIMs. <br /><img src="/images/resources/images/infinity/multi-sim-changestate.png" alt="Multi Sim Changestate" />  <ul> <li>If the SIM state changes successfully, a green notification appears: "The states have been updated". The SIMs appears in the <span>Search results</span> list with their respective updated states.</li> <li>If processing is not instant, the SIM statuses are listed as <span>InProgress</span>.</li> <li>If you encounter errors, contact Support.</li> </ul> </li> </ol>
