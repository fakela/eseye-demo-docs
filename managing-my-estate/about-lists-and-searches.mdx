<p>Use the <span>My lists &amp; searches</span> page to view your list of saved searches, SIM collections and search views managed in the <a href="/managing-my-estate/viewing-estate-details" target="_blank">SIM and Device Search</a>. </p>

<p>To access <span>My lists &amp; searches</span>: </p>

<ul> <li> On the left menu, select <span>My Lists and Searches &amp; Device Type</span>. <br /><img src="/images/resources/images/infinity/mylistsicon-dt.png" alt="Mylistsicon Dt" />  </li> </ul>

<h2 id="Viewingsavedsearches">Viewing saved searches</h2>

<p>To display your saved searches:</p>

<ul> <li>On the left menu, select <span>My Lists &amp; Searches</span> &gt; <span>SAVED SEARCHES</span>.</li> </ul>

<p>The <span>Saved Searches</span> tab lists the names given to your saved search views. For more information, see <a href="/managing-my-estate/viewing-estate-details#To2">To display SIM activity information:</a>.</p>

<p><img src="/images/resources/images/infinity/savedsearches.png" alt="Savedsearches" /> </p>

<p>The <span>Saved Searches</span> list contain the following fields:</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Name</span> </td> <td> The saved search name. </td> </tr> <tr> <td><span>Last updated by</span> </td> <td>The most recent user to update the saved search.</td> </tr> <tr> <td><span>Updated date</span> </td> <td>The date and time when the saved search was last changed. </td> </tr> <tr> <td> <img src="/images/resources/images/infinity/iconedit_24x24.png" alt="Iconedit 24X24" /> </td> <td> Select to rename the saved search. </td> </tr> <tr> <td> <img src="/images/resources/images/infinity/icondelete_24x24.png" alt="Icondelete 24X24" /> </td> <td>Select to delete the saved search.</td> </tr> <tr> <td> <img src="/images/resources/images/infinity/iconjump_24x24.png" alt="Iconjump 24X24" /> </td> <td> Select to view the SIM details in the <span>Search results</span> table and perform actions on the <span>SIM details</span> rows. For more information, see <a href="/managing-my-estate/viewing-estate-details#Viewing" target="_blank">Viewing search results list</a>. </td> </tr> </tbody> </table>

<h2 id="Viewingsearchviews">Viewing search views</h2>

<p>To display your search views:</p>

<ul> <li>On the left menu, select <span>My Lists &amp; Searches</span> &gt; <span>My Search Views</span>.</li> </ul>

<p>The <span>My Search Views</span> tab lists the names given to your previously saved search views, which defines the list of columns displayed in the <span>Search results</span> table. For more information, see <a href="/managing-my-estate/viewing-estate-details#Managing" target="_blank">Managing search views</a>.</p>

<p><img src="/images/resources/images/infinity/mysearchviews.png" alt="Mysearchviews" /> </p>

<p>The <span>Search Views</span> list contain the following fields:</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Name</span> </td> <td> The name of the search view on the <span>SIM and Device Search</span> page (For more information, see <a href="/managing-my-estate/viewing-estate-details#To" target="_blank">Finding your SIMs and devices</a>. </td> </tr> <tr> <td><span>Columns displayed</span> </td> <td>Expand the name of the search view to list the columns that are included in this search view.</td> </tr> <tr> <td> <img src="/images/resources/images/infinity/iconedit_24x24.png" alt="Iconedit 24X24" /> </td> <td>Select to rename the saved search view.</td> </tr> <tr> <td> <img src="/images/resources/images/infinity/icondelete_24x24.png" alt="Icondelete 24X24" /> </td> <td>Select to delete the saved search view.</td> </tr> </tbody> </table>

<h2 id="Viewingdevicetypes">Viewing device types</h2>

<p>To view device types:</p>

<ol> <li>On the left menu, select <span>My Lists and Searches</span> &gt; <span>Device Types</span>.</li> <li> Select <span>Settings</span> &gt; <span>Device Types</span> to configure the Infinity device models.  </li> </ol>

<p><img src="/images/resources/images/infinity/settingspage.png" alt="Settingspage" /> </p>

<p>The <span>Device Types</span> list contain the following fields:</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Name</span> </td> <td> A string that displays the name of the device to uniquely identify the device containing the SIM.  </td> </tr> <tr> <td> <img src="/images/resources/images/infinity/iconedit_24x24.png" alt="Iconedit 24X24" /> </td> <td> Select to rename the device type. </td> </tr> <tr> <td> <img src="/images/resources/images/infinity/icondelete_24x24.png" alt="Icondelete 24X24" /> </td> <td> Select to delete the device type. </td> </tr> <tr> <td> <img src="/images/resources/images/infinity/iconjump_24x24.png" alt="Iconjump 24X24" /> </td> <td> Select to perform a search on the <a href="/managing-my-estate/viewing-estate-details#SIM">Finding your SIMs and devices</a> with the Device model filter set to the device type of this row. </td> </tr> </tbody> </table>

<p> </p>

<h2 id="Addingdevicetypes">Adding device types </h2>

<p>To add a new device type:</p>

<ol> <li>On the left menu, select <span>Portfolio</span> &gt; <span>Settings</span>. </li> <li>Select the <span>Device types</span> tab.</li> <li>Select <span>Actions</span> &gt; <span>Add new device type</span>.</li> <li> Type a unique name for the device model and select <span>Save</span>. <span><strong>Tip: </strong>The added device model appears in the list of <a href="/managing-my-estate/viewing-estate-details#Model">Model drop-down list</a>.</span> </li> </ol>

<h2 id="Updatingdevicetypes">Updating device types </h2>

<p>To modify a device type:</p>

<Note>If you modify a device type, the updated device type will appear in the associated menus. For more information, see <a href="/managing-my-estate/viewing-estate-details#ToChangeDeviceSettings">To link IMEI with device model:</a></Note>

<ol> <li>On the left menu, select <span>Portfolio</span> &gt; <span>Settings</span>. </li> <li>Select the <span>Device types</span> tab.</li> <li>Alongside the device type you want to modify, select <img src="/images/resources/images/infinity/iconedit_24x24.png" alt="Iconedit 24X24" />. </li> <li> Type a new unique name for the device model, then select <span>Save</span>. </li> </ol>

<h2 id="Deletingdevicetypes">Deleting device types </h2>

<p>To delete a device type:</p>

<Note>If you delete a device type, it will no longer appear in the associated menus. For more information, see <a href="/managing-my-estate/viewing-estate-details#ToChangeDeviceSettings">To link IMEI with device model:</a></Note>

<ol> <li>On the left menu, select select <span>Portfolio</span> &gt; <span>Settings</span>. </li> <li>Select the <span>Device types</span> tab.</li> <li>Alongside the device type you want to remove, select<img src="/images/resources/images/infinity/icondelete_24x24.png" alt="Icondelete 24X24" />. </li> <li>Select <span>Confirm</span> to delete the device.</li> </ol>
