<p>The <span>Audit Log</span> allows you to search and review records of every action taken and every change made to all SIMs, portfolios, packages associated with your user account. Overall, Infinity audit feature provides valuable insights into your device usage, helping you optimize costs, enhance security, and ensure regulatory compliance. </p>

<p>The purpose of having an audit log is to provide a central repository for all important actions and events that have occurred under your portfolio. This is to ensure we are compliant with security best practices whilst promoting transparency.</p>

<h4>Benefits of <span>Audit Log</span></h4>

<ul> <li>Users can access tamper-proof event records​</li> <li> Allows filtering of events by <span>Portfolio</span>. </li> <li> Enables filtering of events by <span>Username</span>. </li> <li> Centralizes all audit categories into one location. </li> </ul>

<p>To access Infinity Audit:</p>

<p>On the left menu, select <span>Audit</span> to display your audit records.</p>

<p> <img src="/images/resources/images/infinity/audit-icon.png" alt="Audit Icon" /> </p>

<h2 id="ViewingAlerts">Viewing Alerts</h2>

<h3>Applying search filters</h3>

<p>The search filters provide different ways to filter the list of audits and only show results which match the criteria defined in <b>all</b> of the applied search filters.</p>

<p>To search for the audit record that matches the specified criteria:</p>

<ol> <li> Enter values for any of the <a href="#About">Search filter attributes</a>. <h4>About <span>Search filter</span> attributes</h4> <table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td> <span>Date</span>  </td> <td>Enter the date in the format dd/mm/yyyy and timestamp format hh:mm:ss.</td> </tr> <tr> <td> <span>Username</span>  </td> <td>Select from the <a href="#User">About Infinity audit</a> drop-down list.</td> </tr> <tr> <td> <span>Portfolio</span>  </td> <td> Select from the <a href="#Portfoli">Portfolio</a> drop-down list. </td> </tr> <tr> <td><span>Reference</span> </td> <td>Enter the <a href="#Referenc">Reference</a>.</td> </tr> </tbody> </table> </li> <li> Select <span>Filter</span> to display the Search results list matching the specified <a href="#About">Search filter attributes</a>. </li> <li>Select <span>Reset</span> to clear the specified values from the search filters to display the <a href="#Search"> Viewing search results list</a>.</li> </ol>

<h3> Viewing search results list</h3>

<h5>Viewing more results</h5>

<p>Some pages return results in a continuous list. Select <span> View More</span> at the end of the list to see more items.</p>

<p><img src="/images/resources/images/infinity/audit-example.png" alt="Audit Example" /> </p>

<p>The <span>Search Results</span> list contain the following fields:</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td> <span>Date</span>  </td> <td> The date and time of the change, formatted as dd/mm/yyyy hh:mm:ss. </td> </tr> <tr> <td> <span>Category</span>  </td> <td>The list of audit categories.</td> </tr> <tr> <td> <span>Summary</span>  </td> <td> A short summary about the change made to the audit record. </td> </tr> <tr> <td> <span>Details</span>  </td> <td> Explanation about the audit change. </td> </tr> <tr> <td> <span>Reference</span>  </td> <td> The <a href="/glossary/iccid">ICCID</a> of the SIM on which the user made the change. </td> </tr> </tbody> </table>
