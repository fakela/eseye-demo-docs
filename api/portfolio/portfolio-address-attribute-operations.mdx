<p>The Portfolio API enables you to: </p>

<ul> <li>View details for <a href="#Viewing2">single</a> or <a href="#Viewing">multiple</a> attributes on a specific portfolio address</li> <li><a href="#Creating">Create</a>, <a href="#Updating">update</a>, and <a href="#Deleting">delete</a> portfolio address attributes on a specific portfolio address</li> </ul>

<p>You can also:</p>

<ul> <li>Create, update, and view <a href="/api/portfolio/portfolio-address-operations" target="_blank">portfolio addresses</a>.</li> <li>Create, update, and view <a href="/api/portfolio/portfolio-operations" target="_blank">portfolios</a>.</li> <li>Create, update, and view <a href="/api/portfolio/portfolio-attribute-operations" target="_blank">portfolio attributes</a>.</li> </ul>

<h2 id="Viewingallportfolioaddressattributesforaportfolioaddress">Viewing all portfolio address attributes for a portfolio address</h2>

<p>To view all attributes for a specific portfolio address:</p>

<ul> <li> Call <span>GET /api/v2/portfolio/&#123;<var>portfolioId</var>&#125;/portfolioAddress/&#123;<var>portfolioAddressId</var>&#125;/attribute/</span> <br />where &#123;<var>portfolioId</var>&#125; identifies the portfolio, and &#123;<var>portfolioAddressId</var>&#125; identifies the portfolio address containing the attributes you want to view. <br /><span>Path parameters:</span> <span>portfolioId</span><b>&#42;</b>; <span>portfolioAddressId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <span><strong>Tip: </strong>You can apply filters in the request path to limit the results, using one or more parameters. The API returns exactly matching records only. Alternatively, remove all filters from the path to return all records. For example, adding <span><span>?status=active</span></span> to the end of the path will return <span>active</span> objects only.</span> <br />The API returns all the portfolio address attribute details for the specified portfolio address. </li> </ul>

<h2 id="Viewingaspecificportfolioaddressattribute">Viewing a specific portfolio address attribute</h2>

<p>To view details for a specific portfolio address attribute:</p>

<ul> <li> Call <span>GET /api/v2/portfolio/&#123;<var>portfolioId</var>&#125;/portfolioAddress/&#123;<var>portfolioAddressId</var>&#125;/attribute/&#123;<var>portfolioAddressAttributeId</var>&#125;</span>, where &#123;<var>portfolioId</var>&#125; identifies the parent portfolio, &#123;<var>portfolioAddressId</var>&#125; identifies the parent portfolio address, and &#123;<var>portfolioAddressAttributeId</var>&#125; identifies which attribute you want to view. <br /><span>Path parameters:</span> <span>portfolioId</span><b>&#42;</b>; <span>portfolioAddressId</span><b>&#42;</b>; <span>portfolioAddressAttributeId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API returns the specified portfolio address attribute details. </li> </ul>

<h4>Example</h4>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/portfolio/a628ebc659f7dac01f9fdcbd1116307a/portfolioAddress/f20bc36623274be6a73fa62a5b9b354c/attribute/38625</span> </p>

<h3>Response schema application/json: </h3>

<p><span>portfolioAddressAttributeId</span>; <span>portfolioAddressId</span>; <span>attribute</span>; <span>value</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/portfolio/portfolio-api-reference#Portfoli2" target="_blank">Portfolio address attribute definition</a>.</Note>

<h4>Example</h4>

<p>&#123;</p>

<p>"portfolioAddressAttributeId": 38625,</p>

<p>"portfolioAddressId": f20bc36623274be6a73fa62a5b9b354c,</p>

<p>"attribute": "orderReference",</p>

<p>"status": "active"</p>

<h2 id="Creatinganewportfolioaddressattribute">Creating a new portfolio address attribute</h2>

<p>To create a new portfolio address attribute:</p>

<ul> <li> Call <span>POST /api/v2/portfolio/&#123;<var>portfolioId</var>&#125;/portfolioAddress/&#123;<var>portfolioAddressId</var>&#125;/attribute/</span>, where &#123;<var>portfolioId</var>&#125; identifies the parent portfolio, and &#123;<var>portfolioAddressId</var>&#125; identifies the parent portfolio address where you want to create the new attribute. <br /><span>Path parameters:</span> <span>portfolioId</span><b>&#42;</b>; <span>portfolioAddressId</span><b>&#42;</b> <br /><span>Body parameters:</span> <span>attribute</span><b>&#42;</b>; <span>value</span> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory. You can leave <span>value</span> empty <span>""</span>, but do not use <span>null</span>. </span> <br />The API creates a new portfolio address attribute. </li> </ul>

<h4>Example</h4>

<p>POST https://portfolio.api.anynetiot.com/api/v2/portfolio/a628ebc659f7dac01f9fdcbd1116307a/portfolioAddress/f20bc36623274be6a73fa62a5b9b354c/attribute/</p>

<p>/&#42; header information &#42;/</p>

<p>&#123;</p>

<p>"attribute":"defaultAddress"</p>

<p>"value":"yes"</p>

<p>&#125;</p>

<h3>Response schema application/json: </h3>

<p><span>portfolioAddressAttributeId</span>; <span>portfolioAddressId</span>; <span>attribute</span>; <span>value</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/portfolio/portfolio-api-reference#Portfoli2" target="_blank">Portfolio address attribute definition</a>.</Note>

<h4>Example</h4>

<p>&#123;</p>

<p>"portfolioAddressAttributeId": 27589,</p>

<p>"portfolioAddressId": f20bc36623274be6a73fa62a5b9b354c,</p>

<p>"attribute": "defaultAddress"</p>

<p>"value": "yes"</p>

<p>"status": "active"</p>

<h2 id="Updatingaspecificportfolioaddressattribute">Updating a specific portfolio address attribute</h2>

<p>To update a specific portfolio address attribute:</p>

<ul> <li> Call <span>PATCH /api/v2/portfolio/&#123;<var>portfolioId</var>&#125;/portfolioAddress/&#123;<var>portfolioAddressId</var>&#125;/attribute/&#123;<var>portfolioAddressAttributeId</var>&#125;</span>, where &#123;<var>portfolioId</var>&#125; identifies the parent portfolio, &#123;<var>portfolioAddressId</var>&#125; identifies the parent portfolio address, and &#123;<var>portfolioAddressAttributeId</var>&#125; identifies which attribute you want to update. <br /><span>Path parameters:</span> <span>portfolioId</span><b>&#42;</b>; <span>portfolioAddressId</span><b>&#42;</b>; <span>portfolioAddressAttributeId</span><b>&#42;</b> <br /> <span>Body:</span> <span>value</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory. You can leave <span>value</span> empty <span>""</span>, but do not use <span>null</span>. </span> <br />The API updates the specified portfolio address attribute <span>value</span> with the new details. </li> </ul>

<h4>Example</h4>

<p>PATCH https://portfolio.api.anynetiot.com/api/v2/portfolio/a628ebc659f7dac01f9fdcbd1116307a/portfolioAddress/f20bc36623274be6a73fa62a5b9b354c/attribute/38625</p>

<p>/&#42; header information &#42;/</p>

<p>&#123;</p>

<p>"value": "GN-26354-abc"</p>

<p>&#125;</p>

<h3>Response schema application/json: </h3>

<p><span>portfolioAddressAttributeId</span>; <span>portfolioAddressId</span>; <span>attribute</span>; <span>value</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/portfolio/portfolio-api-reference#Portfoli2" target="_blank">Portfolio address attribute definition</a>.</Note>

<h4>Example</h4>

<h2 id="Deletingaspecificportfolioaddressattribute">Deleting a specific portfolio address attribute</h2>

<Tip>You cannot delete the <span>defaultAddress</span> attribute. Instead, set the <span>defaultAddress</span> against a different portfolio address, using <span>Create new portfolio address attribute</span>. The system will remove it automatically from its current address.</Tip>

<p>To delete a specific portfolio address attribute:</p>

<ul> <li> Call <span>DELETE <span>/api/v2/portfolio/&#123;<var>portfolioId</var>&#125;/portfolioAddress/&#123;<var>portfolioAddressId</var>&#125;/attribute/&#123;<var>portfolioAddressAttributeId</var>&#125;</span></span>, where &#123;<var>portfolioId</var>&#125; identifies the parent portfolio, &#123;<var>portfolioAddressId</var>&#125; identifies the parent portfolio address, and &#123;<var>portfolioAddressAttributeId</var>&#125; identifies which attribute you want to delete. <br /><span>Path parameters:</span> <span>portfolioId</span><b>&#42;</b>; <span>portfolioAddressId</span><b>&#42;</b>; <span>portfolioAddressAttributeId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API removes the specified portfolio address attribute. </li> </ul>

<h4>Example</h4>

<p>DELETE https://portfolio.api.anynetiot.com/api/v2/portfolio/a628ebc659f7dac01f9fdcbd1116307a/portfolioAddress/f20bc36623274be6a73fa62a5b9b354c/attribute/38625</p>

<h3>Response schema application/json: </h3>

<p>&#123;</p>

<p>"PORTFOLIO&#95;ADDRESS&#95;ATTRIBUTE": 1</p>
