<h2 id="HandlingthenextPageTokentag">Handling the nextPageToken tag</h2>

<p>When multiple records are returned that exceed the <span>requestedPageSize</span>, the API also returns a <span>nextPageToken</span>. The <span>nextPageToken</span> enables you to tag the results, which you can leverage to jump to a particular page.</p>

<h4>Example</h4>

<p><span>GET /api/v2/&lt;<var>object</var>&gt;/?pageSize=5</span> returns five records per page, and if there are more than 5 records, the response may also contain something like:</p>

<p><span>"nextPageToken":"ApBHMkC8flM6kpi6wpm6tYuwZkCpirBY",</span> </p>

<p>Perform the same request, appending the <span>pageToken</span> key with the <span>nextPageToken</span> value to see the next set of results.</p>

<p><span>GET /api/v2/&lt;<var>object</var>&gt;/?pageSize=5&amp;pageToken=ApBHMkC8flM6kpi6wpm6tYuwZkCpirBY</span> </p>

<p><span>nextPageToken</span> is only returned after you send the initial (and subsequent) requests, until there are no further records to fetch.</p>
