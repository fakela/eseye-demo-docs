<h2 id="Viewingallindustrysectors">Viewing all industry sectors</h2>

<p>To view all industry sectors:</p>

<ul> <li> Call <span>GET /api/v2/industrySector/</span> <span><strong>Tip: </strong>You can apply filters in the request path to limit the results, using one or more parameters. The API returns exactly matching records only. Alternatively, remove all filters from the path to return all records. For example, adding <span><span>?status=active</span></span> to the end of the path will return <span>active</span> objects only.</span> <br />The API returns all the industry sectors. </li> </ul>

<h4>Example</h4>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/industrySector/</span> </p>

<p>or</p>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/industrySector/?status=active</span> </p>

<h2 id="Viewingaspecificindustrysector">Viewing a specific industry sector</h2>

<p>To view details for a specific industry sector:</p>

<ul> <li> Call <span>GET /api/v2/industrySector/&#123;<var>industrySectorId</var>&#125;</span> <br /> where &#123;<var>industrySectorId</var>&#125; identifies which industry sector you want to view. <br /><span>Path parameters:</span>[CDATA[ ]]<span>industrySectorId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API returns the specified industry sector details. </li> </ul>

<h4>Example</h4>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/industrySector/2</span> </p>

<h3>Response schema application/json: </h3>

<p><span>industrySectorId</span>; <span>name</span>; <span>title</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/portfolio/portfolio-api-reference#Industry" target="_blank">Industry sector definition</a>.</Note>

<h4>Example</h4>

<p>&#123;</p>

<p>"industrySectorId": "2",</p>

<p>"name": "Aerospace industries",</p>

<p>"title": "AEROSPACE",</p>

<p>"status": "setup"</p>

<p>&#125;</p>

<h2 id="Creatinganewindustrysector">Creating a new industry sector</h2>

<p>To create a new industry sector:</p>

<ul> <li> Call <span>POST /api/v2/industrySector/</span> <br /><span>Body parameters:</span> <span>name</span><b>&#42;</b>; <span>title</span><b>&#42;</b> <span><strong>Note: </strong> JSON keys marked <b>&#42;</b> are mandatory. <br />Do not use <span>null</span> or <span>""</span> (empty) for mandatory parameters. </span> <br />The API creates a new industry sector. </li> </ul>

<h2 id="Updatingaspecificindustrysector">Updating a specific industry sector</h2>

<p>To update a specific industry sector:</p>

<p>Call <span>PATCH /api/v2/industrySector/&#123;<var>industrySectorId</var>&#125;</span>, where &#123;<var>industrySectorId</var>&#125; identifies which industry sector you want to update.</p>

<p><span>Path parameters:</span> <span>industrySectorId</span><b>&#42;</b>; </p>

<p><span>Body parameters:</span> <span>name</span>; <span>title</span>; <span>status</span></p>

<Note> <p>JSON keys marked <b>&#42;</b> are mandatory.</p> <p>Do not use <span>null</span> or <span>""</span> (empty) for mandatory parameters.</p> <p>If required, you can update more than one optional parameter.</p> </Note>

<p> </p>

<p> </p>

<p>The API updates the specified industry sector parameters with the new details.</p>

<h4>Example</h4>

<p>PATCH https://portfolio.api.anynetiot.com/api/v2/industrySector/2</p>

<p>/&#42; header information &#42;/</p>

<p>&#123;</p>

<p>"status": "active"</p>

<p>&#125;</p>

<h3>Response schema application/json: </h3>

<p><span>industrySectorId</span>; <span>name</span>; <span>title</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/portfolio/portfolio-api-reference#Industry" target="_blank">Industry sector definition</a>.</Note>

<p>"industrySectorId": "2",</p>
