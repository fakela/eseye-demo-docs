<p>The Package API enables you to: </p>

<ul> <li>View details for <a href="#Viewing2">single</a> or <a href="#Viewing">multiple</a> attributes for a specific portfolio</li> <li><a href="#Creating">Create</a>, <a href="#Updating">update</a>, and <a href="#Deleting">delete</a> portfolio attributes</li> </ul>

<p>You can also:</p>

<ul> <li>Create, update, delete and view <a href="/api/portfolio/portfolio-operations" target="_blank">portfolios</a>.</li> </ul>

<h2 id="Viewingallportfolioattributesforaportfolio">Viewing all portfolio attributes for a portfolio</h2>

<p>To view all attributes for a specific portfolio:</p>

<ul> <li> Call <span>GET /api/v2/portfolio/&#123;<var>portfolioId</var>&#125;/attribute/</span> <br />where &#123;<var>portfolioId</var>&#125; identifies the parent portfolio containing the attributes you want to view. <br /><span>Path parameters:</span> <span>portfolioId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <span><strong>Tip: </strong>You can apply filters in the request path to limit the results, using one or more parameters. The API returns exactly matching records only. Alternatively, remove all filters from the path to return all records. For example, adding <span><span>?status=active</span></span> to the end of the path will return <span>active</span> objects only.</span> <br />The API returns all the portfolio attribute details for the specified portfolio. </li> </ul>

<h3>Example</h3>

<p><span>GET /api/v2/portfolio/a628ebc659f7dac01f9fdcbd1116307a/attribute/</span> </p>

<h2 id="Viewingaspecificportfolioattribute">Viewing a specific portfolio attribute</h2>

<p>To view details for a specific portfolio attribute:</p>

<ul> <li> Call <span>GET /api/v2/portfolio/&#123;<var>portfolioId</var>&#125;/attribute/&#123;<var>portfolioAttributeId</var>&#125;</span>, where &#123;<var>portfolioId</var>&#125; identifies the parent portfolio, and &#123;<var>portfolioAttributeId</var>&#125; identifies which attribute you want to view. <br /><span>Path parameters:</span> <span>portfolioId</span><b>&#42;</b>; <span>portfolioAttributeId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API returns the specified portfolio details. </li> </ul>

<h4>Example</h4>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/portfolio/a628ebc659f7dac01f9fdcbd1116307a/attribute/222874</span> </p>

<h3>Response schema application/json: </h3>

<p><span>portfolioAttributeId</span>; <span>portfolioId</span>; <span>attribute</span>; <span>value</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/portfolio/portfolio-api-reference#Package2" target="_blank">Portfolio API reference tables</a>.</Note>

<h4>Example</h4>

<p>&#123;</p>

<p>"portfolioAttributeId": 222874,</p>

<p>"portfolioId": a628ebc659f7dac01f9fdcbd1116307a,</p>

<p>"attribute": "zendeskId",</p>

<p>"value": "654321"</p>

<p>"status": "active"</p>

<p>&#125;</p>

<h2 id="Creatinganewportfolioattribute">Creating a new portfolio attribute</h2>

<p>To create a new portfolio attribute:</p>

<ul> <li> Call <span>POST /api/v2/portfolio/&#123;<var>portfolioId</var>&#125;/attribute/</span>, where &#123;<var>portfolioId</var>&#125; identifies the parent portfolio where you want to create the new attribute. <br /><span>Path parameters:</span> <span>portfolioId</span><b>&#42;</b> <br /><span>Body parameters:</span> <span>attribute</span><b>&#42;</b>; <span>value</span> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory. You can leave <span>value</span> empty <span>""</span>, but do not use <span>null</span>. </span> <br />The API creates a new portfolio attribute. </li> </ul>

<h4>Example</h4>

<p>POST https://portfolio.api.anynetiot.com/api/v2/portfolio/a628ebc659f7dac01f9fdcbd1116307a/attribute/</p>

<p>/&#42; header information &#42;/</p>

<p>&#123;</p>

<p>"attribute":"secondaryEmailAddress"</p>

<p>"value":"support@company.com"</p>

<p>&#125;</p>

<h3>Response schema application/json: </h3>

<p><span>portfolioAttributeId</span>; <span>portfolioId</span>; <span>attribute</span>; <span>value</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/portfolio/portfolio-api-reference#Package2" target="_blank">Portfolio API reference tables</a>.</Note>

<h4>Example</h4>

<p>&#123;</p>

<p>"portfolioAttributeId": 222932,</p>

<p>"portfolioId": a628ebc659f7dac01f9fdcbd1116307a,</p>

<p>"attribute": "secondaryEmailAddress"</p>

<p>"value": "support@company.co.uk"</p>

<p>"status": "active"</p>

<p>&#125;</p>

<h2 id="Updatingaspecificportfolioattribute">Updating a specific portfolio attribute</h2>

<p>To update a specific portfolio attribute:</p>

<ul> <li> Call <span>PATCH /api/v2/portfolio/&#123;<var>portfolioId</var>&#125;/attribute/&#123;<var>portfolioAttributeId</var>&#125;</span>, where &#123;<var>portfolioId</var>&#125; identifies the parent portfolio, and &#123;<var>portfolioAttributeId</var>&#125; identifies which attribute you want to update. <br /><span>Path parameters:</span> <span>portfolioId</span><b>&#42;</b>; <span>portfolioAttributeId</span><b>&#42;</b> <br /> <span>Body:</span> <span>value</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory. You can leave <span>value</span> empty <span>""</span>, but do not use <span>null</span>. </span> <br />The API updates the specified portfolio attribute <span>value</span> with the new details. </li> </ul>

<h4>Example</h4>

<p>PATCH https://portfolio.api.anynetiot.com/api/v2/portfolio/a628ebc659f7dac01f9fdcbd1116307a/attribute/222932</p>

<p>/&#42; header information &#42;/</p>

<p>&#123;</p>

<p>"value": "support@company.com"</p>

<p>&#125;</p>

<h3>Response schema application/json: </h3>

<p><span>portfolioAttributeId</span>; <span>portfolioId</span>; <span>attribute</span>; <span>value</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/portfolio/portfolio-api-reference#Package2" target="_blank">Portfolio API reference tables</a>.</Note>

<h4>Example</h4>

<p>&#123;</p>

<p>"portfolioAttributeId": 222932,</p>

<p>"portfolioId": "a628ebc659f7dac01f9fdcbd1116307a",</p>

<p>"attribute": "secondaryEmailAddress"</p>

<p>"value": "support@company.com"</p>

<p>"status": "active"</p>

<h2 id="Deletingaspecificportfolioattribute">Deleting a specific portfolio attribute</h2>

<p>To delete a specific portfolio attribute:</p>

<ul> <li> Call <span>DELETE <span>/api/v2/portfolio/&#123;<var>portfolioId</var>&#125;/attribute/&#123;<var>portfolioAttributeId</var>&#125;</span></span>, where &#123;<var>portfolioId</var>&#125; identifies the parent portfolio, and &#123;<var>portfolioAttributeId</var>&#125; identifies which attribute you want to delete. <br /><span>Path parameters:</span> <span>portfolioId</span><b>&#42;</b>; <span>portfolioAttributeId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API removes the specified portfolio attribute. </li> </ul>

<h4>Example</h4>

<p>DELETE https://portfolio.api.anynetiot.com/api/v2/portfolio/a628ebc659f7dac01f9fdcbd1116307a/attribute/222874</p>

<h3>Response schema application/json: </h3>

<p>&#123;</p>

<p>"PORTFOLIO&#95;ATTRIBUTE": 1</p>
