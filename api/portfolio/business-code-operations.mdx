<h2 id="Viewingallbusinesscodes">Viewing all business codes</h2>

<p>To view all business codes:</p>

<ul> <li> Call <span>GET /api/v2/businessCode/</span> <span><strong>Tip: </strong>You can apply filters in the request path to limit the results, using one or more parameters. The API returns exactly matching records only. Alternatively, remove all filters from the path to return all records. For example, adding <span><span>?status=active</span></span> to the end of the path will return <span>active</span> objects only.</span> <br />The API returns all the business codes. </li> </ul>

<h4>Example</h4>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/businessCode/</span> </p>

<p>or</p>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/businessCode/?status=active</span> </p>

<h2 id="Viewingaspecificbusinesscode">Viewing a specific business code</h2>

<p>To view details for a specific business code:</p>

<ul> <li> Call <span>GET /api/v2/businessCode/&#123;<var>businessCodeId</var>&#125;</span> <br /> where &#123;<var>businessCodeId</var>&#125; identifies which business code you want to view. <br /><span>Path parameters:</span>[CDATA[ ]]<span>businessCodeId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API returns the specified business code details. </li> </ul>

<h4>Example</h4>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/businessCode/2</span> </p>

<h3>Response schema application/json: </h3>

<p><span>businessCodeId</span>; <span>businessCode</span>; <span>name</span>; <span>title</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/portfolio/portfolio-api-reference#Business" target="_blank">Business code definition</a>.</Note>

<h4>Example</h4>

<p>&#123;</p>

<p>"businessCodeId": "2",</p>

<p>"businessCode": "EU",</p>

<p>"name": "Customer is based in the European Union.",</p>

<p>"title": "EU",</p>

<p>"status": "active"</p>

<p>&#125;</p>

<h2 id="Creatinganewbusinesscode">Creating a new business code</h2>

<p>To create a new business code:</p>

<ul> <li> Call <span>POST /api/v2/businessCode/</span> <br /><span>Body parameters:</span> <span>businessCode</span><b>&#42;</b>; <span>name</span><b>&#42;</b>; <span>title</span><b>&#42;</b> <span><strong>Note: </strong> JSON keys marked <b>&#42;</b> are mandatory. <br />Do not use <span>null</span> or <span>""</span> (empty) for mandatory parameters. </span> <br />The API creates a new business code. </li> </ul>

<h2 id="Updatingaspecificbusinesscode">Updating a specific business code</h2>

<p>To update a specific business code:</p>

<ul> <li> Call <span>PATCH /api/v2/businessCode/&#123;<var>businessCodeId</var>&#125;</span>, where &#123;<var>businessCodeId</var>&#125; identifies which business code you want to update. <br /><span>Path parameters:</span> <span>businessCodeId</span><b>&#42;</b>;  <br /><span>Body:</span> <span>name</span>; <span>title</span>; <span>status</span> <span><strong>Note: </strong> JSON keys marked <b>&#42;</b> are mandatory. <br />Do not use <span>null</span> or <span>""</span> (empty) for mandatory parameters. <br />If required, you can update more than one optional parameter. </span> <br />The API updates the specified business code parameters with the new details. </li> </ul>

<h4>Example</h4>

<p>PATCH https://portfolio.api.anynetiot.com/api/v2/businessCode/1</p>

<p>/&#42; header information &#42;/</p>

<p>&#123;</p>

<p>"status": "active"</p>

<p>&#125;</p>
