<h2 id="Viewingallpaymentmethods">Viewing all payment methods</h2>

<p>To view all payment methods:</p>

<ul> <li> Call <span>GET /api/v2/paymentMethod/</span> <span><strong>Tip: </strong>You can apply filters in the request path to limit the results, using one or more parameters. The API returns exactly matching records only. Alternatively, remove all filters from the path to return all records. For example, adding <span><span>?status=active</span></span> to the end of the path will return <span>active</span> objects only.</span> <br />The API returns all the payment methods. </li> </ul>

<h4>Example</h4>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/paymentMethod/</span> </p>

<p>or</p>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/paymentMethod/?status=active</span> </p>

<h2 id="Viewingaspecificpaymentmethod">Viewing a specific payment method</h2>

<p>To view details for a specific payment method:</p>

<ul> <li> Call <span>GET /api/v2/paymentMethod/&#123;<var>paymentMethodId</var>&#125;</span> <br /> where &#123;<var>paymentMethodId</var>&#125; identifies which payment method you want to view. <br /><span>Path parameters:</span>[CDATA[ ]]<span>paymentMethodId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API returns the specified payment method details. </li> </ul>

<h4>Example</h4>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/paymentMethod/4</span> </p>

<h3>Response schema application/json: </h3>

<p><span>paymentMethodId</span>; <span>name</span>; <span>title</span>; <span>description</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/portfolio/portfolio-api-reference#Payment" target="_blank">Payment method definition</a>.</Note>

<h4>Example</h4>

<p>&#123;</p>

<p>"paymentMethodId": "4",</p>

<p>"name": "Credit Card",</p>

<p>"title": "RepeatBillingAuthorized",</p>

<p>"description": "Pay by Credit Card",</p>

<p>"status": "setup"</p>

<p>&#125;</p>

<h2 id="Creatinganewpaymentmethod">Creating a new payment method</h2>

<p>To create a new payment method:</p>

<ul> <li> Call <span>POST /api/v2/paymentMethod/</span> <br /><span>Body parameters:</span> <span>paymentMethod</span><b>&#42;</b>; <span>name</span><b>&#42;</b>; <span>title</span><b>&#42;</b> <span><strong>Note: </strong> JSON keys marked <b>&#42;</b> are mandatory. <br />Do not use <span>null</span> or <span>""</span> (empty) for mandatory parameters. </span> <br />The API creates a new payment method. </li> </ul>

<h2 id="Updatingaspecificpaymentmethod">Updating a specific payment method</h2>

<p>To update a specific payment method:</p>

<ul> <li> Call <span>PATCH /api/v2/paymentMethod/&#123;<var>paymentMethodId</var>&#125;</span>, where &#123;<var>paymentMethodId</var>&#125; identifies which payment method you want to update. <br /><span>Path parameters:</span> <span>paymentMethodId</span><b>&#42;</b>;  <br /><span>Body:</span> <span>name</span>; <span>title</span>; <span>status</span> <span><strong>Note: </strong> JSON keys marked <b>&#42;</b> are mandatory. <br />Do not use <span>null</span> or <span>""</span> (empty) for mandatory parameters. <br />If required, you can update more than one optional parameter. </span> <br />The API updates the specified payment method parameters with the new details. </li> </ul>

<h4>Example</h4>

<p>PATCH https://portfolio.api.anynetiot.com/api/v2/paymentMethod/4</p>

<p>/&#42; header information &#42;/</p>

<p>&#123;</p>

<p>"status": "active"</p>

<p>&#125;</p>
