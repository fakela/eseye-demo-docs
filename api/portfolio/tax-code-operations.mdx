<h2 id="Viewingalltaxcodes">Viewing all tax codes</h2>

<p>To view all tax codes:</p>

<ul> <li> Call <span>GET /api/v2/taxCode/</span> <span><strong>Tip: </strong>You can apply filters in the request path to limit the results, using one or more parameters. The API returns exactly matching records only. Alternatively, remove all filters from the path to return all records. For example, adding <span><span>?status=active</span></span> to the end of the path will return <span>active</span> objects only.</span> <br />The API returns all the tax codes. </li> </ul>

<h4>Example</h4>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/taxCode/</span> </p>

<p>or</p>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/taxCode/?status=active</span> </p>

<h2 id="Viewingaspecifictaxcode">Viewing a specific tax code</h2>

<p>To view details for a specific tax code:</p>

<ul> <li> Call <span>GET /api/v2/taxCode/&#123;<var>taxCodeId</var>&#125;</span> <br /> where &#123;<var>taxCodeId</var>&#125; identifies which tax code you want to view. <br /><span>Path parameters:</span>[CDATA[ ]]<span>taxCodeId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API returns the specified tax code details. </li> </ul>

<h4>Example</h4>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/taxCode/1</span> </p>

<h3>Response schema application/json: </h3>

<p><span>taxCodeId</span>; <span>taxCode</span>; <span>name</span>; <span>title</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/portfolio/portfolio-api-reference#Tax" target="_blank">Tax code definition</a>.</Note>

<h4>Example</h4>

<p>&#123;</p>

<p>"taxCodeId": "2",</p>

<p>"taxCode": "ESZ",</p>

<p>"name": "The company is based within the European Union, EU tax rates apply.",</p>

<p>"title": "ESZ",</p>

<p>"status": "setup"</p>

<p>&#125;</p>

<h2 id="Creatinganewtaxcode">Creating a new tax code</h2>

<p>To create a new tax code:</p>

<ul> <li> Call <span>POST /api/v2/taxCode/</span> <br /><span>Body parameters:</span> <span>taxCode</span><b>&#42;</b>; <span>name</span><b>&#42;</b>; <span>title</span><b>&#42;</b> <span><strong>Note: </strong> JSON keys marked <b>&#42;</b> are mandatory. <br />Do not use <span>null</span> or <span>""</span> (empty) for mandatory parameters. </span> <br />The API creates a new tax code. </li> </ul>

<h2 id="Updatingaspecifictaxcode">Updating a specific tax code</h2>

<p>To update a specific tax code:</p>

<ul> <li> Call <span>PATCH /api/v2/taxCode/&#123;<var>taxCodeId</var>&#125;</span>, where &#123;<var>taxCodeId</var>&#125; identifies which tax code you want to update. <br /><span>Path parameters:</span> <span>taxCodeId</span><b>&#42;</b>;  <br /><span>Body:</span> <span>name</span>; <span>title</span>; <span>status</span> <span><strong>Note: </strong> JSON keys marked <b>&#42;</b> are mandatory. <br />Do not use <span>null</span> or <span>""</span> (empty) for mandatory parameters. <br />If required, you can update more than one optional parameter. </span> <br />The API updates the specified tax code parameters with the new details. </li> </ul>

<h4>Example</h4>

<p>PATCH https://portfolio.api.anynetiot.com/api/v2/taxCode/2</p>

<p>/&#42; header information &#42;/</p>

<p>&#123;</p>

<p>"status": "active"</p>

<p>&#125;</p>
