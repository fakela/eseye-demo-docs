<p>A <span>portfolio</span> is a collection of a customer's account information, including the company name, billing information such as tax and account currency details, associated portal, postal addresses, and user information.</p>

<p>The Portfolio API enables you to manage your account portfolios, including:</p>

<ul> <li>Retrieving and updating details for an individual portfolio or range of portfolios, such as tax related information and the current currency.</li> <li>Viewing, creating and updating one or more portfolios, or associated objects (such as portfolio attributes or users).</li> </ul>
