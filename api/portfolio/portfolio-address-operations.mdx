<p>The Portfolio API enables you to: </p>

<ul> <li>View details for <a href="#Viewing2">single</a> or <a href="#Viewing">multiple</a> portfolio addresses</li> <li><a href="#Creating">Create</a> and <a href="#Updating">update</a> portfolio addresses</li> </ul>

<ul> <li>Create, update, delete and view <a href="#Viewing" target="_blank">portfolio address attributes</a>.</li> </ul>

<h2 id="Viewingallportfolioaddresses">Viewing all portfolio addresses</h2>

<p>To view all portfolio addresses:</p>

<ul> <li> Call <span>GET /api/v2/portfolio/&#123;<var>portfolioId</var>&#125;/portfolioAddress/</span>, where &#123;<var>portfolioId</var>&#125; identifies the parent portfolio containing the portfolio addresses you want to view. <span><strong>Tip: </strong>You can apply filters in the request path to limit the results, using one or more parameters. The API returns exactly matching records only. Alternatively, remove all filters from the path to return all records. For example, adding <span><span>?status=active</span></span> to the end of the path will return <span>active</span> objects only.</span> <br />The API returns all the portfolio addresses. </li> </ul>

<h4>Example</h4>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/portfolio/a628ebc659f7dac01f9fdcbd1116307a/portfolioAddress/</span> </p>

<p>or</p>

<p><span>GET https://portfolio.api.anynetiot.com/api/v2/portfolio/a628ebc659f7dac01f9fdcbd1116307a/portfolioAddress/?addressType=billing;status=active</span> – this will return all active <span>billing</span> portfolio addresses.</p>

<h2 id="Viewingaspecificportfolioaddress">Viewing a specific portfolio address</h2>

<p>To view details for a specific portfolio address:</p>

<ul> <li> Call <span>GET /api/v2/portfolio/&#123;<var>portfolioId</var>&#125;/portfolioAddress/&#123;<var>portfolioAddressId</var>&#125;</span> <br /> where &#123;<var>portfolioId</var>&#125; identifies the portfolio and &#123;<var>portfolioAddressId</var>&#125; identifies which portfolio address you want to view. <br /><span>Path parameters:</span> <span>portfolioId</span><b>&#42;</b>; <span>portfolioAddressId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API returns the specified portfolio address details. </li> </ul>

<h2 id="Creatinganewportfolioaddress">Creating a new portfolio address</h2>

<p>To create a new portfolio address:</p>

<ul> <li> Call <span>POST /api/v2/portfolio/&#123;<var>portfolioId</var>&#125;/portfolioAddress/</span> <br /> where &#123;<var>portfolioId</var>&#125; identifies the portfolio where you want to create the new portfolio address. <br /><span>Body parameters:</span>[CDATA[ ]]<span>firstName</span><b>&#42;</b>; <span>lastName</span><b>&#42;</b>; <span>companyName</span><b>&#42;</b>; <span>friendlyName</span>; <span>addressType</span><b>&#42;</b>; <span>addressLine1</span><b>&#42;</b>; <span>addressLine2</span>; <span>townOrCity</span><b>&#42;</b>; <span>countyOrState</span><b>&#42;</b>; <span>countryCodeId</span><b>&#42;</b>; <span>postCode</span><b>&#42;</b>; <span>mobileNumber</span><b>&#42;</b>; <span>landLineNumber</span>; <span>emailAddress</span><b>&#42;</b>; <span>status</span> <span><strong>Note: </strong> JSON keys marked <b>&#42;</b> are mandatory. <br />Do not use <span>null</span> or <span>""</span> (empty) for mandatory parameters. <br />If required, you can use more than one optional parameter. </span> <br />The API creates a new portfolio address. </li> </ul>

<h2 id="Updatingaspecificportfolioaddress">Updating a specific portfolio address</h2>

<p>To update a specific portfolio address:</p>

<ul> <li> Call <span>PATCH /api/v2/portfolio/&#123;<var>portfolioId</var>&#125;/portfolioAddress/&#123;<var>portfolioAddressId</var>&#125;</span>, where &#123;<var>portfolioId</var>&#125; identifies the parent portfolio, and &#123;<var>portfolioAddressId</var>&#125; identifies which portfolio address you want to update. <br /><span>Path parameters:</span> <span>portfolioId</span><b>&#42;</b>; <span>portfolioAddressId</span><b>&#42;</b>;  <br /><span>Body:</span> <span>firstName</span>; <span>lastName</span>; <span>companyName</span>; <span>friendlyName</span>; <span>addressType</span>; <span>addressLine1</span>; <span>addressLine2</span>; <span>townOrCity</span>; <span>countyOrState</span>; <span>countryCodeId</span>; <span>postCode</span>; <span>mobileNumber</span>; <span>landLineNumber</span>; <span>emailAddress</span>; <span>status</span> <span><strong>Note: </strong> JSON keys marked <b>&#42;</b> are mandatory. <br />Do not use <span>null</span> or <span>""</span> (empty) for mandatory fields. <br />The <span>countryCodeId</span> value must already exist in the system. <br />If required, you can update more than one optional parameter. </span> <br />The API updates the specified portfolio address parameters with the new details. </li> </ul>

<h4>Example</h4>

<p>PATCH https://portfolio.api.anynetiot.com/api/v2/portfolio/a628ebc659f7dac01f9fdcbd1116307a/portfolioAddress/ebdc7678a1404f2c856a6a0b7ecbb186</p>
