<h2 id="ViewingtheiccPortfolioPackageforaspecificSIM">Viewing the iccPortfolioPackage for a specific SIM</h2>

<p>To view iccPortfolioPackage details for a specific SIM:</p>

<ul> <li> Call <span>GET /api/v2/sim/&#123;<var>iccId</var>&#125;/iccPortfolioPackage/</span>, where &#123;<var>iccId</var>&#125; identifies the SIM that contains the iccPortfolioPackage you want to view. <br /><span>Path parameters:</span> <span>iccId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API returns the specified iccPortfolioPackage details. </li> </ul>

<h4>Example</h4>

<p><span>GET https://sim.api.anynetiot.com/api/v2/8953449647428546/iccPortfolioPackage/</span> </p>

<h3>Response schema application/json:</h3>

<p><span>iccPortfolioPackageId</span>; <span>iccId</span>; <span>portfolioId</span>; <span>portfolioStartDate</span>; <span>packageId</span>; <span>contractStartDate</span>; <span>contractEndDate</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/sim/simapi-reference#iccPortfolioPackage" target="_blank">iccPortfolioPackage definition</a>.</Note>

<h4>Example</h4>

<p>&#123;</p>

<p>"iccPortfolioPackageId": 9953449274684574,</p>

<p>"iccId": "8953449647428546",</p>

<p>"portfolioId": "a628ebc659f7dac01f9fdcbd1116307a",</p>

<p>"portfolioStartDate": "2022-03-15T16:21:08Z",</p>

<p>"packageId": 12345,</p>

<p>"contractStartDate": "2024-06-22T10:35:09Z",</p>

<p>"contractEndDate": "2026-06-22T10:35:09Z",</p>

<p>"status": "active"</p>

<h2 id="UpdatingtheiccPortfolioPackageforaspecificSIM">Updating the iccPortfolioPackage for a specific SIM</h2>

<p>To update iccPortfolioPackage details for a specific SIM:</p>

<ul> <li> Call <span>PATCH /api/v2/sim/&#123;<var>iccId</var>&#125;/iccPortfolioPackage/</span> where &#123;<var>iccId</var>&#125; identifies the SIM that contains the iccPortfolioPackage you want to update. <br /><span>Path parameters:</span> <span>iccId</span><b>&#42;</b> <br /><span>Body:</span> <span>packageId</span>; <span>portfolioId</span> <span><strong>Note: </strong> JSON keys marked <b>&#42;</b> are mandatory. <br />Do not use <span>null</span> or <span>""</span> (empty) for mandatory parameters. <br />If required, you can update more than one optional parameter. </span> <br />The API updates the specified iccPortfolioPackage parameters with the new details. </li> </ul>

<h4>Example</h4>

<p>PATCH https://sim.api.anynetiot.com/api/v2/iccPortfolioPackage/9953449274684574</p>

<p>/&#42; header information &#42;/</p>

<p>&#123;</p>

<p>"packageId": "98765"</p>

<p>&#125;</p>

<h3>Response schema application/json: </h3>

<p><span>iccPortfolioPackageId</span>; <span>iccId</span>; <span>portfolioId</span>; <span>portfolioStartDate</span>; <span>packageId</span>; <span>contractStartDate</span>; <span>contractEndDate</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/sim/simapi-reference#iccPortfolioPackage" target="_blank">iccPortfolioPackage definition</a>.</Note>
