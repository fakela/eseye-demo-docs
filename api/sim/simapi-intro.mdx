<p>Eseye's SIM API enables you to manage your global IoT SIM estate, including:</p>

<ul> <li>Activating and deactivating SIMs</li> <li>Retrieving SIM details for an individual SIM or range of SIMs, such as identifiers and the current status</li> <li>Updating a SIM's details, such as the package or portfolio it belongs to, or suspending a SIM</li> <li>Creating, updating, and deleting SIM attributes</li> </ul>

<h2 id="Gettingstarted">Getting started</h2>

<h2 id="UsingthesuppliedPostmancollectionandenvironment">Using the supplied Postman collection and environment</h2>

<p>We have supplied a Postman collection and environment to help you understand how the API works. The Postman collection is set up to use the variables listed in the environment. </p>

<p>You must configure the environment with your own test system values and an access token, which will authenticate your API session. </p>

<p>Use test objects to understand how the API works. Eseye Ltd is not responsible for the loss of live data as a result of using the API.</p>

<p>For more information, see <a href="/api/sim/setting-up-postman-for-simapi">Setting up Postman for SIM API</a>.</p>

<h2 id="SIMAPIoperations">SIM API operations</h2>

<p>After you have access to the API, you can use the following operations to make requests, retrieve data, and manage resources:</p>

<h4>SIM</h4>

<p>These endpoints enable you to see and manage the SIM details such as core IDs (iccId, simId, eId), if the SIM is suspended, SIM status, linked package and portfolio, and high level contract details.</p>

<ul> <li><a href="/api/sim/si-moperations#Viewing" target="_blank">Viewing all SIMs</a> </li> <li><a href="/api/sim/si-moperations#Viewing2" target="_blank">Viewing a specific SIM</a> </li> <li><a href="/api/sim/si-moperations#Updating" target="_blank">Updating a specific SIM</a> </li> </ul>

<h4>SIM attribute</h4>

<p>These endpoints enable you to see and manage the extra details about the SIM that enable efficient SIM administration.</p>

<ul> <li><a href="/api/sim/icc-attribute-operations#Viewing" target="_blank">Viewing all SIM attributes</a> </li> <li><a href="/api/sim/icc-attribute-operations#Creating" target="_blank">Creating a new SIM attribute</a> </li> <li><a href="/api/sim/icc-attribute-operations#Viewing2" target="_blank">Viewing a specific SIM attribute</a> </li> <li><a href="/api/sim/icc-attribute-operations#Updating" target="_blank">Updating a specific SIM attribute</a> </li> <li><a href="/api/sim/icc-attribute-operations#Deleting" target="_blank">Deleting a specific SIM attribute</a> </li> </ul>

<h4>iccPortfolioPackage</h4>

<p>These endpoints enable you to view and manage the portfolio and package relationship with the SIM in greater depth.</p>

<ul> <li><a href="/api/sim/portfolio-package-operations#Viewing2" target="_blank">Viewing the iccPortfolioPackage for a specific SIM</a> </li> <li><a href="/api/sim/portfolio-package-operations#Updating" target="_blank">Updating the iccPortfolioPackage for a specific SIM</a> </li> </ul>

<h4>Billing snapshot</h4>

<p>This endpoint provides an overview of the SIM's bundled service and usage charges, as well as its most recently recorded activity.</p>

<p><a href="/api/sim/billing-snapshot-operations#Viewing2" target="_blank">Viewing the billing snapshot for a specific SIM</a> </p>

<h4>Network snapshot</h4>

<p>This endpoint provides an overview of the SIM's physical network connections, such as network provider details; RAT and mast details; APN, IP address and last activity RADIUS details, and so on.</p>

<p><a href="/api/sim/network-snapshot-operations#Viewing2" target="_blank">Viewing the network snapshot for a specific SIM</a> </p>
