<p>The SIM API enables you to: </p>

<ul> <li>View details for <a href="#Viewing2">single</a> or <a href="#Viewing">multiple</a> attributes on a specific SIM</li> <li><a href="#Creating">Create</a>, <a href="#Updating">update</a>, and <a href="#Deleting">delete</a> SIM attributes on a specific SIM</li> </ul>

<p>You can also:</p>

<ul> <li>View and update <a href="/api/sim/si-moperations" target="_blank">SIMs</a>.</li> </ul>

<ul> <li>User required to login</li> <li>Bearer token required</li> </ul>

<h2 id="ViewingallSIMattributes">Viewing all SIM attributes</h2>

<p>To view all attributes for a specific SIM:</p>

<ul> <li> Call <span>GET /api/v2/sim/&#123;<var>iccId</var>&#125;/attribute/</span> <br />where &#123;<var>iccId</var>&#125; identifies the SIM containing the attributes you want to view. <br /><span>Path parameters:</span> <span>iccId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <span><strong>Tip: </strong>You can apply filters in the request path to limit the results, using one or more parameters. The API returns exactly matching records only. Alternatively, remove all filters from the path to return all records. For example, adding <span><span>?status=active</span></span> to the end of the path will return <span>active</span> objects only.</span> <br />The API returns all the SIM attribute details for the specified SIM. </li> </ul>

<h4>Example</h4>

<p><span>GET https://sim.api.anynetiot.com/api/v2/sim/8953448576492474/attribute/</span> </p>

<h2 id="ViewingaspecificSIMattribute">Viewing a specific SIM attribute</h2>

<p>To view details for a specific SIM attribute:</p>

<ul> <li> Call <span>GET /api/v2/sim/&#123;<var>iccId</var>&#125;/attribute/&#123;<var>iccAttributeId</var>&#125;</span>, where &#123;<var>iccId</var>&#125; identifies the parent SIM, and &#123;<var>iccAttributeId</var>&#125; identifies which attribute you want to view. <br /><span>Path parameters:</span> <span>iccId</span><b>&#42;</b>; <span>iccAttributeId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API returns the specified SIM attribute details. </li> </ul>

<h4>Example</h4>

<p><span>GET https://sim.api.anynetiot.com/api/v2/sim/8953448576492474/attribute/11506792</span> </p>

<h3>Response schema application/json: </h3>

<p><span>iccAttributeId</span>; <span>iccId</span>; <span>attribute</span>; <span>value</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/sim/simapi-reference#SIM2" target="_blank">SIM attribute definition</a>.</Note>

<h4>Example</h4>

<p>&#123;</p>

<p>"iccAttributeId": 11506792,</p>

<p>"iccId": 8953448576492474,</p>

<p>"attribute": "orderId",</p>

<p>"value": "5791"</p>

<p>"status": "active"</p>

<p>&#125;</p>

<h2 id="CreatinganewSIMattribute">Creating a new SIM attribute</h2>

<p>To create a new SIM attribute:</p>

<ul> <li> Call <span>POST /api/v2/sim/&#123;<var>iccId</var>&#125;/attribute/</span>, where &#123;<var>iccId</var>&#125; identifies the parent SIM where you want to create the new attribute. <br /><span>Path parameters:</span> <span>iccId</span><b>&#42;</b> <br /><span>Body parameters:</span> <span>attribute</span><b>&#42;</b>; <span>value</span> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory. You can leave <span>value</span> empty <span>""</span>, but do not use <span>null</span>. </span> <br />The API creates a new SIM attribute. </li> </ul>

<h4>Example</h4>

<p>POST https://sim.api.anynetiot.com/api/v2/sim/8953448576492474/attribute/</p>

<p>/&#42; header information &#42;/</p>

<p>&#123;</p>

<p>"attribute":"friendlyName"</p>

<p>"value":"50kW – Rapid EV charger"</p>

<p>&#125;</p>

<h3>Response schema application/json: </h3>

<p><span>iccAttributeId</span>; <span>iccId</span>; <span>attribute</span>; <span>value</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/sim/simapi-reference#SIM2" target="_blank">SIM attribute definition</a>.</Note>

<h4>Example</h4>

<p>&#123;</p>

<p>"iccAttributeId": 11506795,</p>

<p>"iccId": 8953448576492474,</p>

<p>"attribute": "friendlyName"</p>

<p>"value": "50kW – Rapid EV charger"</p>

<p>"status": "active"</p>

<p>&#125;</p>

<h2 id="UpdatingaspecificSIMattribute">Updating a specific SIM attribute</h2>

<p>To update a specific SIM attribute:</p>

<ul> <li> Call <span>PATCH /api/v2/sim/&#123;<var>iccId</var>&#125;/attribute/&#123;<var>iccAttributeId</var>&#125;</span>, where &#123;<var>iccId</var>&#125; identifies the parent SIM, and &#123;<var>iccAttributeId</var>&#125; identifies which attribute you want to update. <br /><span>Path parameters:</span> <span>iccId</span><b>&#42;</b>; <span>iccAttributeId</span><b>&#42;</b> <br /> <span>Body:</span> <span>value</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory. You can leave <span>value</span> empty <span>""</span>, but do not use <span>null</span>. </span> <br />The API updates the specified SIM attribute <span>value</span> with the new details. </li> </ul>

<h4>Example</h4>

<p>PATCH https://sim.api.anynetiot.com/api/v2/sim/8953448576492474/attribute/11506792</p>

<p>/&#42; header information &#42;/</p>

<p>&#123;</p>

<p>"value": "579"</p>

<p>&#125;</p>

<h3>Response schema application/json: </h3>

<p><span>iccAttributeId</span>; <span>iccId</span>; <span>attribute</span>; <span>value</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/sim/simapi-reference#SIM2" target="_blank">SIM attribute definition</a>.</Note>

<h4>Example</h4>

<p>&#123;</p>

<p>"iccAttributeId": 11506792,</p>

<p>"iccId": 8953448576492474,</p>

<p>"attribute": "orderId"</p>

<p>"value": "579"</p>

<p>"status": "active"</p>

<p>&#125;</p>

<h2 id="DeletingaspecificSIMattribute">Deleting a specific SIM attribute</h2>

<p>To delete a specific SIM attribute:</p>

<ul> <li> Call <span>DELETE <span>/api/v2/sim/&#123;<var>iccId</var>&#125;/attribute/&#123;<var>iccAttributeId</var>&#125;</span></span>, where &#123;<var>iccId</var>&#125; identifies the parent SIM, and &#123;<var>iccAttributeId</var>&#125; identifies which attribute you want to delete. <br /><span>Path parameters:</span>[CDATA[ ]]<span>iccId</span><b>&#42;</b>; <span>iccAttributeId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API removes the specified SIM attribute. </li> </ul>

<h4>Example</h4>

<p>DELETE https://sim.api.anynetiot.com/api/v2/sim/8953448576492474/attribute/11506792</p>

<h3>Response schema application/json: </h3>

<p>&#123;</p>

<p>"ICC&#95;ATTRIBUTE": 1</p>
