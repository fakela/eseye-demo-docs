---

<p>To view and use the supplied Postman files:</p>

<ol> <li> Download the API collection and environment: <a href="https://docs.eseye.com/Content/Resources/Files/AnyNetSIMAPI.postman_collection_environment.zip" target="_blank" rel="noopener">AnyNetSIMAPI.postman&#95;collection&#95;environment.zip</a> <br />The zip file contains the following files: <ul> <li><span>AnyNetSIMAPI.postman&#95;collection.json</span> </li> <li><span>AnyNetSIMAPI.postman&#95;environment.json</span> </li> </ul> </li> <li>Extract the JSON files into your chosen destination.</li> <li>Import the extracted collection and environment into your Postman application.</li> <li> In the Postman <span>Environments</span> workspace, ensure <span>AnyNet SIM API</span> is the active environment. <br /><img src="/images/resources/images/apis/sim/activeenvironment.png" alt="Activeenvironment" />  </li> <li> In the <span>AnyNetSIM API</span> environment, insert an appropriate <span>Current value</span> for the listed variables. <span><strong>Note: </strong>You will mostly edit the supplied <span>Environment</span> in order to make the <span>AnyNet SIM API collection</span> work for you, apart from editing the collection <span>Body</span> parameters when using <span>Create</span> or <span>Update</span> endpoints.</span> <span><strong>Note: </strong>The referenced objects must already exist in your Infinity account. Alternatively, where possible, create new objects for testing purposes.</span> <table> <col /> <col /> <col /> <thead> <tr> <th>Variable</th> <th>Current value</th> <th>How to find or create value</th> </tr> </thead> <tbody> <tr> <td><span>Token</span> </td> <td>The current access token, built from the authentication variables. Tokens expire after an hour, so you will need to refresh this variable regularly. </td> <td>Use the <span>Get Access Token</span> endpoint in the <span>AnyNet SIM API Collection</span>. For more information, see the steps below.</td> </tr> <tr> <td><span>Base64string</span> </td> <td>The supplied <span>Base64 string</span>, for authentication purposes. </td> <td>Supplied when you requested API access.</td> </tr> <tr> <td><span>clientId</span> </td> <td>The supplied <span>Client ID</span>, for authentication purposes. </td> <td>Supplied when you requested API access.</td> </tr> <tr> <td><span>clientSecret</span> </td> <td>The supplied <span>Client Secret</span>, for authentication purposes. </td> <td>Supplied when you requested API access.</td> </tr> <tr> <td><span>URL</span> </td> <td>The supplied base URL for the APIs, for example <span>sim.api.anynetiot.com</span></td> <td>Supplied when you requested API access.</td> </tr> <tr> <td><span>GetTokenURL</span> </td> <td>The supplied base URL for the Get Access Token endpoint, for example <span>eseye-idp.auth.eu-west-1.amazoncognito.com</span></td> <td>Supplied when you requested API access.</td> </tr> <tr> <td><span>iccId</span> </td> <td>An ICCID, for example <span>8953449647428546</span>.</td> <td> <a href="/api/sim/si-moperations#Viewing" target="_blank">Viewing all SIMs</a>  <span><strong>Note: </strong>SIMs are supplied. You cannot create a new SIM.</span> </td> </tr> <tr> <td><span>iccAttributeId</span> </td> <td>A SIM attribute for the given <span>iccId</span>, for example <span>14180650</span>.</td> <td> <a href="/api/sim/icc-attribute-operations#Viewing" target="_blank">Viewing all SIM attributes</a>  <br /><a href="/api/sim/icc-attribute-operations#Creating" target="_blank">Creating a new SIM attribute</a>  </td> </tr> </tbody> </table> </li> <li> In the Postman <span>Collections</span> workspace, send <span>POST Get Access Token</span> to authenticate your API session. <br />The Response includes the <span>access&#95;token</span>, for example: <br /><img src="/images/resources/images/apis/sim/exampleaccesstoken.png" alt="Exampleaccesstoken" />  </li> <li>Copy the <span>access&#95;token</span>, ensuring that you do not include the quotation marks (as demonstrated in the screenshot above).</li> <li> In the <span>AnyNet SIM API</span> environment, paste the token into the <span>Token</span>[CDATA[ ]]<span>Current value</span>.  <span><strong>Tip: </strong>You may need to clear an existing value first. You must refresh this token every hour.</span> <br /><img src="/images/resources/images/apis/sim/postmanenvironmenttoken.png" alt="Postmanenvironmenttoken" />  </li> <li>Save your changes.</li> <li>Start using the API collection.</li> </ol>
