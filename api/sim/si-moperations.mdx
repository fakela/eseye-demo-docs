<h2 id="ViewingallSIMs">Viewing all SIMs</h2>

<p>To view all SIMs:</p>

<ul> <li> Call <span>GET /api/v2/sim/</span> <span><strong>Tip: </strong>You can apply filters in the request path to limit the results, using one or more parameters. The API returns exactly matching records only. Alternatively, remove all filters from the path to return all records. For example, adding <span><span>?status=active</span></span> to the end of the path will return <span>active</span> objects only.</span> <br />The API returns all the SIMs. </li> </ul>

<h4>Example</h4>

<p><span>GET https://sim.api.anynetiot.com/api/v2/sim/</span> </p>

<p>or</p>

<p><span>GET https://sim.api.anynetiot.com/api/v2/sim?status=active</span> </p>

<h2 id="ViewingaspecificSIM">Viewing a specific SIM</h2>

<p>To view details for a specific SIM:</p>

<ul> <li> Call <span>GET /api/v2/&#123;<var>iccId</var>&#125;</span>, where &#123;<var>iccId</var>&#125; identifies which SIM you want to view. <br /><span>Path parameters:</span> <span>iccId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API returns the specified SIM details. </li> </ul>

<h4>Example</h4>

<p><span>GET https://sim.api.anynetiot.com/api/v2/sim/8953448576492474</span> </p>

<h3>Response schema application/json:</h3>

<p><span>iccId</span>; <span>simId</span>; <span>iccMnoId</span>; <span>iccSpecId</span>; <span>iccTypeId</span>; <span>adminSuspend</span>; <span>systemSuspend</span>; <span>customerSuspend</span>; <span>eId</span>; <span>status</span>; <span>activatedDate</span>; <span>portfolioPackage</span></p>

<Note>For more information, see <a href="/api/sim/simapi-reference#SIM" target="_blank">SIM definition</a>.</Note>

<p>Each SIM has a <span>portfolioPackage</span> array, which includes: </p>

<p><span>packageId</span>; <span>portfolioId</span>; <span>contractStartDate</span>; <span>contractEndDate</span></p>

<Note>For more information, see <a href="/api/sim/simapi-reference#PackageP" target="_blank">SIM API reference tables</a>.</Note>

<p>If a SIM contains attributes, these are returned in the <span>attributes</span> array. Each returned attribute includes:</p>

<p><span>iccAttributeId</span>; <span>iccId</span>; <span>attribute</span>; <span>value</span>; <span>status</span></p>

<Note>For more information, see <a href="/api/sim/simapi-reference#SIM2" target="_blank">SIM attribute definition</a>.</Note>

<h2 id="UpdatingaspecificSIM">Updating a specific SIM</h2>

<p>To update a specific SIM:</p>

<ul> <li> Call <span>PATCH /api/v2/sim/&#123;<var>iccId</var>&#125;</span>, where &#123;<var>iccId</var>&#125; identifies which SIM you want to update. <br /><span>Path parameters:</span> <span>iccId</span><b>&#42;</b> <br /><span>Body:</span> <span>iccMnoId</span>; <span>iccSpecId</span>; <span>iccTypeId</span>; <span>adminSuspend</span>; <span>systemSuspend</span>; <span>customerSuspend</span>; <span>eId</span>; <span>status</span> <span><strong>Note: </strong> JSON keys marked <b>&#42;</b> are mandatory. <br />Do not use <span>null</span> or <span>""</span> (empty) for mandatory parameters. <br />If required, you can update more than one optional parameter. <br />If you are updating the following: <span>iccMnoId</span>, <span>iccSpecId</span>, <span>iccSpecId</span>, then that value must already exist in the system. </span> <br />The API updates the specified SIM parameters with the new details. </li> </ul>

<h4>Example</h4>

<p>PATCH https://sim.api.anynetiot.com/api/v2/sim/8953448576492474</p>

<p>/&#42; header information &#42;/</p>

<p>&#123;</p>

<p>"status": "active"</p>
