<h2 id="ViewingthebillingsnapshotforaspecificSIM">Viewing the billing snapshot for a specific SIM</h2>

<p>To view the billing snapshot for a specific SIM:</p>

<ul> <li> Call <span>GET /api/v2/sim/&#123;<var>iccId</var>&#125;/billingSnapshot</span>, where &#123;<var>iccId</var>&#125; identifies the SIM for which you want to generate a billing snapshot. <br /><span>Path parameters:</span> <span>iccId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API returns the billing details for the specified SIM. </li> </ul>

<h4>Example</h4>

<p><span>GET https://sim.api.anynetiot.com/api/v2/sim/8953448576492474/billingSnapshot</span> </p>

<h3>Response schema application/json:</h3>

<p><span>iccId</span>; <span>simId</span>; <span>price</span></p>

<Note>For more information, see <a href="/api/sim/simapi-reference#billingSnap" target="_blank">Billing snapshot definition</a>.</Note>

<p>Each <span>billingSnapshot</span> includes the following arrays:</p>

<table> <col /> <col /> <thead> <tr> <th>Array</th> <th>Returned parameters</th> </tr> </thead> <tbody> <tr> <td><span>portfolioPackage</span> </td> <td> <span>portfolioId</span>; <span>packageId</span> </td> </tr> <tr> <td><span>packageItemCategory</span> </td> <td> <span>packageItemCategoryId</span>; <span>title</span>; <span>amount</span>; <span>updateTime</span> </td> </tr> </tbody> </table>

<p>If the SIM contains attributes, these are returned in the <span>attributes</span> array. Each returned attribute includes:</p>

<p><span>iccAttributeId</span>; <span>iccId</span>; <span>attribute</span>; <span>value</span>; <span>status</span></p>
