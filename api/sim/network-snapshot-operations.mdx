<p>The SIM API enables you to view a network snapshot for a <a href="#Viewing2">single</a> SIM.</p>

<ul> <li>User required to login</li> <li>Bearer token required</li> </ul>

<h2 id="ViewingthenetworksnapshotforaspecificSIM">Viewing the network snapshot for a specific SIM</h2>

<p>To view the network snapshot for a specific SIM:</p>

<ul> <li> Call <span>GET /api/v2/sim/&#123;<var>iccId</var>&#125;/networkSnapshot</span>, where &#123;<var>iccId</var>&#125; identifies the SIM for which you want to generate a network snapshot. <br /><span>Path parameters:</span> <span>iccId</span><b>&#42;</b> <span><strong>Note: </strong>JSON keys marked <b>&#42;</b> are mandatory.</span> <br />The API returns the network details for the specified SIM. </li> </ul>

<h4>Example</h4>

<p><span>GET https://sim.api.anynetiot.com/api/v2/sim/8953448576492474/networkSnapshot</span> </p>

<h3>Response schema application/json:</h3>

<p><span>iccId</span>; <span>simId</span>; <span>price</span></p>

<Note>For more information, see <a href="/api/sim/simapi-reference#networkSnap" target="_blank">Network snapshot definition</a>.</Note>

<p>Each <span>networkSnapshot</span> includes the following arrays:</p>

<table> <col /> <col /> <thead> <tr> <th>Array</th> <th>Returned parameters</th> </tr> </thead> <tbody> <tr> <td><span>network</span> </td> <td> <span>imsi</span>; <span>rat</span>; <span>mcc</span>; <span>mnc</span>; <span>updateTime</span> </td> </tr> <tr> <td><span>device</span> </td> <td> <span>imei</span>; <span>updateTime</span> </td> </tr> <tr> <td><span>location</span> </td> <td> <span>lat</span>; <span>lon</span>; <span>updateTime</span> </td> </tr> <tr> <td><span>data</span> </td> <td> <span>apn</span>; <span>ip</span>; <span>username</span>; <span>startTime</span>; <span>stopTime</span>; <span>updateTime</span> </td> </tr> <tr> <td><span>portfolioPackage</span> </td> <td> <span>portfolioId</span>; <span>packageId</span> </td> </tr> </tbody> </table>

<p>If the SIM contains attributes, these are returned in the <span>attributes</span> array. Each returned attribute includes:</p>

<p><span>iccAttributeId</span>; <span>iccId</span>; <span>attribute</span>; <span>value</span>; <span>status</span></p>
