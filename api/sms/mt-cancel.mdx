---
title: "Cancelling undelivered SMS messages"
description: "Cancels a sent or queued message when the following conditions are met:"
---

<p>Cancels a sent or queued message when the following conditions are met:</p>

<ul> <li>The authenticated user has access rights to send an SMS</li> <li>If an optional IP address Allowlist exists, the source device is listed on it</li> <li> The SIM is not barred from receiving SMS messages <span><strong>Note: </strong>Enabling or disabling SMS Messages is linked to the SIM package, which is outside the scope of the SMS API.</span> </li> </ul>

<Note>If the message is already delivered, the API cannot cancel it.</Note>

<h2 id="Authentication">Authentication</h2>

<ul> <li>SMS API credentials </li> </ul>
