<p>Delivers Mobile Terminated (MT) SMS messages to an AnyNet or third party SIM when the following conditions are met:</p>

<ul> <li>The authenticated user has access rights to send an SMS</li> <li>If an optional IP address Allowlist exists, the source device is listed on it</li> <li> The SIM is not barred from receiving SMS messages <span><strong>Note: </strong>Enabling or disabling SMS Messages is linked to the SIM package, which is outside the scope of the SMS API.</span> </li> </ul>

<h2 id="MTSMSpathway">MT SMS pathway</h2>

<p>The following diagram displays the main pathways used by mobile terminated SMS, with the codes for the pathway in square brackets. The SMS API serves the message to the AnyNet Messaging Service, which then distributes the message to the correct SIM.</p>

<h3>Creating 16-bit Unicode messages</h3>

<p>You can convert text into Unicode using freely available internet tools, such as <a href="http://smstools3.kekekasvi.com/topic.php?id=288" target="_blank">http://smstools3.kekekasvi.com/topic.php?id=288</a>.</p>

<h4>Example</h4>

<p>The Chinese characters for <span>Hello World</span> are: 你好世界</p>

<ol> <li>Using the <span>SMS Server Tools 3 PDU Converter</span>, select an <span>Alphabet Size</span> of <span>16</span>.</li> <li>Type 你好世界 into the <span>PDU SMS message creator</span> <span>Text</span> box.</li> <li> Select <span>Convert &gt;</span>. <br />The USSD Entry/Display returns the Unicode for 你好世界, which is <span>4F60597D4E16754C</span>.  <br /><img src="/images/resources/images/apis/smsoriginal/pduconverterexample.png" alt="Pduconverterexample" />  </li> <li> Copy and paste the Unicode into your request. </li> </ol>
