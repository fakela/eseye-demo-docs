<p>When an error occurs, an error code and message are generated in the returned result, in XML format:</p>

<p>&lt;statusResponse&gt;</p>

<p>&lt;status&gt;error&lt;/description&gt;</p>

<p> &lt;description&gt;An error occurred&lt;/description&gt;</p>

<p>&lt;/statusResponse&gt;</p>
