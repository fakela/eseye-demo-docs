<p>The Margay API provides high level insight into device connection across different network operators, so that you can observe patterns, problems and anomalies with your network performance. For example, the Margay API can show if all the devices on a particular network have stopped sending or receiving information, or reveal server issues if there is a spike in authentications.</p>

<p>The Margay API provides authentication count and throughput information about your devices over a set period of time. The information is fetched from Eseye Ltd servers.</p>
