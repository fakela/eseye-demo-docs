---
title: "Viewing authentication or data throughput statistics"
description: "Using the Margay web interface, you can view the following:"
---

<p>Using the Margay web interface, you can view the following: </p>

<ul> <li>the number of successful device authentications on a network</li> <li>the rate of successful message delivery (throughput) in bytes on a network </li> </ul>

<p>You can view the statistics over three different time periods:</p>

<ul> <li>Per minute in a selected hour (0 to 60 minutes)</li> <li>Per hour in a selected day (0 to 24h)</li> <li>Per day in a selected month (0 to the number of days in that month)</li> </ul>

<div> <h3>Before you begin</h3> <p>Contact your Eseye Account Manager about using Margay.</p> <p>You need the following:</p> <ul> <li>Username</li> <li>Password</li> </ul> </div>

<h2 id="Loggingin">Logging in</h2>

<ol> <li>Navigate to the Margay portal: <a href="http://margay.eseye.com/Niam/index">http://margay.eseye.com/Niam/index</a>.</li> <li> Type your credentials, then select <span>Sign In</span>. <br />The Margay - Country Selector Page appears. </li> </ol>

<h2 id="Settingthecountrycode">Setting the country code</h2>

<p>Margay displays authentication and throughput information by geographic region using the three-digit Mobile Country Code (MCC). For more information and a list of country codes, see <a href="https://en.wikipedia.org/wiki/Mobile_country_code" target="_blank">https://en.wikipedia.org/wiki/Mobile&#95;country&#95;code</a>. </p>

<p><img src="/images/resources/images/margay/countryselectorpagecustomer.png" alt="Countryselectorpagecustomer" /> </p>

<p>To set the country code:</p>

<ol> <li> Type a country name or code number, or select the code from the <span>MCC</span> drop-down list.  <span><strong>Note: </strong>This field is mandatory.</span> </li> <li> Select either: <table> <col /> <col /> <tbody> <tr> <td> <span>Authentication </span>  </td> <td>  to view the number of devices that have successfully opened a data session in the selected country </td> </tr> <tr> <td> <span>Throughput</span>  </td> <td>  to view successful data rates for devices in the selected country </td> </tr> </tbody> </table> <br />Based on your selection, Margay displays a line graph of the current statistics per minute, generated in the current hour.  <br />Use the key at the bottom of the graph to understand which line colour represents which network operator within the selected country. The key lists the mobile operators by their Mobile Network Code (MNC). <br /> <img src="/images/resources/images/margay/key.png" alt="Key" />  <span><strong>Tip: </strong>Scroll down to see the table of values on which the graph is based.</span> </li> </ol>

<h3>Changing the data time period</h3>

<p>You can view data statistics for the following time periods:</p>

<ul> <li>Per minute in a selected hour (0 to 60 minutes)</li> <li>Per hour in a selected day (0 to 24h)</li> <li>Per day in a selected month (0 to the number of days in that month)</li> </ul>

<Tip>When you log in, the initial graph and table display the number of successful authentications per minute in the current hour.</Tip>

<p>Change the data time period using the navigation bar:</p>

<p><img src="/images/resources/images/margay/timeperiodnavigationbar.png" alt="Timeperiodnavigationbar" /> </p>

<ul> <li>Select <img src="/images/resources/images/margay/returnarrow_20x20.png" alt="Returnarrow 20X20" /> to return to the month, day, or previous hour </li> <li>Select <img src="/images/resources/images/margay/forwardarrow_20x20.png" alt="Forwardarrow 20X20" /> to advance to the next hour</li> </ul>

<p>Alternatively, select a link in the data table to view statistics for that time period:</p>

<p><img src="/images/resources/images/margay/timeperiodlink.png" alt="Timeperiodlink" /> </p>
