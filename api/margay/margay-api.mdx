<p>You can call the Margay API in any API development app using the following commands:</p>

<ul> <li><span>login</span>– see <a href="/api/margay/login">POST /JAPI/Niam/login</a></li> <li><span>getCookieName </span>– see <a href="/api/margay/get-cookie-name">POST /JAPI/Niam/getCookieName</a></li> <li><span>ping </span>– see <a href="/api/margay/ping">POST /JAPI/Niam/ping</a></li> <li><span>getAuthenticationInformation </span>– see <a href="/api/margay/get-authentication-information">POST /JAPI/Niam/getAuthenticationInformation</a></li> <li><span>getThroughputInformation </span>– see <a href="/api/margay/get-throughput-information">POST /JAPI/Niam/getThroughputInformation</a></li> <li><span>logout </span>– see <a href="/api/margay/logout">POST /JAPI/Niam/logout</a></li> </ul>

<Note>Margay API information is grouped by MCC and MNC.</Note>

<p>All information is sent in JSON.</p>

<Note>Use plain text to send requests.</Note>

<Note>All Eseye API examples are written for Postman. For information about using Postman, see the <a href="https://docs.eseye.com/Content/Resources/Files/8314-Postman-User-Guide.pdf" target="_blank">8314 Postman User Guide (PDF)</a>.</Note>

<p>Using Postman, if required, you can select <span>Code</span> to generate code snippets.</p>
