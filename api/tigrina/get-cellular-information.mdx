<p>Retrieves a set of cellular WAN information relating to a specific Hera 604 router.</p>

<h2 id="Requestkey">Request key</h2>

<table> <col /> <col /> <col /> <thead> <tr> <th> JSON key </th> <th> Data type </th> <th> Description </th> </tr> </thead> <tbody> <tr> <td> ESN </td> <td> 20 digit hex </td> <td> The router electronic serial number </td> </tr> </tbody> </table>

<h3>Request example</h3>

<p>&#123;</p>

<p>“ESN”: “12345678901234567890”</p>

<p>&#125;</p>
