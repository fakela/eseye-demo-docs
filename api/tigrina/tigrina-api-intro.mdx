<p>The Hera 604 is a 2G, 3G, and 4G enabled Wi-Fi router with embedded Eseye AnyNet Secure SIM card to provide connectivity worldwide. You can monitor, access and reconfigure the Hera 604 remotely.</p>

<p>The Eseye Tigrina API enables you to access the following Hera 604 information:</p>

<ul> <li>Device parameters</li> <li>Cellular connectivity information</li> <li>Network information</li> <li>Cellular RSSI history</li> </ul>

<p>This data is captured when you first start each Hera 604 router, and stored in the ACS server. By default, each Hera 604 router is polled every eight hours for updates. You can configure the polling period – for more information, speak to Support. </p>

<h2 id="Sessionbasedaccess">Session-based access</h2>

<p>You must log in to access your router information. Each session is authenticated using your Tigrillo account. </p>

<p>After login is authenticated, the system uses server sessions to maintain access. You can only access information about the Hera 604 devices that have active SIMs on your Tigrillo account.</p>

<h3>Production URL</h3>

<p>The customer interface makes an HTTP POST request to the following URL:</p>

<p>https://tigrina.eseye.com/Japi/Diam/<var>&lt;function&gt;</var></p>

<p>where <span>&lt;function&gt;</span> is the accessed API function.</p>

<p>For example, to request the Hera 604 cellular information:</p>

<p>https://tigrina.eseye.com/Japi/Diam/getCellularInformation</p>
