<p>You can call the Tigrina API in any API development app, using the following commands:</p>

<ul> <li><span>login</span> – for more information, see <a href="/api/tigrina/login">POST /Japi/Diam/login</a>.</li> <li><span>getDeviceInformation</span> – for more information, see <a href="/api/tigrina/get-device-information">GET /Japi/Diam/getDeviceInformation</a></li> <li><span>getCellularInformation</span> – for more information, see <a href="/api/tigrina/get-cellular-information">GET /Japi/Diam/getCellularInformation</a>.</li> <li><span>getNetworkInformation</span> – for more information, see <a href="/api/tigrina/get-network-information">GET /Japi/Diam/getNetworkInformation</a>.</li> <li><span>getCellularRSSIHistory</span> – for more information, see <a href="/api/tigrina/get-cellular-rssi-history">GET /Japi/Diam/getCellularRSSIHistory</a>.</li> <li><span>logout</span> – for more information, see <a href="/api/tigrina/logout">POST /Japi/Diam/logout</a></li> </ul>

<p>All information is sent in JSON.</p>

<Note>Examples are written for Postman. For more information about using Postman, see the <a href="https://eseye.zendesk.com/hc/en-us/articles/360013066514-Postman-API-user-guide">Postman API user guide</a>.</Note>

<p>Using Postman, if required, you can select Code to generate code snippets.</p>
