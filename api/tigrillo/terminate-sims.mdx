<p>Terminates multiple SIMs, and assigns them a pendingDeprovision status. If terminated (unprovisioned), the SIM will stop connecting to the network. Service charges will stop unless the SIM is in contract, in which case the customer is liable for the rest of the service charges up until the end date of the contract. Termination is not instant – Eseye must verify the termination.In some instances, you can reprovision a terminated SIM. For more information, see <a href="/api/tigrillo/activate-sims">POST /Japi/Tigrillo/activateSIMs</a>. Allow two hours for the reprovisioning process to complete. </p>
