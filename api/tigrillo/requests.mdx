<p>You can call the Tigrillo API in any API development app using the following commands:</p>

<ul> <li><span>login </span>– see <a href="/api/tigrillo/login">POST /Japi/Tigrillo/login</a></li> <li><span>getCookieName </span>– <a href="/api/tigrillo/get-cookie-name">POST /Japi/Tigrillo/getCookieName</a></li> <li><span>ping </span>– see <a href="/api/tigrillo/ping">POST /Japi/Tigrillo/ping</a></li> <li><span>getSIMs </span>– see <a href="/api/tigrillo/get-sims">POST /Japi/Tigrillo/getSIMs</a></li> <li><span>getSIMStateOptions </span>– see <a href="/api/tigrillo/get-sim-state-options">POST /Japi/Tigrillo/getSIMStateOptions</a></li> <li><span>setSIMState </span>– see <a href="/api/tigrillo/set-sim-state">POST /Japi/Tigrillo/setSIMState</a></li> <li><span>activateSIMs </span>– see <a href="/api/tigrillo/activate-sims">POST /Japi/Tigrillo/activateSIMs</a></li> <li><span>suspendSIMs </span>– see <a href="/api/tigrillo/suspend-sims">POST /Japi/Tigrillo/suspendSIMs</a></li> <li><span>unsuspendSIMs </span>– see <a href="/api/tigrillo/unsuspend-sims">POST /Japi/Tigrillo/unsuspendSIMs</a></li> <li><span>terminateSIMs </span>- see <a href="/api/tigrillo/terminate-sims">POST /Japi/Tigrillo/terminateSIMs</a></li> <li><span>setSIMDetail </span>– see <a href="/api/tigrillo/set-sim-detail">POST /Japi/Tigrillo/setSIMDetail</a></li> <li><span>getSIMBilling </span>– see <a href="/api/tigrillo/get-sim-billing">POST /Japi/Tigrillo/getSIMBilling</a></li> <li><span>getSIMLastActivity </span>– see <a href="/api/tigrillo/get-sim-last-activity">POST /Japi/Tigrillo/getSIMLastActivity</a></li> <li><span>logout </span>– see <a href="/api/tigrillo/logout">POST /Japi/Tigrillo/logout</a></li> </ul>

<p>All information is sent in JSON.</p>

<Note>Use plain text to send requests.</Note>

<Note>All Eseye API examples are written for Postman. For information about using Postman, see the <a href="https://docs.eseye.com/Content/Resources/Files/8314-Postman-User-Guide.pdf" target="_blank">8314 Postman User Guide (PDF)</a>.</Note>

<p>Using Postman, if required, you can select <span>Code</span> to generate code snippets.</p>
