<p>Logs the user into Infinity Classic. </p>

<h2 id="Authentication">Authentication</h2>

<ul> <li>User account must exist in Infinity Classic</li> <li>PortfolioID</li> </ul>

<h2 id="Headers">Headers</h2>

<p>For information about headers, see <a href="/api/tigrillo/headers">Configuring headers</a>.</p>
