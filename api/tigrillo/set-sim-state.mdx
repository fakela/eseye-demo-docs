title: "POST /Japi/Tigrillo/setSIMState"
description: "Changes the state of a single SIM to either of the following states:"
openapi: "/api-specs/legacy/tigrillo.json POST /Japi/Tigrillo/setSIMState"
---

<p>Changes the state of a single SIM to either of the following states:</p>

<ul> <li><span>suspended</span> – displayed when you <span>suspend</span> a SIM using this method</li> <li><span>available</span> – displayed when you <span>terminate</span> a SIM using this method. The SIM will cycle through <span>pendingDeprovision</span> &gt; <span>unprovisioned</span> &gt; <span>available</span>. If the SIM is <span>pendingDeprovision</span>, you cannot change its state.</li> </ul>

<p>If you want to set the state of multiple SIMs, use any of the following methods:</p>

<ul> <li>activateSIMs – see <a href="/api/tigrillo/activate-sims">POST /Japi/Tigrillo/activateSIMs</a>.</li> <li>suspendSIMs – see <a href="/api/tigrillo/suspend-sims">POST /Japi/Tigrillo/suspendSIMs</a>.</li> <li>unsuspendSIMs – see <a href="/api/tigrillo/unsuspend-sims">POST /Japi/Tigrillo/unsuspendSIMs</a>.</li> <li>terminateSIMs – see <a href="/api/tigrillo/terminate-sims">POST /Japi/Tigrillo/terminateSIMs</a>.</li> </ul>
