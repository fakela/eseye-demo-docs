<p>Unsuspends multiple SIMs that are currently suspended.</p>

<p>Do not send an unsuspend request until the SIM card state is "suspended", as these may be discarded or delayed.</p>

<h2 id="Authentication">Authentication</h2>

<ul> <li>User required to login </li> </ul>
