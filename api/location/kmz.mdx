<p>Returns a compressed Keyhole Markup Language file detailing the Google Earth location of one or more SIMs. You will need to install Google Earth or Google My Maps in order to import the file and view the mapped locations.</p>

<p><span>kmz</span> – Keyhole Markup Language Zipped</p>

<ul> <li>User required to login</li> </ul>

<h3>Test request using <span>Send</span></h3>

<p>If you test the request using <span>Send</span>, then a successful response is similar to the following:</p>

<p> <img src="/images/resources/images/apis/location/kmztestrequest_650x104.png" alt="Kmztestrequest 650X104" /> </p>

<h3>Download file using <span>Send and Download</span></h3>

<p>If you download the location map file using <span>Send and Download</span>, then a successful response immediately opens a dialog box to save the file to your default <span>Downloads</span> folder:</p>

<p><img src="/images/resources/images/apis/location/savefile.png" alt="Savefile" /> </p>

<ol> <li>Add a <span>.kml</span> extension to the <span>File name</span>.</li> <li>Browse to where you want to save the file, then select <span>Save</span>.</li> <li>Import the file into <span>Google Earth</span> or <span>Google My Maps</span> to view the SIM locations.</li> </ol>
