<p>The Push API streams live raw SIM network data from various data sources to a customer's web-based endpoint, so that customers can monitor and administer their SIM estate network usage. </p>

<Note>Eseye makes data available in real-time with a slight delay, and does not supply historic data. You must store any data you want to retain.</Note>

<p>The PUSH API returns network information for all SIMs that are linked to your Infinity Classic account. </p>

<Note>The information returned does not include information from SIMs assigned to child accounts. For example, if a customer reseller account (with their own SIM customers) requests data, it will not include information for SIMs that belong to their customer accounts.</Note>

<h2 id="Datastreamservices">Data stream services</h2>

<p>You can subscribe to one or more of the following network data stream services:</p>

<ul> <li> NetFlow Raw Push – aggregates a wide variety of IP network traffic data per customer SIM, including:  <ul> <li>Traffic source and destination IP addresses and port numbers</li> <li>The number of bytes and IP packets observed in the traffic flow</li> </ul> <br />Network administrators can analyse the data to understand the traffic flow and volume across their SIM estate network, which may inform business decisions. For example, a business may need to purchase new hardware in areas of high usage to ease traffic flow congestion, or retire a thing that is never used. </li> <li> RADIUS Authentication – controls a user's network resource access. Network administrators can use the data to track network and device usage, and analyse networking issues. </li> <li>RADIUS Accounting – accurately measures a RADIUS network user's start and stop times and traffic usage. This information is used for billing purposes and general network monitoring..</li> </ul>

<p>Each subscription to a data steam service is independent and therefore can be assigned a different customer endpoint (URL).</p>
