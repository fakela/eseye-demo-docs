<p>After you have created the Foundation CloudFormation stack, you can view the AWS resources that are deployed with the CloudFormation<span>anynet-secure-foundation-stack</span> template.</p>

<p>To view the Foundation CloudFormation template:</p>

<ol> <li> Using AWS Management Console, select the <span>CloudFormation</span> service. <br />The current stacks are listed. </li> <li> Select the <span>anynet-secure-foundation-stack</span> option. <br />The stack overview appears. </li> <li> Select the <span>Template</span> tab to view a list of AWS resources. <br /> <img src="/images/resources/images/iris/stacktemplate_302x103.png" alt="Stacktemplate 302X103" />  </li> </ol>
