---
title: "Unsubscribing from AnyNet Cellular Connectivity for AWS IoT"
description: "To cancel your AnyNet Cellular Connectivity for AWS IoT subscription:"
---

<p>To cancel your AnyNet Cellular Connectivity for AWS IoT subscription:</p>

<ol> <li>Using AWS Management Console, search for <span>AWS Marketplace Subscriptions</span>.</li> <li>In the <span>AnyNet Cellular Connectivity for AWS IoT</span> card, select <span>Manage</span>.</li> <li> In the <span>Actions</span> drop-down list, select <span>Cancel subscription</span>. <br />A Cancel subscription message box appears, asking you to confirm that you want to cancel your subscription. </li> <li>Select the <span>I understand that I will continue to be charged for all running software</span> checkbox.</li> <li> Select <span>Yes, cancel subscription</span>. <br />The AnyNet Cellular Connectivity for AWS IoT subscription is removed. </li> </ol>
