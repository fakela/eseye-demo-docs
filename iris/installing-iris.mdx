<div> <h3>Before you begin</h3> <p>To configure AnyNet IRIS you need:</p> <ul> <li>AWS Marketplace Customer ID</li> <li>AWS IAM user credentials</li> </ul> </div>

<Accordion title="Installing AnyNet IRIS for Windows 10">
 <ol> <li> On the AnyNet IRIS Welcome page, select: <br /><img src="/images/resources/images/iris/windows-ic.png" alt="Windows Ic" />  <br />The AnyNet IRIS Setup.exe downloads. </li> <li> Run <span>AnyNet IRIS Setup &lt;<var>version</var>&gt;.exe</span>, where &lt;<var>version</var>&gt; is the latest version. <span><strong>Note: </strong>Depending on your Windows security, a warning may appear. Ensure that you run the <span>AnyNet IRIS Setup &lt;version&gt;.exe</span> app anyway.</span> <br />AnyNet IRIS installs and opens. </li> <li> Next, configure AnyNet IRIS. <br />For more information, see<a href="#Configur">Configuring AnyNet IRIS</a>. </li> </ol> 
</Accordion>

<Accordion title="Installing AnyNet IRIS for macOS">
 <ol> <li> On the AnyNet IRIS Welcome page, select: <br /><img src="/images/resources/images/iris/mac_ic.png" alt="Mac Ic" />  <br />The AnyNet IRIS-&lt;<var>version</var>&gt;.dmg file downloads. </li> <li>Double-click <span>AnyNet IRIS-&lt;<var>version</var>&gt;.dmg</span>, where &lt;<var>version</var>&gt; is the latest version.</li> <li>Drag the AnyNet IRIS icon into the <span>Applications</span> folder.</li> <li>In the <span>Applications</span> folder, double-click the AnyNet IRIS application to open it.</li> <li> Next, configure AnyNet IRIS. <br />For more information, see<a href="#Configur">Configuring AnyNet IRIS</a>. </li> </ol> 
</Accordion>

<Accordion title="">
 <p>To configure AnyNet IRIS:</p> <ol> <li> Using the AnyNet IRIS application, on the <span>AWS credentials</span> page, type the IAM user credentials, then select <span>Next</span>. <br />You can find the user credentials in the Download.csv file. For more information, see <a href="/iris/iam-permissions/create-ia-museraccount#CSV">Download.csv file</a>. <br />Do not use your AWS root user account to configure AnyNet IRIS. For more information, see <a href="/iris/iam-permissions/create-ia-museraccount" target="_blank">Creating a dedicated IAM user account for AnyNet IRIS</a>. <br />If required, follow the onscreen instructions to create an access key.  <span><strong>Tip: </strong>You may prefer to copy and paste the credentials using keyboard shortcuts.</span> </li> <li> On the <span>Customer ID</span> page, type the AWS Marketplace<span>Customer ID</span> for AnyNet IRIS. <br />If required, follow the onscreen instructions to discover your Customer ID. </li> <li>Select <span>Next</span>.</li> <li> On the <span>Foundation Stack</span> page, select the region where you want to deploy the initial Foundation CloudFormation stack. <br />The options displayed in the drop-down list depend on the IAM permissions and regions you set up when you created the custom IAM user account. </li> <li>If required, select <span>Stack Preview</span> to expand the preview and ensure the stack is compliant with your organisation's stack requirements.</li> <li> Select <span>Next</span> to deploy the stack to the selected region. <br />The initial stack enables Eseye cross-account access to your customer account. <br /> <img src="/images/resources/images/iris/success_ic.png" alt="Success Ic" /> indicates the deployment has completed successfully. <br /> <img src="/images/resources/images/iris/failed_ic.png" alt="Failed Ic" /> indicates the deployment has failed, which may occur because AWS is not ready. Try again. If required, select the supplied link on the <span>Foundation Stack</span> page to view the full log files. If you cannot progress, contact Support: <a href="mailto:cloudconnect@eseye.com">cloudconnect@eseye.com</a>. </li> <li>Select <span>Continue</span>.</li> <li> On the <span>Email Notifications</span> page, type the email addresses where you want to send AnyNet IRIS advisory and alert notifications, then select <span>Next</span>. <br />AnyNet IRIS emails each supplied email address with a verification link.  </li> <li> Open each email, then select the <span>Confirm subscription</span> link to verify that email address. <br />The AWS Subscription confirmed page appears for each subscription. The AnyNet IRIS Email Notifications page updates. <br /> <img src="/images/resources/images/iris/success_ic.png" alt="Success Ic" /> indicates successful subscription. <br /> <img src="/images/resources/images/iris/failed_ic.png" alt="Failed Ic" /> indicates the subscription has failed. If you cannot progress, contact Support: <a href="mailto:cloudconnect@eseye.com">cloudconnect@eseye.com</a>. </li> <li>Using AnyNet IRIS, select <span>Continue</span>.</li> <li> On the <span>Service Region</span> page, select the AWS IoT regions where you want to deploy AnyNet IRIS resources. </li> <li> Select <span>Complete</span> to deploy the resource stack to the selected regions. <br />This may take a few minutes to complete. </li> <li> Select <span>Review</span> to view Configuration, Status and Updates information. </li> </ol> 
</Accordion>

<Tip>For information about deploying IAM roles and policies, see <a href="/iris/about-iris">About AnyNet IRIS</a>.</Tip>

<Tip>Use AWS CloudFormation to view the deployed stacks.</Tip>

<Tip>For information about reviewing thing connectivity, including viewing Cloud Formation template contents, and managing configuration and updates, see <a href="/iris/review-overview" target="_blank">Reviewing thing connectivity using AnyNet IRIS</a>.</Tip>
