<p>The AnyNet Cellular Connectivity for AWS IoT service and AnyNet IRIS app require the following AWS Managed Policies.</p>

<Note>When you attach these policies to the IAM user, you will also create an inline policy to deny all user and organization related actions. For more information, see <a href="/iris/iam-permissions/attach-policies-to-user">Attaching IAM policies to an existing user</a>.</Note>

<p>Ensure that enabling access to these policies does not breach your IT security procedures.</p>
