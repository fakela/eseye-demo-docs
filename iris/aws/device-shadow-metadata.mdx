---
title: "Viewing Device Shadow metadata"
---

<ol> <li> Using AWS IoT Console, select <span>Manage</span> &gt; <span>Things</span>. <br />The <span>Things</span> section appears, listing all your thing s. </li> <li>Select a thing to view the <span>Details</span> page.</li> <li> In the left-hand menu, select <span>Shadow</span>. <br /><img src="/images/resources/images/shadowarn.png" alt="Shadowarn" />  </li> </ol>

<Tip>For information about updating the Shadow, see <a href="/iris/aws/shadow-manual-update">Manually update the Device Shadow metadata</a>.</Tip>
