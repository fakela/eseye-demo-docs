<p>At initial registration, each device receives a unique certificate and policies that deliver security and identity information. You can view certificates and policies using the AWS IoT Console.</p>

<ol> <li> Using AWS IoT Console, select <span>Manage</span> &gt; <span>Things</span>. <br />The <span>Things</span> section appears, listing all your thing s. </li> <li>Select a thing to view the <span>Details</span> page.</li> <li> In the left-hand menu, select <span>Security</span>. <br /><img src="/images/resources/images/awssecurity.png" alt="Awssecurity" />  <br />One or more certificates are listed by their alphanumeric strings. </li> <li>Select a string to view the certificate details.</li> <li> In the left-hand menu, select <span>Policies</span>. <br /><img src="/images/resources/images/aws_policies.png" alt="Aws Policies" />  <br />One or more policies are listed by their policy name. </li> <li>Select a policy to view the details.</li> </ol>
