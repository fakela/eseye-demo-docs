<p>Using AWS IoT Core, you can add shadows to AWS IoT thing s. Shadows provide a shared data store for devices, apps, and other cloud services to access and control the current state of a device. This information is available in the cloud when the device itself is offline. AWS things can have multiple shadows to enable devices to connect to other apps and services. For more information, see: <a href="https://docs.aws.amazon.com/iot/latest/developerguide/iot-device-shadows.html">AWS IoT Device Shadow service</a>.</p>

<p>Use the AWS device shadow to check AnyNet Secure SIM data for a specific thing, including status, location, usage, and details about the certificate delivery.</p>

<p>For more information, see <a href="/iris/aws/device-shadow-metadata">Viewing Device Shadow metadata</a>.</p>
