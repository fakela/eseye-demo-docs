<h2>Viewing configuration settings</h2>

<p>Using the <span>Configuration</span> tab, you can view the settings you chose when you installed AnyNet IRIS.</p>

<p> <img src="/images/resources/images/iris/configurationtab_325x138.png" alt="Configurationtab 325X138" /> </p>

<p>To update the displayed email addresses:</p>

<ol> <li> Using the left-hand menu, select <span>Notifications</span> to change the advisory and alert recipient email addresses. <br /> <img src="/images/resources/images/iris/irismenu_151x109.png" alt="Irismenu 151X109" />  </li> <li>Make your changes, then select <span>Next</span> to update the email addresses.</li> </ol>

<p>To add more regions:</p>

<ol> <li> Using the left-hand menu, select <span>Service Region</span>. <br />The current selected regions are highlighted in blue and have an Integration Completed status <img src="/images/resources/images/iris/success_ic.png" alt="Success Ic" />, for example: <br /> <img src="/images/resources/images/iris/selectedregions.png" alt="Selectedregions" />  </li> <li>Select one or more regions to add them to the list.</li> <li>Select <span>Complete</span> to update your choices.</li> </ol>

<p>To delete a selected region:</p>

<ol> <li> Select a region to deselect it. </li> <li>Select <span>Complete</span> to update your choices.</li> </ol>
