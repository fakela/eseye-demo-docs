<h2>Viewing status settings</h2>

<p>Using the<span>Status</span> tab, you can view the following events across your IoT estate:</p>

<ul> <li><span>Activated</span> – the device SIM has registered on the network and enabled sending and receiving data</li> <li><span>Unable to Deliver Certificate</span> – the Eseye identification and security certificate failed to reach the device after multiple attempts</li> <li><span>Suspended</span> – the SIM can no longer send or receive data</li> <li><span>Unable to Suspend</span> – the SIM is in an intermediate state and cannot be suspended until it changes state</li> </ul>

<p> <img src="/images/resources/images/iris/status_302x101.png" alt="Status 302X101" /> </p>

<p>You can filter the displayed events by <span>Date</span>, <span>Thing</span>, and <span>Region</span>.</p>

<p>To filter by Date:</p>

<ol> <li> Select the<span> Date</span> drop-down list. <br />Three options are displayed. <br /> <img src="/images/resources/images/iris/datefilter_302x221.png" alt="Datefilter 302X221" />  </li> <li> Select any of the following: <ul> <li><span>Last 7 Days</span> to display the previous week's Status, up to and including today</li> <li><span>Last 30 days</span> to display the previous month's Status, up to and including today</li> <li><span>Custom Range</span> to select your own dates</li> </ul> </li> <li>Select <span>Update</span> to refresh the returned results.</li> </ol>

<p>To filter by thing:</p>

<ol> <li> Select the <span>Thing</span> drop-down list. <br /> <img src="/images/resources/images/iris/thingfilter_302x62.png" alt="Thingfilter 302X62" />  </li> <li>Type the thing name.</li> <li>Select <span>Update</span> to refresh the returned results.</li> </ol>

<p>You can filter the results by all regions or one specific region.</p>

<p>To filter by region:</p>

<ol> <li> Select the <span>Region</span> drop-down list. <br /> <img src="/images/resources/images/iris/regionfilter_302x91.png" alt="Regionfilter 302X91" />  </li> <li>Select a specific region from the drop-down list, or select <span>All</span> to display all regions.</li> <li>Select <span>Update</span> to refresh the returned results.</li> </ol>
