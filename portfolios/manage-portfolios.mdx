<p>A <span>portfolio</span> is a collection of a customer's account information, including the company name, billing information such as tax and account currency details, associated portal, postal addresses, and user information.</p>

<Note>You will only have access to managing portfolios if you have the relevant permissions.</Note>

<p>Depending on your permissions, you can view, create, activate, update and delete portfolios, addresses and users. For more information, see <a href="/portfolios/viewing-portfolios" target="_blank">About viewing the current portfolio in Infinity</a>.</p>

<Note>For user roles and permissions, see <a href="/permissions/about-user-roles" target="_blank">About user roles</a>.</Note>
