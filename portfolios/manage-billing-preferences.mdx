<Note>This read-only page is accessible only to the users with <span>Admin</span> or <span>Finance</span> Infinity roles. For more information, see Integra Roles. </Note>

<p> Billing preferences are settings that define how the customer's account is billed. These preferences allows you to view invoice delivery details, payment methods, and billing information.</p>

<h2 id="Viewingthebillingpreferences">Viewing the billing preferences</h2>

<p>To view the Infinity billing preferences:</p>

<ol> <li> On the left menu, select <span>Finance</span>. <br /> <img src="/images/resources/images/infinity/financeicon-only_80x78.png" alt="Financeicon Only 80X78" />  </li> <li> Select <span>Billing Preferences</span>.  <br /><img src="/images/resources/images/infinity/billingpreferences.png" alt="Billingpreferences" />  <br />The <span>Billing Preferences</span> page contains the following information: <table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Portfolio Name</span> </td> <td> A friendly name for identifying the portfolio, usually the company name followed by a geographic region or project. </td> </tr> <tr> <td><span>Parent Portfolio ID</span> </td> <td> The unique identifier for the super portfolio. </td> </tr> <tr> <td><span>Billing Name</span> </td> <td> A short friendly description for the business code. </td> </tr> <tr> <td><span>Billing Currency</span> </td> <td> The currency assigned to the portfolio for billing purposes, which is the three digit ISO 4217 alphabetic code. </td> </tr> <tr> <td><span>Tax Reference</span> </td> <td> The unique tax code identifier for the company assigned to the portfolio, which informs the VAT rates that are applied to the invoice. Tax codes, together with business codes, inform how tax is handled on the invoice. </td> </tr> <tr> <td><span>Company Id</span> </td> <td> A unique identifier assigned to the company when it is officially registered with a government authority or regulatory body. </td> </tr> <tr> <td><span>Order Reference</span> </td> <td> The customer's Purchase Order reference that appears on the invoice. </td> </tr> <tr> <td><span>Preferred Payment Method</span> </td> <td> The customer's preferred way to pay invoices, using either: <ul> <li>Direct debit</li> <li>AWS </li> <li>Invoice</li> <li>Credit card</li> </ul> </td> </tr> <tr> <td><span>Billing Address</span> </td> <td> The customer's billing address for receiving invoices and correspondence. <span><strong>Note: </strong> This is a read-only field. To edit this address, see <a href="/support/manage-tickets#Creating" target="_blank">Creating a ticket</a>.</span> </td> </tr> </tbody> </table> </li> </ol>

<p>To sort the search results view by column name:</p>

<ul> <li>Select a column name to sort the results in ascending (a-z; 1-9) or descending (z-a; 9-1) order.</li> </ul>

<h2 id="Updatingthebillingpreferences">Updating the billing preferences</h2>

<p>You can update the following fields:</p>

<ul> <li>Tax Reference</li> <li>Company Id</li> <li>Order Reference</li> </ul>

<p>To update the billing preferences:</p>

<ol> <li> On the <span>Billing Preferences</span> page, replace the existing values as required. <br /><img src="/images/resources/images/infinity/billingpreferences.png" alt="Billingpreferences" />  </li> <li> Select <span>Save</span> to commit your changes. </li> </ol>
