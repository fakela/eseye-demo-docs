<p>To access the current portfolio in Infinity:</p>

<ol> <li> On the left menu, select <span>Portfolio</span>. <br /> <img src="/images/resources/images/infinity/portfoliomgmticon_80x78.png" alt="Portfoliomgmticon 80X78" />  </li> <li> Select <span>Your Portfolio</span> to view an overview of the current portfolio, including address and user information.  <br /><img src="/images/resources/images/infinity/youraccountpage.png" alt="Youraccountpage" />  <table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Portfolio name</span> </td> <td> A friendly name for identifying the portfolio, usually the company name followed by a geographic region or project. </td> </tr> <tr> <td><span>Company</span> </td> <td> A unique identifier assigned to the company when it is officially registered with a government authority or regulatory body. </td> </tr> <tr> <td><span>Country</span> </td> <td> The name of the country where the address to send the invoice is located. </td> </tr> <tr> <td><span>Billing Currency</span> </td> <td> The currency assigned to the portfolio for billing purposes, which is the three digit ISO 4217 alphabetic code. </td> </tr> <tr> <td><span>Addresses</span> </td> <td> The addresses associated with the current portfolio.  </td> </tr> <tr> <td><span>Users</span> </td> <td> The users associated with the current portfolio, listed by: <br /><span>Username</span> – The unique user account name for the current portfolio. <br /><span>Name</span> – The user first and last names. <br />The user account <span>Status</span>, either: <ul> <li><span>Active</span>: Users who can log in and access the portal.</li> <li><span>Inactive</span>: Users whose account is suspended because they have not interacted with the portal for a specified period of time.</li> <li><span>Verified</span>: Users who have authenticated their account via the referral code link.</li> <li><span>Unverified</span>: Users who have not yet authenticated their account via the referral code link. </li> </ul> <br /><span>Email</span> – A valid business email address, where the user receives emails. </td> </tr> </tbody> </table> </li> </ol>
