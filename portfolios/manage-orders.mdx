<h3>Order processing</h3>

<p>Integra retrieves the order details by extracting the order ID for the product, updates the order status, and generates an email in the background to initiate the shipping process. It also adds the tracking number to the order information.</p>

<Note>If the customer placed the order in Infinity, an order is automatically created in Integra with a status of <span>Ordered</span>. Integra updates the order to check and complete the order details.</Note>

<Tip>For Telus customers, their backend team can update the order status.</Tip>

<h2 id="Updatingorders">Updating orders</h2>

<p>If the details for an existing order require to be amended, the order is updated in Integra.</p>

<p>An existing order is amended in any of the below scenarios:</p>

<ul> <li>Rectify errors – If there was a mistake in the original order (for example, incorrect quantities, wrong items, or pricing errors).</li> <li>Adhere to customer requests – Customers contacted Support to change details after placing an order, such as modifying shipping addresses, changing payment methods, or updating item selections.</li> <li>Adjust inventory – If an item is out of stock or unavailable, staff may need to substitute it with a different product or adjust the order accordingly.</li> <li>Modify shipping – Changes in shipping preferences or options, such as expedited shipping, may require order adjustments.</li> <li>Cancel an order – If the order is no longer required.</li> <li>Returns and exchanges – If a customer returns an item and requests an exchange or refund, staff might need to edit the order to reflect these changes accurately.</li> </ul>
