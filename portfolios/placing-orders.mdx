<p>To access Orders in Infinity:</p>

<ul> <li>On the left menu, select <span>Order Management</span> to display the <span>Orders List</span>.</li> </ul>

<p> <img src="/images/resources/images/infinity/ordermanagement-icon.png" alt="Ordermanagement Icon" /> </p>

<p> </p>

<h4>Order statuses </h4>

<p>A new order is created when a customer places an order in Infinity or by Sales Admin creating an order in Lynx in response to a PO or other request.</p>

<p>The following table describes the order states.</p>

<table> <col /> <col /> <thead> <tr> <th>Status</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Ordered</span> </td> <td>An order has been created by placing an order in Infinity.</td> </tr> <tr> <td><span>Processing</span> </td> <td>Integra is processing the order. For more information, see<a href="#Order">Placing an order using Infinity</a>.</td> </tr> <tr> <td><span>Shipped</span> </td> <td> The Logistics team has shipped the hardware items for the order. </td> </tr> <tr> <td><span>Returned by </span><span href="#Carrier">Carrier</span> </td> <td>Package was returned to the sender (merchant or warehouse) by the shipping carrier.</td> </tr> <tr> <td><span>Delivered</span> </td> <td>The order is delivered to the customer who had placed the order.</td> </tr> <tr> <td><span>Cancelled</span> </td> <td>The order was invalid.</td> </tr> <tr> <td><span>Return Raised</span> </td> <td> A return process is initiated after the product was successfully delivered to the customer. This is due to a faulty or defective product that did not meet the customer's expectations.</td> </tr> <tr> <td><span>Return Received</span> </td> <td> The seller, merchant, or warehouse has received the product that the customer sent back after initiating a return.</td> </tr> <tr> <td><span>Return Completed</span> </td> <td>The entire return is successfully processed.</td> </tr> <tr> <td><span>Return Rejected</span> </td> <td> The seller or merchant has declined the customer's return request after receiving and reviewing the returned item. This occurs for various reasons, such as: <ul> <li>Product does not meet return criteria: The item does not comply with the seller's return policy either because it was damaged, or missing the original packaging or accessories.</li> <li>Expired return window: The return was initiated after the allowable return period had expired, resulting in the rejection.</li> <li>Non-refundable items: Some products, like customised or perishable goods, are not eligible for returns, and if these were returned, the request is denied.</li> <li>Evidence of tampering or improper use by the customer.</li> </ul> </td> </tr> </tbody> </table>

<h2 id="Viewingorders">Viewing orders</h2>

<p>To view orders:</p>

<ol> <li> On the left menu, select <span>Order Management</span>. </li> <li> Select <span>Order history</span> to view the <span>Orders List</span>.  </li> </ol>

<Note>From the <span>Orders list</span>, you can view orders by applying <a href="#About"> The Search filters list contain the following fields:</a>.</Note>

<h3>Filtering orders</h3>

<p>The search filters provide different ways to filter the list of orders and only show results which match the criteria defined in <b>any</b> of the applied search filters.</p>

<p><img src="/images/resources/images/integra/searchtable-orders.png" alt="Searchtable Orders" /> </p>

<p>To search for the order that matches the specified criteria:</p>

<ol> <li> Enter values for any of the Search filters. <h6> The <span>Search filters</span> list contain the following fields:</h6> <table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td> <span>Order Id</span>  </td> <td>Enter the <a href="#Order4">Order Id</a>.</td> </tr> <tr> <td><span>Status</span> </td> <td>Select from the <a href="#Status">Status</a> drop-down list.</td> </tr> </tbody> </table> </li> <li> Select <span>Filter</span> to display the Search results table matching the specified <a href="#About">Search filter attributes</a>. </li> <li> Select the <a href="#Order4">Order Id</a> to view the <span>Summary</span> and <span>Order details</span> pages for that specific item.  <br /><img src="/images/resources/images/infinity/orderdetails.png" alt="Orderdetails" />  <span><strong>Note: </strong>Select <span>Back to History</span> to return to the <span>Orders List</span>. </span> </li> <li>Select <span>Reset</span> to clear the specified values from the search filters.</li> </ol>

<h2 id="Creatinganorder">Creating an order</h2>

<p>You can create an order to customize and configure equipment to meet your specific requirements, obtain new hardware, order SIMs for activation with your new mobile device, replace faulty SIMs, and more.</p>

<Note>Orders for SIMs or for hardware that includes a SIM, such as a Hera router must specify the package in which the SIMs are activated against.</Note>

<p>To create an order in Infinity:</p>

<ol> <li> On the left menu, select <span>Order Management</span>. </li> <li> Select <span>New order</span> to view all the available products. <br /><img src="/images/resources/images/infinity/ordercreate.png" alt="Ordercreate" />  </li> <li>From the list of items under the relevant tab, select <span>More Details</span> to access the purchasing page for that particular product.</li> <li> Select the <span>SIM</span>, <span>Router</span>, or <span>Accessories</span> tab depending on your selection. <ul> <li> Enter the necessary details for the selected <a href="#SIM">SIM</a>, <a href="#Router">Router</a>, <a href="#Accessor">Accessories</a> attributes: <table> <col /> <col /> <thead> <tr> <th>Attribute</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td colspan="2"><span>SIM</span> </td> </tr> <tr> <td> <span>Package</span>  </td> <td> The short unique label (no spaces) for identifying the package. <br />Select the preferred package to associate with the SIM from the package drop-down list. </td> </tr> <tr> <td> <span>Pre-activated</span>  </td> <td> Activating SIMs after delivery, ensuring they are ready to use as soon as they arrive. This guarantees that the SIMs are functional immediately upon receipt. <ul> <li>Select <span>Yes</span> to activate the SIMs after delivery for immediate use.</li> <li>Select <span>No</span> to activate the SIM, using the Infinity portal or API. </li> </ul> </td> </tr> <tr> <td> <span>Substitution allowed</span>  </td> <td> Sometimes one or more items are not in stock when the order is processed. If you allow substitutions (for example, if you order a package and do not specify SIM type), we will offer a suitable alternative if this is available. <ul> <li>Select <span>Yes</span> to receive a similar alternative if the item is unavailable.</li> <li>Select <span>No</span> to wait until the item is back in stock (if unavailable).</li> </ul> </td> </tr> <tr> <td><span>Quantity</span> </td> <td> Number of units charged. <br />Use the scroll bar to select the quantity, with a minimum value set at 25. </td> </tr> <tr> <td colspan="2"><span>Router</span> </td> </tr> <tr> <td><span>Configuration</span> </td> <td> The unique product configuration identifier. <br />To apply any additional configuration to your router, select it from the configuration drop-down list. </td> </tr> <tr> <td><span>Quantity</span> </td> <td> Number of units charged. <br />Use the scroll bar to select the quantity, with a minimum value set at 1. </td> </tr> <tr> <td colspan="2"><span>Accessories</span> </td> </tr> <tr> <td><span>Quantity</span> </td> <td> Number of units charged. <br />Use the scroll bar to select the quantity, with a minimum value set at 1. </td> </tr> </tbody> </table> </li> </ul> </li> <li> Select <span>Add to cart</span> to receive a green notification that the selected item is added to the cart. <br /><img src="/images/resources/images/infinity/moredetails.png" alt="Moredetails" />  </li> <li> To purchase more items, select <span>Continue shopping</span> and return to the <span>Orders List</span> page. <span><strong>Note: </strong>Consequently, the newly added item appears in the <span>Orders List</span> on the <span>Orders History</span> page, with a unique <a href="#Order4">Order Id</a> assigned to it. For more information, see <a href="#Viewing">Viewing orders</a>.</span> </li> <li> Select <img src="/images/resources/images/infinity/cart-icon_80x47.png" alt="Cart Icon 80X47" /> to display the <span>Summary</span> page. <br /><img src="/images/resources/images/infinity/summarypage-orders.png" alt="Summarypage Orders" />  </li> <li> Select <span>Checkout</span> and provide your payment details to complete the purchase. </li> </ol>

<h3>Adding items to an order</h3>

<p>To add items to an existing order in Infinity:</p>

<ol> <li>  From the list of items, select <span>More Details</span> to access the purchasing page for the item you wish to add. </li> <li> Select <span>Add to Cart</span> to receive a green notification that the selected item is added to the cart. <br /><img src="/images/resources/images/infinity/moredetails.png" alt="Moredetails" />  <span><strong>Note: </strong>Similarly, follow the above procedure to add another item to the shopping cart. For more information, see <a href="#AddingOrderItems">Creating an order</a>.</span> </li> </ol>
