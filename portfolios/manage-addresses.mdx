title: "About managing portfolio addresses"
---

<p> </p>

<p>The <span>Addresses page</span> displays the different address types associated with the portfolio where each address type has a distinct role in the management of customer accounts.</p>

<ul> <li> Billing: where to send invoices. <span><strong>Note: </strong>The billing address is used for invoicing and should only ever be one address. This ensures that all billing-related communications are directed to the correct location.</span> </li> <li> Deployment: address to where to deploy goods and services. </li> <li> Shipping: where to deliver orders. <br /><img src="/images/resources/images/integra/yourportfolioaddresstab.png" alt="Yourportfolioaddresstab" />  <span><strong>Note: </strong>Deployment addresses are utilized for testing and configuring services, whereas shipping addresses can vary and are used to send products to various customer locations. This approach enhances the efficiency of logistics management.</span> </li> </ul>

<h3>Adding addresses</h3>

<p>To add new addresses:</p>

<ol> <li> Select <span>Add</span> alongside the type of address to display the <span> Add Address page</span>.  <br /><img src="/images/resources/images/integra/add-deployment address.png" alt="Add Deployment Address" />  </li> <li> Provide values or choose from the respective drop-down list for the <a href="#About2">attributes</a> in the <span>Add Address page</span>.  <h4>About <span> Add Address</span> attributes</h4> <table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Address Name</span> </td> <td> Friendly name. A human readable label that defines the address.  </td> </tr> <tr> <td><span>Company Name</span> </td> <td> The company that is located at this portfolio address. </td> </tr> <tr> <td><span>Country</span> </td> <td>The name of the country where the address to send the invoice is located.</td> </tr> <tr> <td><span>Address line 1</span> </td> <td> The street address (including the building number and street name) or the P.O. Box. </td> </tr> <tr> <td><span>Address line 2</span> </td> <td> The apartment or suite number, building name, or floor. </td> </tr> <tr> <td><span>Town / City</span> </td> <td>The town or city where the portfolio address is located.</td> </tr> <tr> <td><span>County / State</span> </td> <td>  The county, state or province where the portfolio address is located. </td> </tr> <tr> <td><span>Postal code</span> </td> <td> A short alphanumeric string that identifies a mail delivery area. </td> </tr> <tr> <td colspan="2"><span>Attributes</span>: Same for all selected portfolios</td> </tr> <tr> <td><span>orderReference</span> </td> <td> The customer's Purchase Order reference that appears on the invoice. </td> </tr> <tr> <td><span>taxReference</span> </td> <td> The customer's unique tax identifier, used for tracking and managing tax obligations, filings, and payments. <p>To set the active address as default:</p>

<ol> <li> Select <span>Make Default</span> alongside the respective address.  <br /><img src="/images/resources/images/integra/make default-address.png" alt="Make Default Address" />  </li> <li> Wait until you receive the green notification that the chosen address is successfully made as the default address. <span><strong>Tip: </strong>The <span>Make Default</span> option is available only for active addresses.</span> </li> </ol>

<h3>Deleting address</h3>

<p>To delete the address:</p>

<ol> <li> Select <span>Delete</span> alongside the respective address.  <br /><img src="/images/resources/images/integra/deleteaddress.png" alt="Deleteaddress" />  </li> <li> Select <span>Remove</span> to delete the selected address. </li> </ol>
