<p>The Settings menu provides the following pages which enable you to view package information, create alerts, generate reports and define device models:</p>

<ul> <li><a href="#SimPackages">SIM Packages page</a> </li> <li><a href="#Alert">Alert Profiles page</a> </li> <li><a href="#Device">Device Models page</a> </li> <li><a href="#Reporting">Reporting Profile page</a> </li> <li><a href="#Messaging">Messaging API page</a> </li> <li><a href="#Audit">Audit Log page</a> </li> </ul>

<h2 id="SIMPackagespage">SIM Packages page</h2>

<p>The SIM packages page lists all of the packages associated with this account, and enables you to view and edit package information. </p>

<p><img src="/images/resources/images/infinityclassic/simpackages.png" alt="Simpackages" /> </p>

<h3>Search</h3>

<p>Use <span>Show</span> to filter the packages to display: Select All, Active, Suspended, Legacy, Unsigned.</p>

<p>Select <span>Search</span> to update the records in the results table.</p>

<h3>Results</h3>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />), export to CSV (<img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<p>The following table describes the columns displayed for the SIM Audit Log records.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td> Package ID </td> <td>Unique identifier for the <a href="/billing/billing-packages">package</a>.</td> </tr> <tr> <td> Package Name </td> <td> Name of the package which typically contains information about the package items. </td> </tr> <tr> <td>Status</td> <td> The status of the package:  <ul> <li>Active</li> <li>Suspended</li> <li>Unsigned</li> <li>Legacy</li> </ul> </td> </tr> <tr> <td>No. of SIMs</td> <td> The number of SIMs assigned to the package. </td> </tr> <tr> <td>Actions</td> <td>  <img src="/images/resources/images/infinityclassic/iconedit_24x24.png" alt="Iconedit 24X24" /> – edit package details (friendly name, status) <br /> <img src="/images/resources/images/infinityclassic/iconpdfblack_24x24.png" alt="Iconpdfblack 24X24" /> – download <a href="/billing/billing-packages#About2">service contracts</a>. </td> </tr> </tbody> </table>

<h2 id="AuditLogpage">Audit Log page</h2>

<p>The Settings Audit Log displays a record of every action taken, and change made, to the settings associated with your account. </p>

<p><img src="/images/resources/images/infinityclassic/settingsauditlog.png" alt="Settingsauditlog" /> </p>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />), export to CSV (<img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<h3>Search</h3>

<p>The search enables you to filter the list of Settings audit log records between a <span>From</span> and <span>To</span> date. </p>

<p>Select the calendar icon (<img src="/images/resources/images/infinityclassic/iconcalendar_24x24.png" alt="Iconcalendar 24X24" />) to display a calendar from which you can select the required date.</p>
