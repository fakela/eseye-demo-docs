<p>The Account menu provides the following menus which enable you to view and edit information specific to your user account:</p>

<ul> <li><a href="#Account">Account Details page</a> </li> <li><a href="#Notifications">Notifications (My Account Settings) page</a> </li> <li><a href="#Reports">Account Reports page</a> </li> <li><a href="#Audit">Account Audit Log page</a> </li> <li>Logout – logs the current user out of the Infinity Classic portal.</li> </ul>

<h2 id="AccountDetailspage">Account Details page</h2>

<p>The Account Details page enables you to view and edit your user details and password.</p>

<p><img src="/images/resources/images/infinityclassic/accountdetails.png" alt="Accountdetails" /> </p>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />), export to CSV (<img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<p>The following table describes the fields displayed on the account details page.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td>User Name</td> <td> The username used to log in to the Infinity Classic portal. <br />Must be between 6 and 32 characters in length, and should contain only letters, digits, '-', '&#95;', or be a valid email address. </td> </tr> <tr> <td>Firstname</td> <td> First name of the user. Must be at least two characters long. </td> </tr> <tr> <td>Surname</td> <td> Surname of the user. Must be at least two characters long. </td> </tr> <tr> <td>Job Title</td> <td> The job title of the user. This is useful to identify their required user privileges. </td> </tr> <tr> <td>Business Sector</td> <td>Description of the business sector in which the user's company operates.</td> </tr> <tr> <td>Email Address</td> <td> The email address for the user.  </td> </tr> <tr> <td>Phone Number</td> <td> A contact number for the user. </td> </tr> <tr> <td><span>Edit Details</span> </td> <td>Select to edit your user account details. </td> </tr> <tr> <td><span>Change Password</span> </td> <td>Select to change your password.</td> </tr> </tbody> </table>

<h2 id="AccountReportspage">Account Reports page</h2>

<p>The Accounts Reports page enables and updates the frequency of <a href="/infinity-classic/managing-reports">reports</a>.</p>

<p><img src="/images/resources/images/infinityclassic/accountreports.png" alt="Accountreports" /> </p>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />), export to CSV (<img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<p>The following table describes the fields on the <span>Accounts Reports</span> page.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td>Report</td> <td> Displays one of the following reports: <ul> <li><span>Admin Reports</span> <ul> <li><span>Account Audit Report</span> – date and time stamps user access to the account, as well as notifications sent. Use this PDF report to track Infinity Classic access and who is responsible for received notifications.</li> <li><span>Account Snapshot Report</span> – lists portfolio details, including active users, current packages, total data usage. This report gives you a broad understanding of the data your SIM card estate consumes, and who is accessing the estate.</li> <li><span>Packages Audit Report</span> – date and time stamps when a signed Service Contract form is uploaded on a SIM Package, and the user responsible for the change.</li> <li><span>Profiles Audit Report</span> – date and time stamps user activity on the Reporting Profile, such as when a new profile is created or edited.</li> </ul> </li> <li><span>Finance Reports</span> <ul> <li><span>Account Statement Report</span> – billing statement that enables you to quickly see any outstanding balance owed to Eseye. Your connectivity will cease functioning when bills are unpaid.</li> <li><span>Finance Audit Report</span> – date and time stamps user changes to billing information, such as password and address updates, or payment method approval and so on.</li> </ul> </li> <li><span>Logistics Reports</span> <ul> <li><span>Logistics Audit Report</span> – date and time stamps user activity on an order, such as when the order is placed, shipping prepared, or dispatched.</li> </ul> </li> <li><span>Engineering Reports</span> <ul> <li><span>Messaging Audit Report</span> – date and time stamps user activity on the SMS API or against an individual SIM card, such as password and MO POST URL changes.</li> <li><span>Networking Audit Report</span> – details network changes, such as assigned IP addresses. This report gives system administrators visibility of network changes to help assess the impact of those changes on devices.</li> </ul> </li> </ul> <br />For information about reports, see <a href="https://docs.eseye.com/Content/Resources/Files/8680-Available-Reports-Overview.pdf" target="_blank">8680 Available Reports Overview (PDF)</a>. </td> </tr> <tr> <td>Enable</td> <td>Select to enable the report generation.</td> </tr> <tr> <td>Interval</td> <td>Specify the frequency of report generation: Daily, Weekly or Monthly.</td> </tr> <tr> <td>Action</td> <td>Select <span>Run Now</span> to generate the report.</td> </tr> <tr> <td><span>Update</span> </td> <td>Select to save the changes.</td> </tr> </tbody> </table>

<h2 id="AccountAuditLogpage">Account Audit Log page</h2>

<p>The Account Audit Log page enables you to search for all changes made on the Account menu. </p>

<p><img src="/images/resources/images/infinityclassic/accountauditlog.png" alt="Accountauditlog" /> </p>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />), export to CSV (<img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<h3>Search</h3>

<p> Filter the list of account audit log records between the <span>From</span> and <span>To</span> dates. </p>

<p>Select the calendar icon (<img src="/images/resources/images/infinityclassic/iconcalendar_24x24.png" alt="Iconcalendar 24X24" />) to display a calendar from which you can select the required date.</p>

<p>Select <span>Search</span> to update the records in the results table.</p>

<h3>Results</h3>

<h5>Viewing more results</h5>

<p>Some pages return results in a continuous list. Select <span> View More</span> at the end of the list to see more items.</p>

<p>The following table lists the fields in the results table on the account audit log page.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><span>Description</span> </td> <td> The account event or change. </td> </tr> <tr> <td><span>Date</span> </td> <td> The date (YY-MM-DD) of the audit log event. </td> </tr> <tr> <td><span>Ref</span> </td> <td> The user action or username of the account.  </td> </tr> <tr> <td><span>Information</span> </td> <td>Detailed description about the change made.</td> </tr> </tbody> </table>
