<p> The following audit reports give you visibility on your user interactions with the SIAM portfolio and the SIM card estate:</p>

<h2 id="AccountAuditReportPDF">Account Audit Report (PDF)</h2>

<p>Date and time stamps user access to the account, as well as notifications sent. Use this file to track Infinity Classic access and who is responsible for received notifications.</p>

<p><img src="/images/resources/images/management/reports/accountaudit.png" alt="Accountaudit" /> </p>

<p>To generate the report in Infinity Classic:</p>

<ol> <li> From the menu, select <span>Account</span> &gt; <span>Reports</span> to display the <a href="/infinity-classic/account#Reports">Account Reports page</a>. </li> <li>Select the <span>Enable</span> checkbox for each report you want to generate.</li> <li>  Select <span>Run Now</span> to generate the report. </li> </ol>

<p>To find the report in Infinity Classic:</p>

<ol> <li>From the menu, select <span>Activity &gt; Reports</span> to display the <a href="/infinity-classic/activity#Reports">Reports page</a>. </li> <li>Alongside <span>Account Audit Report</span>, select <img src="/images/resources/images/infinityclassic/iconpdfblack_24x24.png" alt="Iconpdfblack 24X24" />.</li> </ol>

<h2 id="SIMAuditReportPDF">SIM Audit Report (PDF)</h2>

<p>To generate the report in Infinity Classic:</p>

<h2 id="SIMAuditLogperSIMPDFCSV">SIM Audit Log – per SIM (PDF, CSV)</h2>

<p>Details every action taken and change made to a selected SIM card by description, date, user and reference. For example, a new SIM audit log entry is added when a SIM is queued for provisioning or for every usage alert that is triggered on the SIM.</p>

<p><img src="/images/resources/images/management/reports/simauditlogpersim.png" alt="Simauditlogpersim" /> </p>

<p>To find and generate the report in Infinity Classic:</p>

<ol> <li> From the menu, select <span>SIMs</span> &gt; <span>SIM List</span>. </li> <li>Alongside a SIM card, select <img src="/images/resources/images/infinityclassic/simactioniconaudit_24x24.png" alt="Simactioniconaudit 24X24" /> to display the <a href="/infinity-classic/sim-actions#Audit">SIM Audit Log</a>.</li> <li>  To download the information in a particular format, select either: <img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" /> or <img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />. </li> </ol>
