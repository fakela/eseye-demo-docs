<p>The Ordering menu enables you to manage previous and new orders using the following pages:</p>

<ul> <li><a href="#Shop">Shop page</a> </li> <li><a href="#Cart">Cart page</a> </li> <li><a href="#Order">Order History page</a> </li> <li><a href="#Delivery">Delivery Address page</a> </li> </ul>

<h2 id="Shoppage">Shop page</h2>

<p>The Shop page displays a list of products you can buy directly through the Infinity Classic portal.</p>

<Note>For a full list of available products or services, contact your Eseye Account Manager.</Note>

<p><img src="/images/resources/images/infinityclassic/orderingshop.png" alt="Orderingshop" /> </p>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />), export to CSV (<img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<p>The following table describes the fields on the Shop page.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td>Order Code</td> <td> Code that uniquely identifies the order.  </td> </tr> <tr> <td>Description</td> <td> Textual description of the package which typically includes data limits or available services. </td> </tr> <tr> <td>Price</td> <td>The cost of purchasing a single order.</td> </tr> <tr> <td>Qty</td> <td>The quantity of the item you wish to purchase.</td> </tr> <tr> <td>Action</td> <td>  <img src="/images/resources/images/infinityclassic/orderingiconcart_24x24.png" alt="Orderingiconcart 24X24" /> – adds the order to the <a href="#Cart">shopping cart</a>. </td> </tr> </tbody> </table>

<h2 id="Cartpage">Cart page</h2>

<p>The Cart page displays the items currently in the shopping cart. </p>

<p><img src="/images/resources/images/infinityclassic/orderingcart.png" alt="Orderingcart" /> </p>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />), export to CSV (<img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<p>The following table describes the fields on the shopping cart page.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td>Description</td> <td> Textual description of the package which typically includes data limits or available services. </td> </tr> <tr> <td>Qty</td> <td> The quantity of the item you wish to purchase. <span><strong>Note: </strong>If you update a quantity, select the <img src="/images/resources/images/infinityclassic/orderingiconupdate_24x24.png" alt="Orderingiconupdate 24X24" /> action to update the shopping cart.</span> </td> </tr> <tr> <td>Amount</td> <td>Displays the cost of each item and the total cost of the order.</td> </tr> <tr> <td>Action</td> <td>  <img src="/images/resources/images/infinityclassic/orderingiconupdate_24x24.png" alt="Orderingiconupdate 24X24" /> – select to update any changes made to the row item in the shopping cart. <br /> <img src="/images/resources/images/infinityclassic/orderingicondelete_24x24.png" alt="Orderingicondelete 24X24" /> – select to remove the item from the cart. </td> </tr> <tr> <td><span>Checkout</span> </td> <td> Select to proceed to the checkout page where you must: <ul> <li>Supply or edit the delivery address and contact details.</li> <li>Specify a billing address.</li> <li>Edit order-specific delivery options</li> <li>Review and place your order.</li> </ul> </td> </tr> </tbody> </table>

<h2 id="OrderHistorypage">Order History page</h2>

<p>The Order History page enables you to search for and view all of the orders in the last four months.</p>

<p><img src="/images/resources/images/infinityclassic/orderingorderhistory.png" alt="Orderingorderhistory" /> </p>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />), export to CSV (<img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<h3>Search</h3>

<p>The search enables you to filter the order history.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td> From <br />To </td> <td>  Filter the list of order records between the <span>From</span> and <span>To</span> dates.  <br />Select the calendar icon (<img src="/images/resources/images/infinityclassic/iconcalendar_24x24.png" alt="Iconcalendar 24X24" />) to display a calendar from which you can select the required date. </td> </tr> <tr> <td>Show</td> <td> Select which types of orders are shown. By default, all order types (New, Pending and Dispatched) are shown. </td> </tr> <tr> <td><span>Search</span> </td> <td>Select to filter the order history records the values in the From, To and Show fields. </td> </tr> </tbody> </table>

<h3>Results</h3>

<p>The following table describes the fields on the Order History page.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td>Reference</td> <td> The unique reference number for the order. Select the reference number to display delivery details and addresses. </td> </tr> <tr> <td>Date</td> <td> The date of the order. </td> </tr> <tr> <td>User</td> <td>The user that placed the order.</td> </tr> <tr> <td>Status</td> <td>The status of the order: New, Pending or Dispatched.</td> </tr> </tbody> </table>

<h2 id="DeliveryAddresspage">Delivery Address page</h2>

<p>The Delivery Address page is where you view and edit the device address and delivery contact details.</p>

<p><img src="/images/resources/images/infinityclassic/orderingdeliveryaddress.png" alt="Orderingdeliveryaddress" /> </p>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />), export to CSV (<img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<p>The following table describes the fields on the Delivery Address page.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><b>Delivery Address Details</b> </td> <td>   </td> </tr> <tr> <td>Company Name</td> <td> The name of the company that placed the order. </td> </tr> <tr> <td>Address Line 1</td> <td> The first line of the address to which the order will be sent. </td> </tr> <tr> <td>Address Line 2</td> <td> The second line of the address to which the order will be sent. </td> </tr> <tr> <td>Town or City</td> <td>The town or city to which the order will be sent.</td> </tr> <tr> <td>State or County</td> <td>The state or county to which the order will be sent.</td> </tr> <tr> <td>Postcode</td> <td>The postcode to which the order will be sent.</td> </tr> <tr> <td>Country</td> <td>The country to which the order will be sent.</td> </tr> <tr> <td><span>Delivery Contact Details</span> </td> <td> </td> </tr> <tr> <td>Contact Name</td> <td> The name of the delivery contact who will receive this order.  </td> </tr> <tr> <td>Telephone Number</td> <td> A contact number for the user. </td> </tr> <tr> <td><span>Edit</span> </td> <td> Select <span>Edit</span> to update the delivery address and delivery contact details. </td> </tr> </tbody> </table>
