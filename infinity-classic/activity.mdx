<p>The Activity menu provides the following pages which enable you to view notification, report and alert information:</p>

<ul> <li><a href="#Notifications">Notifications page</a> </li> <li><a href="#Alerts">Alerts page</a> </li> <li><a href="#Reports">Reports page</a> </li> </ul>

<h2 id="Notificationspage">Notifications page</h2>

<p>The Notifications page displays all of the notification messages sent.</p>

<p><img src="/images/resources/images/infinityclassic/notifications.png" alt="Notifications" /> </p>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />), export to CSV (<img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<h3>Search</h3>

<p>The search enables you to filter the notifications.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td> From <br />To </td> <td>  Filter the list of notifications between the <span>From</span> and <span>To</span> dates.  <br />Select the calendar icon (<img src="/images/resources/images/infinityclassic/iconcalendar_24x24.png" alt="Iconcalendar 24X24" />) to display a calendar from which you can select the required date. </td> </tr> <tr> <td>Show</td> <td> Select which notification classifications to show. By default, all activities (Network, Management and Application) are shown. </td> </tr> <tr> <td><span>Search</span> </td> <td>Select to filter the notification records using the values in the From, To and Show fields. </td> </tr> </tbody> </table>

<h3>Results</h3>

<p>The following table describes the columns displayed for the SIM Audit Log records.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td> Date Time </td> <td> The date and time of the notification. </td> </tr> <tr> <td> Classification </td> <td>Whether the notification is related to network, management or application information.</td> </tr> <tr> <td>Severity</td> <td> The severity of the notification: Minor, Major or Severe. </td> </tr> <tr> <td>Message</td> <td> The notification text. </td> </tr> </tbody> </table>

<h2 id="Alertspage">Alerts page</h2>

<p>The Alerts page displays read-only information about Finance, Data, Messaging and IMEI Alerts. Alerts are also sent via e-mails to Engineering users.</p>

<p><img src="/images/resources/images/infinityclassic/alerts.png" alt="Alerts" /> </p>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />), export to CSV (<img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<h3>Search</h3>

<p>The search enables you to filter the alerts.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td> From <br />To </td> <td>  Filter the list of alerts between the <span>From</span> and <span>To</span> dates.  <br />Select the calendar icon (<img src="/images/resources/images/infinityclassic/iconcalendar_24x24.png" alt="Iconcalendar 24X24" />) to display a calendar from which you can select the required date. </td> </tr> <tr> <td>Show</td> <td> Select which alerts type to show. By default, all types (Finance, Data, Messaging, IMEI and Calls) are shown. </td> </tr> <tr> <td>ICCID</td> <td>  Integrated Circuit Card Identifier for SIM cards or profiles (when using an eUICC SIM). For more information, see <a href="/getting-started/esim/profiles#top" target="_blank">About eUICC SIM profiles</a>. <span><strong>Note: </strong>Contains 14 and 20 numeric digits and starts with 89 or 99.</span> </td> </tr> <tr> <td><span>Search</span> </td> <td>Select to filter the notification records using the values in the From, To and Show fields. </td> </tr> </tbody> </table>

<h3>Results</h3>

<p>The following table describes the columns displayed for the SIM Audit Log records.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td> Date Time </td> <td> The date and time of the alert. </td> </tr> <tr> <td> Type </td> <td> Whether the alert is about Finance, Data, Messaging, IMEI or Calls. </td> </tr> <tr> <td>ICCID</td> <td>  Integrated Circuit Card Identifier for SIM cards or profiles (when using an eUICC SIM). For more information, see <a href="/getting-started/esim/profiles#top" target="_blank">About eUICC SIM profiles</a>. <span><strong>Note: </strong>Contains 14 and 20 numeric digits and starts with 89 or 99.</span> </td> </tr> <tr> <td>Subject</td> <td> Whether reaching this limit triggered an alert or a suspension </td> </tr> </tbody> </table>
