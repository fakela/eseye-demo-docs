<p>The Finance menu enables you to make payments, edit billing details and view transaction history using the following pages:</p>

<ul> <li><a href="#Overview">Overview page</a> </li> <li><a href="#Details">Details page</a> </li> <li><a href="#Transactions">Transactions page</a> </li> <li><a href="#Audit">Audit Log page</a> </li> </ul>

<h2 id="Overviewpage">Overview page</h2>

<p>The Overview page enables you to download forms to change the contractual name and payment methods, and to make payments.</p>

<p><img src="/images/resources/images/infinityclassic/financeoverview.png" alt="Financeoverview" /> </p>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<p>The following table describes the fields on the Overview page.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td>Contractual Name</td> <td> The name of the portfolio who is liable for this account.  <br /> <img src="/images/resources/images/infinityclassic/financeiconedit_24x24.png" alt="Financeiconedit 24X24" /> – select to download the party name change request form, which you must complete and upload to request the name change. </td> </tr> <tr> <td><b>Finance Settings</b> </td> <td> </td> </tr> <tr> <td>Payment method</td> <td> Displayed the current payment method, which is one of: <ul> <li>Invoiced – payments are made in response to invoices. </li> <li>Credit Card (Monthly) – Eseye will contact you to confirm credit card details.</li> <li>Direct Debit (Monthly) – Eseye will contact you and request you to complete a DirectDebit form.</li> </ul> <br /> <img src="/images/resources/images/infinityclassic/financeiconedit_24x24.png" alt="Financeiconedit 24X24" /> – select to edit the payment method and select <span>Request change</span>. </td> </tr> <tr> <td>Email Statement</td> <td> Select the check box to enable receiving monthly statements via email.  <br />If you change this setting, select the update icon (<img src="/images/resources/images/infinityclassic/financeiconupdate_24x24.png" alt="Financeiconupdate 24X24" />) to request the changes. </td> </tr> <tr> <td><b>Statement</b> </td> <td> </td> </tr> <tr> <td>Amount due</td> <td>The amount overdue on this account. </td> </tr> <tr> <td>Account Balance</td> <td>The total amount outstanding on this account.</td> </tr> <tr> <td>Amount</td> <td>The amount of the payment made using the Make Ad-hoc Payment by Credit Card button.</td> </tr> <tr> <td><b>Make Ad-hoc Payment by Credit Card</b> </td> <td>Select to make a payment to Eseye for the amount in the Amount field above.</td> </tr> </tbody> </table>

<h2 id="Detailspage">Details page</h2>

<p>The Details page enables you to view and edit billing details.</p>

<p><img src="/images/resources/images/infinityclassic/financedetails.png" alt="Financedetails" /> </p>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />), export to CSV (<img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<p>The following table describes the fields on the Details page.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td><b>Billing Address Details</b> </td> <td> </td> </tr> <tr> <td>Billing Company</td> <td> The name of the company that placed the order. </td> </tr> <tr> <td>Address Line 1</td> <td> The first line of the address to which the order will be sent. </td> </tr> <tr> <td>Address Line 2</td> <td> The second line of the address to which the order will be sent. </td> </tr> <tr> <td>Town or City</td> <td>The town or city to which the order will be sent.</td> </tr> <tr> <td>State or County</td> <td>The state or county to which the order will be sent.</td> </tr> <tr> <td>Postcode</td> <td>The postcode to which the order will be sent.</td> </tr> <tr> <td>Country</td> <td>The country to which the order will be sent.</td> </tr> <tr> <td><b>Billing Contact Details</b> </td> <td> </td> </tr> <tr> <td>Contact Name</td> <td> The name of the billing contact who will pay for this order.  <span><strong>Note: </strong>This should be a finance specific contact. For example, an employee in the accounts department.</span> </td> </tr> <tr> <td>Telephone</td> <td> A contact number for the billing contact. </td> </tr> <tr> <td>Email Address</td> <td>Email address of the billing contact. </td> </tr> <tr> <td><b>Billing References</b> </td> <td> </td> </tr> <tr> <td>VAT reference</td> <td>The VAT reference number for the company.</td> </tr> <tr> <td>Company number</td> <td>The unique company registration number for the company.</td> </tr> <tr> <td>PO reference</td> <td>The unique reference number for the purchase order.</td> </tr> <tr> <td><span>Edit</span> </td> <td>Select to edit the billing details.</td> </tr> </tbody> </table>

<h2 id="Transactionspage">Transactions page</h2>

<p>The Transactions page displays a list of historical transaction records.</p>

<p><img src="/images/resources/images/infinityclassic/financetransactions.png" alt="Financetransactions" /> </p>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />), export to CSV (<img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<h3>Search</h3>

<p>The search enables you to filter the transaction records.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td> From <br />To </td> <td>  Filter the list of transaction records between the <span>From</span> and <span>To</span> dates.  <br />Select the calendar icon (<img src="/images/resources/images/infinityclassic/iconcalendar_24x24.png" alt="Iconcalendar 24X24" />) to display a calendar from which you can select the required date. </td> </tr> <tr> <td>Show</td> <td> Select which transaction records are shown. By default, all activities (Data, SMS and Voice) are shown. </td> </tr> <tr> <td><span>Search</span> </td> <td>Select to filter the billing records using the values in the From, To and Show fields. </td> </tr> </tbody> </table>

<h3>Results</h3>

<h5>Viewing more results</h5>

<p>Some pages return results in a continuous list. Select <span> View More</span> at the end of the list to see more items.</p>

<p>The following table describes the fields in the results table on the Transactions page.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td>Type of transaction</td> <td> The type of transaction, which is one of the following: <ul> <li>Credit Note</li> <li>Cheque</li> <li>DirectDebit</li> <li>Bank Transfer</li> <li>Credit Card</li> <li>Debit Card</li> <li>Cash</li> <li>Standing Order</li> <li>Journal</li> <li>Payment</li> <li>Invoice</li> </ul> </td> </tr> <tr> <td>Transaction reference number</td> <td> Unique identifier for this transaction. </td> </tr> <tr> <td>Date of transaction</td> <td>The date of the transaction.</td> </tr> <tr> <td>Due date, only for invoices</td> <td>The due date for payments made after receiving invoices.</td> </tr> <tr> <td>Amount (in the agreed currency)</td> <td>The amount of money in the transaction (in the agreed currency).</td> </tr> <tr> <td>Tax</td> <td>The amount of tax in the transaction (in the agreed currency).</td> </tr> <tr> <td>Balance (in the agreed currency)</td> <td>The balance (in the agreed currency) before the transaction was made.</td> </tr> </tbody> </table>

<h2 id="AuditLogpage">Audit Log page</h2>

<p>The Audit Log page enables you to search for all changes made on the Finance menu. </p>

<p><img src="/images/resources/images/infinityclassic/financeauditlog.png" alt="Financeauditlog" /> </p>

<p>Select the export to PDF (<img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" />), export to CSV (<img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />) or print (<img src="/images/resources/images/infinityclassic/iconprint_24x24.png" alt="Iconprint 24X24" />) buttons to export or print the information displayed on the page.</p>

<h3>Search</h3>

<p> Filter the list of audit log records between the <span>From</span> and <span>To</span> dates. </p>

<p>Select the calendar icon (<img src="/images/resources/images/infinityclassic/iconcalendar_24x24.png" alt="Iconcalendar 24X24" />) to display a calendar from which you can select the required date.</p>

<p>Select <span>Search</span> to update the records in the results table.</p>

<h3>Results</h3>

<h5>Viewing more results</h5>

<p>Some pages return results in a continuous list. Select <span> View More</span> at the end of the list to see more items.</p>

<p>The following table describes the fields in the results table on the audit log page.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td>Description</td> <td> The type of change made. </td> </tr> <tr> <td>Date</td> <td> The date of the change. </td> </tr> <tr> <td>User</td> <td>The user who made the changes.</td> </tr> <tr> <td>Reference</td> <td>The reference number for the change.</td> </tr> <tr> <td>Information</td> <td>Textual description of the change.</td> </tr> </tbody> </table>
