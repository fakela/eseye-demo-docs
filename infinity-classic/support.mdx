<p>The Support page is where you raise new, and manage existing, support tickets.</p>

<h2 id="Ticketspage">Tickets page</h2>

<p>The Tickets page lets you search all of your support tickets.</p>

<p><img src="/images/resources/images/infinityclassic/supporttickets.png" alt="Supporttickets" /> </p>

<h3>Search</h3>

<p> Filter the list of support tickets created between the <span>From</span> and <span>To</span> dates. </p>

<p>Select the calendar icon (<img src="/images/resources/images/infinityclassic/iconcalendar_24x24.png" alt="Iconcalendar 24X24" />) to display a calendar from which you can select the required date.</p>

<p>Select <span>Search</span> to update the records in the results table.</p>

<h3>Results</h3>

<h5>Viewing more results</h5>

<p>Some pages return results in a continuous list. Select <span> View More</span> at the end of the list to see more items.</p>

<p>The following tables describes the fields on the tickets page.</p>

<table> <col /> <col /> <thead> <tr> <th>Field</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td>Ticket</td> <td> Unique identifier for the ticket that Infinity Classic assigns after the ticket is raised. </td> </tr> <tr> <td>Created Date</td> <td> The date (YY-MM-DD) and time (hh:mm:ss) that the ticket was raised. </td> </tr> <tr> <td>Status</td> <td> The current status of the ticket, which can be: New, Pending, On hold, Solved or Closed. </td> </tr> <tr> <td>Subject</td> <td> The title of the ticket.  <br />Select the text to display ticket details on the <a href="#ViewTicket">View Ticket page</a>.  </td> </tr> <tr> <td>Priority</td> <td>The importance of the ticket: Low, Normal, High and Critical.</td> </tr> <tr> <td>Reported</td> <td>The name of the user who raised the ticket.</td> </tr> <tr> <td>Action</td> <td>  <img src="/images/resources/images/infinityclassic/supportticketiconedit_24x24.png" alt="Supportticketiconedit 24X24" /> – Edit ticket information. <br /> <img src="/images/resources/images/infinityclassic/supportticketiconclose_24x24.png" alt="Supportticketiconclose 24X24" /> – Close the ticket. </td> </tr> <tr> <td><span>Raise New Ticket</span> </td> <td>Select to raise a new ticket.</td> </tr> </tbody> </table>

<h3>To raise a new ticket</h3>

<ol> <li> Display the <span>Support &gt; Ticket</span> menu. </li> <li> Select <span>Raise New Ticket</span>, which displays the New Ticket page. </li> <li> Supply the following information:  <ul> <li>Title</li> <li>Message</li> <li>Classification</li> <li>Priority</li> </ul> </li> <li> Select <span>Create New Ticket</span>. </li> </ol>

<h3>To edit an existing ticket</h3>

<ol> <li> Display the <span>Support &gt; Ticket</span> menu. </li> <li> Search for and identify the ticket you want to edit in the results table. </li> <li> Select the <img src="/images/resources/images/infinityclassic/supportticketiconedit_24x24.png" alt="Supportticketiconedit 24X24" /> icon, which displays the View Ticket dialog. </li> <li> Edit the following information: <ul> <li> Title </li> <li> Message </li> <li> Classification </li> <li> Priority </li> </ul> </li> <li> Select <span>Create New Ticket</span>. </li> </ol>
