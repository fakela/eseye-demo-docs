<p>The following finance reports help you to understand your agreed contract and current costs across your entire SIM card estate, or per SIM card:</p>

<ul> <li><a href="#Summary">Summary invoice</a> </li> <li><a href="#Detailed">Detailed invoice</a> </li> <li><a href="#Account">Account statement report</a> </li> <li><a href="#SMS">SMS billing activity</a> </li> <li><a href="#Service">Service contract</a> </li> </ul>

<h2 id="SummaryInvoicePDF">Summary Invoice (PDF)</h2>

<p>Billing overview for your SIM card estate. The amount owing is summarised on the first page of this report and the total (including VAT) is listed under the <span>TOTAL</span> field. You may want to print the first page (shown below) financial overview to keep on record.</p>

<p><img src="/images/resources/images/management/reports/summaryinvoice.png" alt="Summaryinvoice" /> </p>

<p>The Eseye-specific payment information is also provided on the first page.</p>

<p>If the estate is under 1000 SIMs, a list of individual SIM card costs is summarised in the PDF. If the estate is over 1000 SIM cards, these records are not provided in the PDF. Use the Detailed Invoice to view the individual SIM card records instead, and to see more details about each SIM card.</p>

<p><img src="/images/resources/images/management/reports/summaryinvoice2.png" alt="Summaryinvoice2" /> </p>

<p>If pooled usage is applied to the current invoice, the section below is also displayed in the billing report. Data, SMS, or Voice charges are listed if the usage is in excess of the pooled allowance. You can see further details in the CSV file (detailed invoice), for example: which package the charge was made against.</p>

<p><img src="/images/resources/images/management/reports/summaryinvoicepooled.png" alt="Summaryinvoicepooled" /> </p>

<p>To generate the report:</p>

<ul> <li>Automatically generated by Eseye.</li> </ul>

<p>To find the report in Infinity Classic:</p>

<ol> <li>From the menu, select <span>Finance &gt; Transactions</span> to display the <a href="/infinity-classic/finance#Transactions">Transactions page</a>. </li> <li>Alongside <span>Invoice</span>, select <img src="/images/resources/images/infinityclassic/iconpdfblack_24x24.png" alt="Iconpdfblack 24X24" />.</li> </ol>

<h2 id="DetailedInvoiceCSV">Detailed Invoice (CSV)</h2>

<p>Detailed billing invoice for your SIM card estate, reporting accounts for each ICCID with details including charge type (Data, SMS, or Voice), charge amount in a chosen currency, charge period, activation, current MCC.</p>

<p>Use this file to analyse your data usage and examine trends.</p>

<p><img src="/images/resources/images/management/reports/detailedinvoice.png" alt="Detailedinvoice" /> </p>

<p>The detailed invoice CSV contains the following fields: Invoice date, Item reference, ICCID, MSISDN, Data MSISDN, Package ID, Package, Package item ID, Description, Start, Stop, Quantity, Currency, Rate, Amount, Friendly name, Group name, Activated, Bundle quantity, In bundle quantity and MCC.</p>

<p>To generate the report:</p>

<ul> <li>Automatically generated by Eseye.</li> </ul>

<p>To find the report in Infinity Classic:</p>

<ol> <li>From the menu, select <span>Finance &gt; Transactions</span> to display the <a href="/infinity-classic/finance#Transactions">Transactions page</a>. </li> <li>Alongside <span>Invoice</span>, select <img src="/images/resources/images/infinityclassic/iconcsvblack_24x24.png" alt="Iconcsvblack 24X24" />.</li> </ol>

<h2 id="AccountStatementReportPDF">Account Statement Report (PDF)</h2>

<p>Billing statement that enables you to quickly see any outstanding balance owed to Eseye. Your connectivity will cease functioning when bills are unpaid.</p>

<p><img src="/images/resources/images/management/reports/accountstatement.png" alt="Accountstatement" /> </p>

<p>To generate the report in Infinity Classic:</p>

<ol> <li> From the menu, select <span>Account</span> &gt; <span>Reports</span> to display the <a href="/infinity-classic/account#Reports">Account Reports page</a>. </li> <li>Select the <span>Enable</span> checkbox for each report you want to generate.</li> <li>  Select <span>Run Now</span> to generate the report. </li> </ol>

<p>To find the report in Infinity Classic:</p>

<ol> <li>In Infinity Classic, select <span>Activity &gt; Reports</span> to display the <a href="/infinity-classic/activity#Reports">Reports page</a>. </li> <li>Alongside <span>Account Statement Report</span>, select <img src="/images/resources/images/infinityclassic/iconpdfblack_24x24.png" alt="Iconpdfblack 24X24" />.</li> </ol>

<h2 id="SMSBillingActivityPDFCSV">SMS Billing Activity (PDF, CSV)</h2>

<p>Displays Data, SMS, and Voice activity for an individual SIM, including start and stop date time stamps, usage quantity and cost information.</p>

<p><img src="/images/resources/images/management/reports/smsbilling.png" alt="Smsbilling" /> </p>

<p>Use the CSV file to analyse your data usage and examine trends.</p>

<p><img src="/images/resources/images/management/reports/smsbillingcsv.png" alt="Smsbillingcsv" /> </p>

<p>To find and generate the report in Infinity Classic:</p>

<ol> <li> From the menu, select <span>SIMs</span> &gt; <span>SIM List</span> to display the <a href="/infinity-classic/sims#SimList">SIM List page</a>. </li> <li>Alongside a SIM card, select <img src="/images/resources/images/infinityclassic/simactioniconbilling_24x24.png" alt="Simactioniconbilling 24X24" /> to display the <a href="/infinity-classic/sim-actions#Billing">Billing page</a>.</li> <li>  To download the information in a particular format, select either: <img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" /> or <img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />. </li> </ol>

<h2 id="ServiceContractPDF">Service Contract (PDF)</h2>

<p>Your service contract with Eseye contains agreed costs per SIM card, such as monthly charge, purchase charge, activation charge, and networking type.</p>

<p><img src="/images/resources/images/management/reports/servicecontract.png" alt="Servicecontract" /> </p>

<p>The terms also include your designated countries for Zones 1, 2 and 3, and the costs for bundled and out of bundle usage for Data, SMS, and Voice, as well as contract length.</p>

<p><img src="/images/resources/images/management/reports/servicecontract2.png" alt="Servicecontract2" /> </p>

<p>To find the document in Infinity Classic:</p>

<ol> <li> From the menu, select <span>Settings</span> &gt; <span>SIM Packages</span> to display the <a href="/infinity-classic/settings#SimPackages">SIM Packages page</a>. </li> <li>Alongside the <span>Package ID</span>, select <img src="/images/resources/images/infinityclassic/iconpdfblack_24x24.png" alt="Iconpdfblack 24X24" />.</li> </ol>
