<p>The simplest way to order SIMs is via the Infinity Classic portal. This ensures that the SIMs you purchase are linked to your account which saves you entering your delivery and payment details.</p>

<p>To order SIMs in Infinity Classic:</p>

<ol> <li>Using Infinity Classic, go to: <span>Ordering</span> &gt; <span>Shop</span>.</li> <li>In the <span>Qty</span> column, type the number of SIMs you want to purchase.</li> <li> Select <img src="/images/resources/images/icons/shop_ic_24x24.png" alt="Shop Ic 24X24" /> to add the items to your shopping cart. <br /><img src="/images/resources/images/infinityclassic/shop.png" alt="Shop" />  </li> <li> Select <img src="/images/resources/images/icons/maincart_ic_24x24.png" alt="Maincart Ic 24X24" /> (top right) to view the shopping cart. </li> <li> If you want to adjust the amount, type the new amount in the <span>Qty</span> column, then select <img src="/images/resources/images/icons/refresh_ic_24x24.png" alt="Refresh Ic 24X24" /> to refresh the amount. </li> <li> If you want to delete a purchase, select <img src="/images/resources/images/icons/deletepurchase_ic_24x24.png" alt="Deletepurchase Ic 24X24" />. </li> <li> Select <span>Checkout</span>. <br />The Delivery Address page appears. </li> <li> If required, select <span>Edit this address</span> to update the details. </li> <li> Select <span>Next Step</span>. <br />The Billing Address page appears. </li> <li> If required, select <span>Edit this address</span> to update the details. </li> <li> Select <span>Next Step</span>. <br />The Options page appears. </li> <li> Select a mandatory Delivery Method. </li> <li> Type an <span>Order Summary</span> to help you identify the account. </li> <li> Type your Purchase Order number, which will appear on the invoice. </li> <li> Request a shipping date. </li> <li> If required, type <span>Additional Notes</span>. </li> <li> Select <span>Next Step</span>. </li> <li> Review your order, then select <span>Place Order</span>. </li> </ol>

<p>Next, you must pay for the SIMs. For more information, see <a href="/billing/billing-making-payment">Making a payment</a>.</p>
