<p>You can set up various system notifications to email you when any of the following events occur:</p>

<ul> <li>New products and upgrades</li> <li>Purchase orders</li> <li>Reported network issues and maintenance</li> <li>Reported Infinity Classic application portal issues</li> <li>Reported management issues</li> <li>User permission changes</li> </ul>

<h2 id="Receivingnotifications">Receiving notifications</h2>

<p>To receive notifications in Infinity Classic:</p>

<ol> <li>Go to: <span>&lt;Username&gt;</span> &gt; <span>Notifications</span>.</li> <li>Select the <span>Enable</span> checkbox for each notification you want to receive.</li> </ol>
