<p>A SIM must be activated before it can connect to a network. You can activate SIMs using the:</p>

<ul> <li> <b>Infinity Classic web interface</b>, which enables you to:<ul><li><a href="#ActivateIndividualSims">Activate individual SIMs</a></li><li><a href="#OneBox">Bulk activate a box of SIMs</a> (containing up to 200 SIMs with consecutive SIM numbers) </li><li><a href="#BulkActivationTool">Bulk activate up to 5000 SIMs non-consecutive SIMs</a> (using the bulk activation tool)</li></ul></li> <li><b>Tigrillo API</b>, which provides an <a href="/api/tigrillo/activate-sims">activateSIMs</a> API command to activate SIMs.</li> </ul>

<h2 id="ActivateindividualSIMs">Activate individual SIMs</h2>

<p>Activating individual SIMs uses the ICCID (SIM ID), which is typically printed on individual SIMs as shown in <a href="/sims/consecutive-sims">Understanding AnyNet SIM numbers</a>.</p>

<div> <h3>Before you begin</h3> <p>Before activating one or more SIM cards, you must:</p> <ul> <li>Have a signed Service Contract form agreeing to the associated costs. </li> <li>Ensure that all SIMs you want to activate in a single operation are of the same type (for example, all AnyNet SIMs and no other operator-supplied SIMs). </li> <li>Associate the SIMs with a package, using the tariff ID. </li> <li>Ensure that the tariff ID is the same for all SIMs you want to activate at the same time.</li> <li>Ensure your payments are up to date.</li> <li>For bulk activations, ensure you have set up Groups, Device Models and Alert Profiles.</li> </ul> </div>

<p>To activate individual SIMs using Infinity Classic:</p>

<ol> <li> Log in to Infinity Classic, then select <span>SIMS &gt; Activate</span>. </li> <li> If you have pre-provisioned SIMs, select the link above the <span>SIM Activation</span> table to insert the SIMs into the table. For example: <br /><img src="/images/resources/images/infinityclassic/simactivationloadsims.svg" alt="Simactivationloadsims" />  </li> <li> For each SIM you want to activate, type the <span>ICCID/SIM ID</span>, the optional <span>Friendly Name</span>, and the <span>IMEI</span>. </li> <li> If you require more lines, select <span>Insert Line</span> in the drop-down list, then select <span>Perform Action</span>. <br />To delete a line, select the checkbox alongside the relevant SIM. Select <span>Delete Line</span> in the drop-down list, then select <span>Perform Action</span>. A confirmation box appears. Select <span>Yes </span>to remove the line. <br />To duplicate a line, select the checkbox alongside the relevant SIM. Select <span>Duplicate Line</span> in the drop-down list, then select <span>Perform Action</span>. A confirmation box appears. Select <span>Yes </span>to remove the line. </li> <li> Activate the SIM(s): <ul> <li>To activate specific SIMs, select the check boxes alongside the relevant SIMs, then select <span>Activate Selected SIMs</span>.</li> <li>To activate all the SIMs in the table, select <span>Activate All SIMs</span>.</li> </ul> <br />The <span>Activate SIM - Step 2 of 2</span> page appears. </li> <li> Using the <span>Select Compatible Package</span> drop-down list, which only displays tariffs assigned to the account, select the tariff to apply to all the SIMs. </li> <li> Using the optional <span>Group</span> drop-down list, select the group name that will appear on invoices for easy SIM identification. </li> <li> Using the optional <span>User Fields</span>, type in free text to help further identify the SIMs. </li> <li> Using the <span>Device Model</span> drop-down list, select the device model to which the SIMs are assigned. </li> <li> Using the <span>Alert Profile</span> drop-down list, select the alert profile for the SIMs. </li> <li> Select <span>Activate Request</span> to activate the SIMs. <span><strong>Note: </strong>The activation process can take up to 60 minutes. </span> <br />When activated, the SIMs become live on the network. </li> </ol>

<h2 id="BulkactivateaboxofSIMs">Bulk activate a box of SIMs</h2>

<p>Eseye provide SIMs in boxes of 200. The SIMs in each box have consecutive SIM numbers which enables you to activate an entire box at a time using the first and last SIM numbers in the sequence of 200. </p>

<div> <h3>Before you begin</h3> <p>Before activating one or more SIM cards, you must:</p> <ul> <li>Have a signed Service Contract form agreeing to the associated costs. </li> <li>Ensure that all SIMs you want to activate in a single operation are of the same type (for example, all AnyNet SIMs and no other operator-supplied SIMs). </li> <li>Associate the SIMs with a package, using the tariff ID. </li> <li>Ensure that the tariff ID is the same for all SIMs you want to activate at the same time.</li> <li>Ensure your payments are up to date.</li> <li>For bulk activations, ensure you have set up Groups, Device Models and Alert Profiles.</li> </ul> </div>

<p>To bulk activate a box of SIMs using Infinity Classic:</p>

<ol> <li> Using Infinity Classic, go to: <span>SIMs</span> &gt; <span>Bulk Activate</span>. </li> <li> In the <span>Bulk SIM boxfull activation</span> box, in the <span>First</span> field, type the lowest numerical SIM number from the box to activate. <br /><img src="/images/resources/images/infinityclassic/bulkactivatebox.png" alt="Bulkactivatebox" />  </li> <li> In the <span>Last</span> field, type the highest numerical SIM number from the box to activate. <br />You can activate up to 200 consecutive SIMs. <br />The final digit in the SIM number is a check digit. For more information, see <a href="/sims/consecutive-sims" target="_blank">Understanding AnyNet SIM numbers</a>. </li> <li> Select <span>Activate</span>. <br />The Activate SIM - Step 2 of 2 page appears. </li> <li> Using the <span>Select Compatible Package</span> drop-down menu, select the agreed tariff to apply to all the SIMs. <br />Only the tariffs assigned to the account are available for selection. </li> <li> Using the optional <span>Group</span> drop-down menu, select the group name that will appear on invoices. <br />Groups can help identify SIMs. </li> <li>Using the optional <span>User Fields</span>, type in free text to help further identify the SIMs.</li> <li>Using the <span>Device Model</span> drop-down menu, select the device model to which the SIMs are assigned.</li> <li>Using the <span>Alert Profile</span> drop-down menu, select the alert profile for the SIMs.</li> <li> Select <span>Activate Request</span> to activate the SIMs. <br />The activation process can take up to 60 minutes. When activated, the SIMs become live on the network.  </li> </ol>

<h2 id="BulkactivateSIMsusingthebulkactivationtool">Bulk activate SIMs using the bulk activation tool</h2>

<p>Infinity Classic provides a bulk activation tool which enables you to active up to 5000 SIMs by uploading a list of ICCID/SIM numbers.</p>

<div> <h3>Before you begin</h3> <p>Before activating one or more SIM cards, you must:</p> <ul> <li>Have a signed Service Contract form agreeing to the associated costs. </li> <li>Ensure that all SIMs you want to activate in a single operation are of the same type (for example, all AnyNet SIMs and no other operator-supplied SIMs). </li> <li>Associate the SIMs with a package, using the tariff ID. </li> <li>Ensure that the tariff ID is the same for all SIMs you want to activate at the same time.</li> <li>Ensure your payments are up to date.</li> <li>For bulk activations, ensure you have set up Groups, Device Models and Alert Profiles.</li> </ul> </div>

<p>To bulk activate SIMs using the Infinity Classic Bulk Activation tool:</p>

<ol> <li> Using Infinity Classic, go to: <span>SIMs</span> &gt; <span>Bulk Activate</span>. </li> <li> In the <span>Bulk Activation</span> box, use the <span>Select Compatible Package</span> drop-down menu to select the agreed tariff to apply to all the SIMs. <br />Only the tariffs assigned to the account are available for selection. <br /><img src="/images/resources/images/infinityclassic/bulkactivationtool.png" alt="Bulkactivationtool" />  </li> <li> Using the optional <span>Group</span> drop-down menu, select the group name that will appear on invoices. <br />Groups can help identify SIMs. </li> <li> In the <span>ICCID</span> box, paste the list of SIM numbers you want to activate, one per line. <br />You can paste up to 5000 non-consecutive SIM numbers. <br />Select <span>Clear</span> to remove all ICCID values from the list. </li> <li> Select <span>Activate SIM</span> to activate the SIMs. <br />The activation process can take up to 60 minutes. When activated, the SIMs become live on the network.  </li> </ol>
