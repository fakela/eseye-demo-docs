<p>The following status and connectivity reports hold information about a SIM card’s status and connection activity so that you can understand the SIM card’s current performance: </p>

<ul> <li><a href="#SIM">SIM contract settings</a> </li> <li><a href="#Suspension">Suspension and termination report</a> </li> </ul>

<h2 id="SIMContractSettingsPDFCSV">SIM Contract Settings (PDF, CSV)</h2>

<p>Lists a SIM card’s status, activation and termination dates, and package, contract and costs information, for administrative purposes.</p>

<p><img src="/images/resources/images/management/reports/simcontractsettings.png" alt="Simcontractsettings" /> </p>

<p>To find and generate the report in Infinity Classic:</p>

<ol> <li> From the menu, select <span>SIMs</span> &gt; <span>SIM List</span>. </li> <li>Alongside a SIM card, select <img src="/images/resources/images/infinityclassic/simactioniconcontract_24x24.png" alt="Simactioniconcontract 24X24" /> to display the <a href="/infinity-classic/sim-actions#Contract">Contract settings</a>.</li> <li>  To download the information in a particular format, select either: <img src="/images/resources/images/infinityclassic/iconpdf_24x24.png" alt="Iconpdf 24X24" /> or <img src="/images/resources/images/infinityclassic/iconcsv_24x24.png" alt="Iconcsv 24X24" />. </li> </ol>

<h2 id="SuspensionandTerminationReportPDFCSV">Suspension and Termination Report (PDF, CSV)</h2>

<p>Details when a SIM card status was initiated, including all activations, terminations, manual suspensions, and alerted suspensions, by ICCID and MSISDN.</p>

<p><img src="/images/resources/images/management/reports/suspensiontermination.png" alt="Suspensiontermination" /> </p>

<p>To generate the report in Infinity Classic:</p>

<ul> <li>See <a href="/infinity-classic/managing-reports#CreatingReportProfiles">Creating report profiles</a>.</li> </ul>

<p>To find the report in Infinity Classic:</p>

<ol> <li>From the menu, select <span>Activity &gt; Reports</span> to display the <a href="/infinity-classic/activity#Reports">Reports page</a>. </li> <li>Alongside <span>Suspension and Termination Report</span>, select <img src="/images/resources/images/infinityclassic/iconpdfblack_24x24.png" alt="Iconpdfblack 24X24" /> or <img src="/images/resources/images/infinityclassic/iconcsvblack_24x24.png" alt="Iconcsvblack 24X24" />.</li> </ol>
