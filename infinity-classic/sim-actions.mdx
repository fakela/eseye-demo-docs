<h2 id="NetFlow">NetFlow</h2>

<p>To display the NetFlow activity for a SIM, on the <a href="/infinity-classic/sims#SimList">SIM List page</a> menu, select <img src="/images/resources/images/infinityclassic/simactioniconnetflow_24x24.png" alt="Simactioniconnetflow 24X24" /> next to the chosen SIM. NetFlow data enables you to track traffic flow and volume per SIM, and outputs Mobile Terminated (MT) and Mobile Originated (MO) data in a graph that you can drill into.</p>

<p>The NetFlow dialog displays a graphical breakdown of data transmission over time. The blue bars are MT (received), and the black bars are MO (sent) data transmissions. The default view displays an entire month of data, with each bar representing one day. You can change the current data window using the arrows above the graph.</p>

<p><img src="/images/resources/images/infinityclassic/simnetflowactivity.png" alt="Simnetflowactivity" /> </p>

<p>Selecting one bar will update to show a new graph with one bar representing an hour and the whole graph covering a day. Selecting an hour bar will update to show a new graph, with one bar representing a minute and the whole graph covering an hour. </p>

<p>If you would like to receive NetFlow data in greater detail, speak to your Account Manager about the Push API. For more information, see <a href="/connectivity/push-intro">Monitoring and administering network usage</a>.</p>
