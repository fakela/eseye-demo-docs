<p>The Infinity Classic web interface enables users to manage SIM and account information. It contains all of the information about your SIM estate and pulls live network data for your subscribers via <a href="/connectivity/understanding-pops">Eseye's localised data centres</a>. </p>

<p> <img src="/images/resources/images/infinityclassic/overview.svg" alt="Overview" /> </p>

<p>The Infinity Classic web interface enables you to:</p>

<ul> <li> <a href="/infinity-classic/manage-accounts">Manage users and accounts</a>  </li> <li> <a href="/infinity-classic/ordering-sims">Order</a>, <a href="/infinity-classic/activating-sims">activate</a>, <a href="/infinity-classic/suspending-sims">suspend</a> and <a href="/infinity-classic/terminating-sims">terminate</a> SIMs. </li> <li> <a href="/infinity-classic/managing-alerts">Configure alerts</a> to trigger on different threshold events (such as data usage or overage values) and take different actions (such as <a href="/infinity-classic/managing-notifications">notifying users</a> or suspending a SIM) </li> <li> View the connection and data flow metrics for individual SIMs </li> <li> <a href="/infinity-classic/managing-reports">Configure ad hoc or scheduled reports</a> for areas such as finance, data usage, and lifecycle </li> <li> <a href="/infinity-classic/manage-sims">Manage SIMs</a> (suspending, terminating, re-activating) </li> <li> Raise <a href="/infinity-classic/support">support tickets</a> </li> <li><a href="/infinity-classic/displaying-sim-timelines">Display SIM Timelines</a> </li> </ul>

<h2 id="Menus">Menus</h2>

<p>The Infinity Classic user interface is organised into the following menus:</p>

<p><img src="/images/resources/images/infinityclassic/menu.png" alt="Menu" /> </p>

<ul> <li><a href="/infinity-classic/sims">SIMs</a> – activate, suspend, terminate SIMs and manage SIM information.</li> <li><a href="/infinity-classic/settings">Settings</a> – configure SIM packages, alert profiles, reporting profiles, devices and the messaging API.</li> <li><a href="/infinity-classic/activity">Activity</a> – configure notifications, alerts, reports and the messaging API.</li> <li><a href="/infinity-classic/admin">Admin</a> – configure users, referral codes and sub accounts. </li> <li><a href="/infinity-classic/finance">Finance</a> – view and edit finance settings, billing details and account transactions.</li> <li><a href="/infinity-classic/ordering">Ordering</a> – order new SIMs and manage order history and delivery addresses.</li> <li><a href="/infinity-classic/support">Support</a> – raise, view, edit and close support tickets. </li> <li><a href="/infinity-classic/account">Account</a> – view account-specific details, notifications, reports, alerts and audit logs.</li> </ul>

<Note>Some menus are hidden based on <a href="/infinity-classic/manage-accounts#User">user roles</a>.</Note>

<h2 id="AccessingInfinityClassic">Accessing Infinity Classic</h2>

<p>After purchasing Eseye SIMs and signing up for a Infinity Classic account, Eseye sends a verification email with the following details:</p>

<ul> <li>From: support@eseye.com</li> <li>Subject: Your Eseye SIM management account</li> <li>Body: Contains an account verification link</li> </ul>

<Note>Before you can log in, you must verify the account via the link in the email.</Note>

<p>To log in to Infinity Classic:</p>

<ol> <li>Navigate to the following URL: <a href="https://siam.eseye.com/user/login/" target="_blank">https://siam.eseye.com/user/login/</a></li> <li> Enter you supplied username and password and select <span>Sign In</span>. <br /><img src="/images/resources/images/infinityclassic/login.png" alt="Login" />  </li> <li> If <a href="/infinity-classic/manage-accounts#2FA">two factor authentication is enabled</a> for your portfolio, specify an authentication code and select <span>Submit</span>. <br /><img src="/images/resources/images/infinityclassic/2factorauthentication.png" alt="2Factorauthentication" />  </li> </ol>

<p> To log out of Infinity Classic:</p>

<p>From the <span>&lt;Username&gt;</span> menu, select <span>Logout</span>.</p>

<p>To reset your Infinity Classic password:</p>

<ol> <li> Navigate to the following URL: <a href="https://siam.eseye.com/user/login/" target="_blank">https://siam.eseye.com/user/login/</a> </li> <li> Select <span>Request password</span>. </li> <li> Specify your <span>Username</span> or <span>Email address</span> and select <span>Reset Password</span>. <br />Support sends an email address with a link to reset your password. </li> <li> Select the link in the Password Reset email. </li> <li> Specify a new password and select <span>Set new password</span>. </li> </ol>

<p>To change your password in Infinity Classic:</p>

<ol> <li>Log in to Infinity Classic then go to: <span>&lt;Username&gt;</span> &gt; <span>Details</span>.</li> <li>Select <span>Change Password</span>.</li> <li>In the <span>Change Password</span> dialog box, type your <span>Current Password</span>.</li> <li>Type a <span>New Password</span>.</li> <li>Alongside <span>Confirm New Password</span>, type the new password again.</li> <li> Select <span>Update</span> to save your changes. </li> </ol>
