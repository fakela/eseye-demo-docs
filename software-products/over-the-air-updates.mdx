<p>AnyNet SMARTconnect‚ĄĘ supports over the air (OTA) updates for the following software, using parameters set in the configuration file:</p>

<ul> <li>Host</li> <li>Kernel (Module operating firmware)</li> <li>AnyNet SMARTconnect‚ĄĘ application</li> <li>Configuration file</li> </ul>

<p>For more information, see <a href="/software-products/configuration-file">Using the AnyNet SMARTconnect‚ĄĘ configuration file</a>.</p>

<h2 id="Updatinghostfirmware">Updating host firmware</h2>

<p>Modify the AnyNet SMARTconnect‚ĄĘ configuration file to contain the URL for the host firmware. The host must request AnyNet SMARTconnect‚ĄĘ to check for updates. If an update is available, AnyNet SMARTconnect‚ĄĘ will download it into flash memory and send a URC to the host, indicating that new firmware is available. The host then reads the firmware image in chunks, using AT commands. A checksum is provided to ensure the copy from AnyNet SMARTconnect‚ĄĘ to host has completed without errors.</p>

<p>For more information, see <a href="/software-products/update-software-firmware#Download2" target="_blank">Updating AnyNet SMARTconnect‚ĄĘ software and host module firmware</a>.</p>

<h2 id="UpdatingkernelAnyNetSMARTconnectapplicationandconfigurationsoftware">Updating kernel, AnyNet SMARTconnect‚ĄĘ application and configuration software</h2>

<p>Trigger updates to the modem kernel, AnyNet SMARTconnect‚ĄĘ application and configuration using either the host, cloud IoT service, or SMS.</p>

<p>The URL used to download the images is set in AnyNet SMARTconnect‚ĄĘ configuration file. In order to apply updates, AnyNet SMARTconnect‚ĄĘ must reboot. For more information, see <span>update&#95;autoreboot</span> in the configuration file.</p>

<p>For more information, see <a href="/software-products/update-software-firmware">Updating AnyNet SMARTconnect‚ĄĘ software and host module firmware</a>.</p>
