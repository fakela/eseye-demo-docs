title: "EMQPERSIST – report a set value to the device shadow"
description: "Report a set value to the device shadow/twin. The operation depends on which AWSSHADOW mode you set, either simple or complete."
---

<p>Report a set value to the device shadow/twin. The operation depends on which <span>AWSSHADOW</span> mode you set, either <span>simple</span> or <span>complete</span>. </p>

<Tip>You set <span>AWSSHADOW</span> in the Configuration File. For more information, see <a href="/software-products/configuration-file">Using the AnyNet SMARTconnect™ configuration file</a>. Alternatively, send: <span>AT+ETMCFG="mqtt","awsshadow","&lt;<var>off/simple/complete</var>&gt;"</span>, for example: <span>AT+ETMCFG="mqtt","awsshadow","simple"</span></Tip>

<h2 id="AWSSHADOWsimplemode">AWSSHADOW <span>simple</span> mode</h2>

<p> The shadow AnyNet SMARTconnect™ performs the following automatically:</p>

<ul> <li>Generates the JSON <span>state</span> and <span>reported</span> or <span>desired</span> objects</li> <li>Subscribes to the <span>shadow/update/delta</span> topic, and filters out the containing objects</li> <li>Presents the host with the state object JSON content only, sent via URC</li> </ul>

<p>The host only sends the content of the <span>reported</span> object.</p>

<h2 id="AWSSHADOWcompletemode">AWSSHADOW <span>complete</span> mode</h2>

<p>The host sends the complete JSON for the shadow to AnyNet SMARTconnect™.</p>

<p>AnyNet SMARTconnect™ sends the complete shadow message as a URC, without filtering.</p>

<h4>Examples</h4>

<p>For <span>Simple</span> mode:</p>

<p>AT+EMQPERSIST="&#123;&#92;"Temperature&#92;":-3.5&#125;"</p>

<p>AT+EMQPERSIST="&#123;&#92;"Colour&#92;":&#92;"Blue&#92;"&#125;"</p>

<p>For <span>Complete</span> mode, including setting AWSSHADOW to complete:</p>

<p>AT+ETMCFG="mqtt","awsshadow","complete"</p>

<p>AT+EMQPERSIST="&#123;&#92;"state&#92;":&#123;&#92;"reported&#92;":&#123;&#92;"Temperature&#92;":28.0&#125;&#125;&#125;"</p>
