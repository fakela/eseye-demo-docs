---
title: "EMQPUBOPEN – create a publish message topic"
description: "Create and view publish message topics, which will enable you to publish data to thing s in the cloud."
---

<p>Create and view publish message topics, which will enable you to publish data to thing s in the cloud.</p>

<Note>The topic name depends on the MQTT <span>&lt;<var>topicsuffix</var>&gt;</span> parameter, unless a token is used. For more information, see the [MQTT] section, <a href="/software-products/configuration-file">Using the AnyNet SMARTconnect™ configuration file</a>. </Note>

<p>You can use the following tokens to construct the topic name:</p>

<ul> <li><span>&#36;t</span> – returns the cloud thing Name</li> <li><span>&#36;i</span> – returns the device International Mobile Equipment Identity (IMEI) number</li> </ul>

<p>For example, <span>status/&#36;i</span> publishes as <span>status/&lt;IMEI&gt;</span></p>

<Note>You can only include one token in a topic.</Note>

<h4>Examples</h4>

<p>AT+EMQPUBOPEN=0,"PublishToCloud0/&#36;i"</p>

<p>OK</p>

<p> </p>

<p>+EMQPUBOPEN: 0,0</p>

<p> </p>

<p>AT+EMQPUBOPEN=1,"PublishToCloud1"</p>

<p>OK</p>

<p> </p>

<p>+EMQPUBOPEN: 1,0</p>

<p> </p>

<p>AT+EMQPUBOPEN?</p>

<p>0: PublishToCloud0/862061040777777</p>

<p>1: PublishToCloud1/AWS thingName</p>

<p> </p>

<p>OK</p>

<p> </p>

<p>AT+EMQPUBLISH=0,1,"&#123;&#92;"imei&#92;":&#92;"&#36;i&#92;",&#92;"user&#92;":&#92;"1&#92;",&#92;"sys&#92;":&#92;"128&#92;",&#92;"dia&#92;":&#92;"112&#92;",&#92;"pul&#92;":&#92;"075&#92;",&#92;"ano&#92;":&#92;"0&#92;",&#92;"time&#92;":&#92;"2023/11/30,20:10:12-20&#92;",&#92;"ver&#92;":&#92;"BP01&#95;2.0&#92;",&#92;"devdata&#92;":&#123;&#92;"r&#92;":&#36;r,&#92;"rat&#92;":&#92;"&#36;a&#92;",&#92;"mcc&#92;":&#36;c,&#92;"mnc&#92;":&#36;n,&#92;"imsi&#92;":&#36;s,&#92;"batt&#92;":87&#125;&#125;"</p>
