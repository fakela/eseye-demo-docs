---
title: "ETMFWCHECK – checks for updates to the AnyNet SMARTconnect™ application"
description: "This command checks the [application] updateurl for a new AnyNet SMARTconnect™ application."
---

<p>This command checks the <span>[application] updateurl</span> for a new AnyNet SMARTconnect™ application.</p>

<p>For more information, see <a href="/software-products/configuration-file" target="_blank">Using the AnyNet SMARTconnect™ configuration file</a>.</p>

<table> <col /> <col /> <col /> <thead> <tr> <th>Type</th> <th>Syntax</th> <th>Returned Result</th> </tr> </thead> <tbody> <tr> <td>Execute</td> <td> AT+ETMFWCHECK </td> <td> OK <br />  <br />+ETMFWCHECK: checking... <br />  <br />+ETMFWCHECK: complete <br />If a new application needs to be installed, the following URC will also appear: <br />+ETM: REBOOT REQUIRED <br />You must reboot the system as soon as possible. <br />or <br /><span>+ETM ERROR: &lt;errormessage&gt;</span> – the command failed. </td> </tr> </tbody> </table>

<h4>Example</h4>

<p>AT+ETMFWCHECK</p>

<p>OK</p>

<p> </p>

<p>+ETMFWCHECK: checking...</p>

<p> </p>

<p>+ETMFWCHECK: complete</p>
