---
title: "ETMCFGCHECK – checks if a new AnyNet SMARTconnect™ configuration file is available"
description: "This command checks the [config] updateurl for a new configuration file."
---

<p>This command checks the <span>[config] updateurl</span> for a new configuration file.</p>

<p>For more information, see <a href="/software-products/configuration-file">Using the AnyNet SMARTconnect™ configuration file</a>.</p>

<table> <col /> <col /> <col /> <thead> <tr> <th>Type</th> <th>Syntax</th> <th>Returned Result</th> </tr> </thead> <tbody> <tr> <td>Execute</td> <td> AT+ETMCFGCHECK </td> <td> OK <br />  <br />+ETMCFGCHECK: checking... <br />  <br />+ETMCFGCHECK: complete <br />If a new configuration file needs to be applied, the following URC will also appear: <br />+ETM: REBOOT REQUIRED <br />You must reboot the system as soon as possible. <br />or <br /><span>+ETM ERROR: &lt;errormessage&gt;</span> – the command failed. </td> </tr> </tbody> </table>

<h4>Example</h4>

<p>AT+ETMCFGCHECK</p>

<p>OK</p>

<p> </p>

<p>+ETMCFGCHECK: checking...</p>

<p> </p>

<p>+ETMCFGCHECK: complete</p>
