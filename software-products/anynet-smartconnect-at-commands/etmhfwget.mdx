---
title: "ETMHFWGET – checks for new host firmware"
description: "Requests a check for new host firmware. Use the Read function to check if new firmware is already downloaded and available."
---

<p>Requests a check for new host firmware. Use the <span>Read</span> function to check if new firmware is already downloaded and available.</p>

<table> <col /> <col /> <col /> <thead> <tr> <th>Type</th> <th>Syntax</th> <th>Response</th> </tr> </thead> <tbody> <tr> <td>Test</td> <td>   <span>AT+ETMHFWGET=?</span>  </td> <td> +ETMHFWGET: &lt;IMEI&gt; <br />OK </td> </tr> <tr> <td>Read</td> <td> Not currently available. </td> <td> </td> </tr> <tr> <td> Write </td> <td> <span>AT+ETMHFWGET=&lt;IMEI&gt;</span>  <br />where <span>&lt;IMEI&gt;</span> is the unique identity of the current device. </td> <td> OK <br />  <br /><span>+ETMHFWGET: checking...</span> – AnyNet SMARTconnect™ is searching the configuration file <span>[host] updateurl</span> for new firmware. If new firmware is available, the file is downloaded. For more information, see <a href="/software-products/configuration-file#updateurl">updateurl</a>. <span><strong>Note: </strong>The host can read the downloaded file using <span>AT+ETMHFWREAD</span>. For more information, see <a href="/software-products/anynet-smartconnect-at-commands/etmhfwread">ETMHFWREAD – reads a section of the new host firmware image</a>.</span> <br /><span>+ETMHFWGET: none</span> – no host firmware is available. Ensure that the URL is configured and accessible <br />or <br /><span>+ETMHFWGET: available</span> – the latest host firmware is downloaded and available <br />or <br /><span>+ETM ERROR: &lt;errormessage&gt;</span>  </td> </tr> </tbody> </table>

<h4>Example</h4>

<p><span>AT+ETMHFWGET=862061234567890</span> </p>

<p>OK</p>

<p> </p>

<p>+ETMHFWGET: checking...</p>

<p> </p>

<p>+ETMHFWGET: none</p>
