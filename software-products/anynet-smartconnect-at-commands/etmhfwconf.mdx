---

<p>Confirms the new host firmware image is applied and then deletes the image from the modem flash memory.</p>

<table> <col /> <col /> <col /> <thead> <tr> <th>Type</th> <th>Syntax</th> <th>Response</th> </tr> </thead> <tbody> <tr> <td> Execute </td> <td> <span>AT+ETMHFWCONF</span>  </td> <td> OK <br />or <br /><span>+ETM ERROR: &lt;errormessage&gt;</span> – may indicate that the host firmware has not successfully downloaded onto the Quectel module. Ensure <span>AT+ETMHFWGET</span> completed successfully, and you have seen the <span>+ETMHFWGET: available</span> URC. For more information, see <a href="/software-products/anynet-smartconnect-at-commands/etmhfwget">ETMHFWGET – checks for new host firmware</a>. </td> </tr> </tbody> </table>

<h4>Example</h4>

<p><span>AT+ETMHFWCONF</span> </p>
