<p>Manages the LwM2M protocol, which enables you to add, delete and update LwM2M resource values. Before you can use ETMLW, ensure that you have enabled the LwM2M protocol using <span>AT+ETMSTATE="startlwm2m"</span>. For more information, see <a href="/software-products/anynet-smartconnect-at-commands/etmstate">ETMSTATE – check current state</a>.</p>

<h4>Example</h4>

<p>AT+ETMLW="add",-1,"/32769/0/1","Integer",123</p>

<p> </p>

<p> </p>

<p>OK</p>

<p> </p>

<p>+ETMLW: 0,"/32769/0/1","ADD",0</p>

<p> </p>

<p>AT+ETMLW="add",-1,"/32769/1/1","String","Temp: 17°C"</p>

<p> </p>

<p>OK</p>

<p> </p>

<p>+ETMLW: 0,"/32769/1/1","ADD",-12</p>

<p> </p>

<p>AT+ETMLW="add",-1,"/32769/1/1","Integer",17</p>

<p> </p>

<p>OK</p>

<p>+ETMLW: 1,"/32769/1/1","integer","ready"</p>
