---
title: "EMQSUBOPEN – create a subscribe message topic"
description: "Create and view subscribe message topics, which will enable you to send data from the MQTT broker to your thing."
---

<p>Create and view subscribe message topics, which will enable you to send data from the MQTT broker to your thing. </p>

<Note>The topic name depends on the MQTT <span>&lt;<var>topicsuffix</var>&gt;</span> parameter, unless a token is used. For more information, see the [MQTT] section, <a href="/software-products/configuration-file">Using the AnyNet SMARTconnect™ configuration file</a>. </Note>

<p>You can use the following tokens to construct the topic name:</p>

<ul> <li><span>&#36;t</span> – returns the cloud thing Name</li> <li><span>&#36;i</span> – returns the device International Mobile Equipment Identity (IMEI) number</li> </ul>

<p>For example, <span>status/&#36;i</span> publishes as <span>status/&lt;IMEI&gt;</span></p>

<Note>You can only include one token in a topic.</Note>

<h4>Example</h4>

<p>AT+EMQSUBOPEN=0,<span>"</span>SubscribeFromCloud0<span>/&#36;i,1"</span></p>

<p>OK</p>

<p>+EMQSUBOPEN: 0,0</p>

<p>OK</p>
