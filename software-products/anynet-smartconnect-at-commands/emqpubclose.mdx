---

<p>Remove a publish message topic.</p>

<table> <col /> <col /> <col /> <thead> <tr> <th>Type</th> <th>Syntax</th> <th>Returned Result</th> </tr> </thead> <tbody> <tr> <td>Test</td> <td> AT+EMQPUBCLOSE=? </td> <td> +EMQPUBCLOSE:(0-7) <br />  <br />OK </td> </tr> <tr> <td>Read</td> <td> AT+EMQPUBCLOSE? </td> <td> OK </td> </tr> <tr> <td>Write</td> <td> AT+EMQPUBCLOSE=(<var>0-7</var>) <br />where: <br /><span>(<var>0-7</var>)</span> is a publish index number in the range from 0, up to and including 7. You must have already opened a publish topic to the selected index number, or the command will return an error. </td> <td> OK <br />  <br />+EMQPUBCLOSE:(0-7),&lt;status&gt; <br />where: <ul> <li><span>(0-7)</span> is the publish index number you selected from the range</li> <li><span>status</span> is either:<ul><li><span>0</span> – subscription cancelled successfully</li><li><span>-2</span> – no topic was registered for the given index</li></ul></li> </ul> <br />or <br /><span>ERROR</span> – the command failed <span><strong>Tip: </strong>Check you have sent the <span>AT+ETMSTATE="startmqtt"</span> command to enable MQTT.</span> </td> </tr> </tbody> </table>

<h4>Example</h4>

<p>AT+EMQPUBCLOSE=0</p>

<p>OK</p>
