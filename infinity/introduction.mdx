<p>Infinity provides tools to manage and monitor your IoT estate Eseye <a href="/managing-my-estate/managing-my-estate#top" target="_blank">Managing my Estate</a>. Eseye Infinity is designed to deliver multi-market cellular IoT connectivity and give customers access to control their SIM lifecycle, view device activity and manage their SIM estate. It captures user requirements in an intuitive manner.</p>

<p> </p>

<p> <img src="/images/resources/images/infinity/overview.svg" alt="Overview" /> </p>

<p>It accesses and streamlines IoT estate information via the Public API and live network data from Eseye's PoPs. For more information, see <a href="/connectivity/understanding-pops#top" target="_blank">About Eseye PoPs</a>.</p>

<h3> Your Infinity account is pre-configured</h3>

<p>When you purchase AnyNet SIMs, your account is configured by the portfolio administrator and you will receive an email with the following details: </p>

<ul> <li>From: support@eseye.com</li> <li>Subject: <strong>Welcome to Eseye</strong></li> <li>Body: Contains a link to log in to Infinity interface available to users with AnyNet+ SIM. It contains login credentials</li> </ul>

<p> for your Infinity user account. </p>

<Note>Ensure that you change the temporary password.</Note>

<h3>Configuring an Eseye SIM management account</h3>

<p>To configure your Eseye SIM management account:</p>

<ol> <li> After signing into your Infinity account, you will receive a verification email with the following details: <ul> <li>From: <span>no&#95;reply@anynetiot.com</span></li> <li>Subject: <span>Welcome to Infinity</span></li> <li> Body: Includes your login credentials (username and temporary password), provides an overview of features, and links to the Infinity documentation website and technical support. </li> </ul> </li> <li>Select <span>log in here</span> and follow the procedure, as described below.</li> </ol>

<Note>The Infinity web interface automatically logs users out after 15 minutes of inactivity.</Note>

<h4>Logging into Infinity</h4>

<p>To access the Infinity interface:</p>

<ol> <li> Navigate to <a href="https://infinity.anynetiot.com/" target="_blank">https://infinity.anynetiot.com</a>. </li> <li> Enter your username, then copy the <span>verification code</span> from the <span>Welcome to Infinity</span> email and paste it into the <span>Password</span> field. <span><strong>Tip: </strong>This temporary password is time-sensitive and should be changed promptly.</span> </li> <li>From the <span>Language</span> drop-down list,, select your preferred language to access the Infinity portal.</li> <li>Select <span>Sign In</span> to navigate to the <a href="/infinity/manage-user-profile#https://docs.eseye.com/Content/Infinity/ManageUserProfile.htm#Two">Two Factor Authentication</a> screen. </li> <li> Enable the <span>MFA</span> as described in <a href="/infinity/manage-user-profile" target="_blank">Managing your Infinity user profile</a>. </li> </ol>

<p>After logging in, select <span>Launch Infinity</span> to see the Infinity landing page.</p>

<p><img src="/images/resources/images/infinity/menu.png" alt="Menu" /> </p>

<Note>You can access various menus depending upon your assigned role, portfolio and package. For more information, see<a href="/managing-my-estate/managing-my-estate">Managing my Estate</a>.</Note>

<p> </p>

<h4>Changing password in Infinity</h4>

<p>To change your Infinity interface password:</p>

<ol> <li>Log in to the Infinity web interface.</li> <li>Select your username in the top-right of the web interface to display the <span>Profile Settings</span> window.</li> <li>Select <span>Change Password</span>.</li> <li>In the <span>Change Password</span> dialog box, type your <span>Current Password</span>.</li> <li>Type a <span>New Password</span>.</li> <li>Alongside <span>Confirm New Password</span>, retype the new password.</li> <li> Select <span>Update</span> to save your changes. </li> </ol>

<h4>Resetting password in Infinity</h4>

<p>To reset your Infinity interface password:</p>

<ol> <li> Navigate to <a href="https://infinity.anynetiot.com/" target="_blank">https://infinity.anynetiot.com</a>. </li> <li> Select <span>Forgotten password?</span>. </li> <li> Specify your <span>Username</span>, then select <span>Send Code</span>. <br />Infinity sends an email (from no&#95;reply@anynetiot.com) displaying a six-digit verification code on the <span>Reset Password</span> screen: <br /><img src="/images/resources/images/infinity/resetpassword.png" alt="Resetpassword" />  </li> <li> Copy the verification code from the <span>Password Reset</span> email into the <span>Verification code</span> field. </li> <li> Specify a new password, then select <span>Update</span>. </li> <li> On the confirmation screen, select <span>Login</span> to return to the log in screen. </li> </ol>

<p>After logging in, you can see the Infinity landing page.</p>

<p><img src="/images/resources/images/infinity/menu.png" alt="Menu" /> </p>

<Note>You can access various menus depending upon your assigned role, portfolio and package. For more information, see <a href="/managing-my-estate/managing-my-estate">Managing my Estate</a>.</Note>

<h4>Logging out of Infinity</h4>

<p>To log out of the Infinity interface:</p>

<ul> <li> At the top-right, select <span>Logout</span>.</li> </ul>
