<h3>Enabling or disabling multi-factor authentication (MFA)</h3>

<p>Watch the below video for enabling MFA.</p>

<p><a href="https://docs.eseye.com/Content/Resources/Videos/How-to-migrate-to-Infinity.mp4" aria-label="How To Migrate To Infinity" title="How To Migrate To Infinity"><video title="How To Migrate To Infinity" aria-label="How To Migrate To Infinity" alt="How To Migrate To Infinity"><source src="https://docs.eseye.com/Content/Resources/Videos/How-to-migrate-to-Infinity.mp4" type="video/mp4" /></video><span> How To Migrate To Infinity</span></a> </p>

<Note>Only portfolio administrators can use this feature.</Note>

<p>Multi- factor authentication provides an additional layer of security by requiring two independent authentication factors:</p>

<ol> <li> Enter user credentials in the <span>Login</span> screen. </li> <li> Use a third-party authenticator application to generate a six-digit authentication code. <span><strong>Note: </strong>Ensure that you enter this authentication code into the the <span>Login</span> screen before it expires.</span> </li> </ol>

<p> </p>

<p>Use either of the following authentication applications:</p>

<ul> <li><a href="https://support.google.com/accounts/answer/1066447" target="_blank">Google Authenticator</a> </li> <li><a href="https://authy.com/" target="_blank">Authy</a> </li> <li><a href="https://lastpass.com/auth/" target="_blank">LastPass Authenticator</a> </li> </ul>

<p>To enable MFA:</p>

<ol> <li>At the top right, select your <span>&lt;<var>username</var>&gt;</span> &gt; <span>Profile Settings</span> to display the <span>Profile Settings</span> dialog box.</li> <li> Under <span>Multi-factor Authentication</span>, select <span>Turn On</span>. </li> <li> Using the authenticator application installed on your phone, scan the QR code displayed on the profile page. <br /><img src="/images/resources/images/integra/2faturnonqr.png" alt="2Faturnonqr" />  </li> <li> Enter the six-digit code from the authenticator application, then select <span>Confirm</span> to accept, else select <span>Cancel setup</span> to abort. <span><strong>Tip: </strong>On every subsequent log in, enter the six-digit authentication code generated by your authentication application.</span> </li> </ol>

<p>To disable MFA:</p>

<ol> <li>At the top right, select your <span>&lt;<var>username</var>&gt;</span> &gt; <span>Profile Settings</span> to display the <span>Profile Settings</span> dialog box.</li> <li> Under <span>Multi-factor Authentication</span>, select <span>Turn Off</span>. <br /><img src="/images/resources/images/infinity/2faturnoff.png" alt="2Faturnoff" />  </li> <li> Select <span>Disable MFA</span> to confirm. <br /><img src="/images/resources/images/infinity/disablemfa.png" alt="Disablemfa" />  <br />You receive a green notification confirming that the MFA is turned off. <br /><img src="/images/resources/images/infinity/mfa-off.png" alt="Mfa Off" />  </li> </ol>

<h3>Configuring your timezone and language</h3>

<p>All dates and times displayed on the user interface are shown in the timezone configured in the <span>Profile Settings</span>. The user interface automatically adjusts the date and time for region-specific differences, such as daylight savings.</p>

<p>All text on the user interface is displayed in your chosen language.</p>

<p>To change the timezone:</p>

<ol> <li>At the top right, select your <span>&lt;<var>username</var>&gt;</span> &gt; <span>Profile Settings</span> to display the <span>Profile Settings</span> dialog box.</li> <li> Under <span>Your Timezone and Language</span>, use the drop-down list to select the required timezone. <br /><img src="/images/resources/images/infinity/timezone.png" alt="Change your time zone" />  </li> <li>Select <span>Save</span> to update the timezone.</li> </ol>

<p>To change the language:</p>

<ol> <li> Under <span>Your Timezone and Language</span>, use the drop-down list to select your preferred language: <ul> <li>English</li> <li>Español</li> <li>Português</li> </ul> </li> <li> Select <span>Save</span> to update the language. </li> </ol>

<h3>Changing your password</h3>

<p>Passwords require at least eight characters, including a mix of upper and lower case letters and at least one number and one special character. </p>

<Note>Underscores (&#95;) are not included as special characters.</Note>

<p>To change your password:</p>

<ol> <li> At the top right, select your <span>&lt;<var>username</var>&gt;</span> &gt; <span>Profile Settings</span> to display the <span>Profile Settings</span> dialog box. </li> <li> Under <span>Change Password</span>, select <span>Change</span>. <br /><img src="/images/resources/images/infinity/changepassword.png" alt="Change password" />  </li> <li> In the <span>New Password</span> window, enter the <span>Old Password</span>, <span>New Password</span> and <span>Confirm Password</span> fields. <br /><img src="/images/resources/images/infinity/newpassworddialog.png" alt="Newpassworddialog" />  <ul> <li>Old Password – the current user's existing password.</li> <li>New Password – the new password.</li> <li>Confirm Password – must match the <span>New Password</span>.</li> </ul> </li> <li> Select <span>Update</span> to accept the password. </li> </ol>
