<Note>Nothing can initiate communication with your devices using these IP addresses. For more information, see <a href="/connectivity/device-configuration#Understa" target="_blank">Configuring devices to initiate communication</a>.</Note>

<p>The following table shows the public IP address ranges and full list of IP addresses for Eseye's <a href="/connectivity/understanding-pops">egress PoPs</a>.</p>

<Tip>We highly recommend that you add all these IP address ranges to your ACL rules, as we sometimes re-route traffic between PoPs for resilience and better performance. For more information, see <a href="/connectivity/non-vpn-network-traffic" target="_blank">Routing non-VPN network traffic</a>. </Tip>
