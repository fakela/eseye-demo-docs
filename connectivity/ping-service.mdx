<p>Ping is a troubleshooting tool to test connectivity, network delay and packet loss between network devices. Internet Control Message Protocol (ICMP), which is one of the protocols of the TCP/IP suite, is used to send and receive ping messages.</p>

<p>The AnyNet Ping service is available as part of our connectivity managed service so IoT devices can check that they can connect to an Eseye PoP.</p>

<h2 id="AbouttheAnyNetPingservice1">About the AnyNet Ping service</h2>

<p>The AnyNet Ping service is provided by highly available, fully redundant servers in all our PoPs. The IP address for the AnyNet Ping service is the same in every PoP. After IoT devices are configured to use the service, they can access it no matter which mobile network they’re using, or if they switch to a different mobile network that connects to a different PoP.</p>

<h3>Benefits of using the AnyNet Ping service</h3>

<p>The ping servers are located as close to the mobile network interface as possible. This helps troubleshooting connectivity issues by reducing the number of network elements between the device and the ping server.</p>

<h3>Using the AnyNet Ping service</h3>
