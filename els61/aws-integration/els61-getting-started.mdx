<h2>Preparing the cloud overview</h2>

<p>In order to integrate your thing s with AWS, you need to perform the following steps:</p>

<ol> <li> Sign up for an Amazon Web Services (AWS) account, or log into an existing account: <a href="https://aws.amazon.com">https://aws.amazon.com</a> <br />For instructions, see: <a href="https://aws.amazon.com/premiumsupport/knowledge-center/create-and-activate-aws-account/" target="_blank">How do I create and activate a new AWS account?</a> </li> <li> Subscribe to <span>AnyNet Cellular Connectivity for AWS IoT</span>. <br />For more information, see<a href="#_Toc25599551">Subscribing to AnyNet Cellular Connectivity for AWS IoT</a>. </li> <li>  Within your AWS root account, create a mandatory dedicated AWS IAM user account for AnyNet IRIS. <br />For more information, see <a href="/els61/aws-integration/els61-iam-permissions/els61-create-ia-museraccount">Creating a dedicated IAM user account for AnyNet IRIS</a> . </li> <li> Install and configure AnyNet IRIS.  <br />For more information, see <a href="/els61/aws-integration/els61-installing-iris">Installing AnyNet IRIS</a> . </li> </ol>

<h2>Subscribing to AnyNet Cellular Connectivity for AWS IoT</h2>

<p>Use AWS Marketplace to subscribe to<span></span><span><span>AnyNet Cellular Connectivity for </span>AWS IoT</span>, which will enable you to connect your thing to AWS IoT <span>Core.</span></p>

<ol> <li> Log into your AWS account. </li> <li> Go to <a href="https://aws.amazon.com/marketplace/pp/B07MC96DYV" target="_blank">AnyNet Cellular Connectivity for AWS IoT</a>. <br />AnyNet Cellular Connectivity for AWS IoT opens in the AWS Marketplace. <br /><img src="/images/resources/images/marketplace3.png" alt="Marketplace3" />  </li> <li> Select <span>Continue to Subscribe</span>.<br />The service pricing options are displayed. </li> <li> Select <span>Subscribe</span>. <br />A Congratulations! You are now subscribed! message appears. </li> <li> Select <span>Set Up Your Account</span>. <br />The AnyNet Cellular Connectivity for AWS IoT Welcome page appears. <br /><img src="/images/resources/images/iris/iriswelcome.png" alt="Iriswelcome" />  <span><strong>Tip: </strong>Make a note of your Customer ID. Leave this window open to refer back to it when you set up the IAM user account, and also when you install and configure AnyNet IRIS.</span> </li> </ol>

<Note>If you previously subscribed to AnyNet Cellular Connectivity for AWS IoT and did not download the AnyNet IRIS executable, then you will need to return to the <span>AnyNet Cellular Connectivity for AWS IoT</span> Welcome page. For more information, see <a href="/els61/aws-integration/els61-view-welcome-page">Viewing the AnyNet IRIS Welcome page</a>.</Note>
