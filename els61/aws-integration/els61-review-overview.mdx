---
title: "Reviewing thing connectivity using AnyNet IRIS"
description: "You can use AnyNet IRIS to review:"
---

<p>You can use AnyNet IRIS to review: </p>

<ul> <li>The status of thing s across your IoT estate</li> <li>AnyNet IRIS configuration</li> <li>AnyNet IRIS version and software updates.</li> </ul>

<Accordion title="Viewing configuration settings">
 <p>Using the <span>Configuration</span> tab, you can view the settings you chose when you installed AnyNet IRIS.</p> <p> <img src="/images/resources/images/iris/configurationtab_325x138.png" alt="Configurationtab 325X138" /> </p> <p>To update the displayed email addresses:</p> <ol> <li> Using the left-hand menu, select <span>Notifications</span> to change the advisory and alert recipient email addresses. <br /> <img src="/images/resources/images/iris/irismenu_151x109.png" alt="Irismenu 151X109" />  </li> <li>Make your changes, then select <span>Next</span> to update the email addresses.</li> </ol> <p>To add more regions:</p> <ol> <li> Using the left-hand menu, select <span>Service Region</span>. <br />The current selected regions are highlighted in blue and have an Integration Completed status <img src="/images/resources/images/iris/success_ic.png" alt="Success Ic" />, for example: <br /> <img src="/images/resources/images/iris/selectedregions.png" alt="Selectedregions" />  </li> <li>Select one or more regions to add them to the list.</li> <li>Select <span>Complete</span> to update your choices.</li> </ol> <p>To delete a selected region:</p> <ol> <li> Select a region to deselect it. </li> <li>Select <span>Complete</span> to update your choices.</li> </ol> 
</Accordion>

<Accordion title="Viewing status settings">
 <p>Using the<span>Status</span> tab, you can view the following events across your IoT estate:</p> <ul> <li><span>Activated</span> – the device SIM has registered on the network and enabled sending and receiving data</li> <li><span>Unable to Deliver Certificate</span> – the Eseye identification and security certificate failed to reach the device after multiple attempts</li> <li><span>Suspended</span> – the SIM can no longer send or receive data</li> <li><span>Unable to Suspend</span> – the SIM is in an intermediate state and cannot be suspended until it changes state</li> </ul> <p> <img src="/images/resources/images/iris/status_302x101.png" alt="Status 302X101" /> </p> <p>You can filter the displayed events by <span>Date</span>, <span>Thing</span>, and <span>Region</span>.</p> <p>To filter by Date:</p> <ol> <li> Select the<span> Date</span> drop-down list. <br />Three options are displayed. <br /> <img src="/images/resources/images/iris/datefilter_302x221.png" alt="Datefilter 302X221" />  </li> <li> Select any of the following: <ul> <li><span>Last 7 Days</span> to display the previous week's Status, up to and including today</li> <li><span>Last 30 days</span> to display the previous month's Status, up to and including today</li> <li><span>Custom Range</span> to select your own dates</li> </ul> </li> <li>Select <span>Update</span> to refresh the returned results.</li> </ol> <p>To filter by thing:</p> <ol> <li> Select the <span>Thing</span> drop-down list. <br /> <img src="/images/resources/images/iris/thingfilter_302x62.png" alt="Thingfilter 302X62" />  </li> <li>Type the thing name.</li> <li>Select <span>Update</span> to refresh the returned results.</li> </ol> <p>You can filter the results by all regions or one specific region.</p> <p>To filter by region:</p> <ol> <li> Select the <span>Region</span> drop-down list. <br /> <img src="/images/resources/images/iris/regionfilter_302x91.png" alt="Regionfilter 302X91" />  </li> <li>Select a specific region from the drop-down list, or select <span>All</span> to display all regions.</li> <li>Select <span>Update</span> to refresh the returned results.</li> </ol> 
</Accordion>

<Accordion title="Viewing version and software information">
 <p>Using the <span>Updates</span> tab, you can view AnyNet IRIS version and software update information.</p> <p><img src="/images/resources/images/iris/updates.png" alt="Updates" /> </p> 
</Accordion>
