You must have already subscribed to the selected index number, or the command will return an error. </td> <td> OK <br />  <br />+EMQSUBCLOSE:(0-7),&lt;status&gt; <br />where: <ul> <li><span>(0-7)</span> is the subscribe index number you selected from the range</li> <li><span>status</span> is either:<ul><li><span>0</span> – subscription cancelled successfully</li><li><span>-1</span> – broker returned an <span>unsubnack</span></li><li><span>-2</span> – no topic was registered for the given index</li></ul></li> </ul> <br />or <br /><span>ERROR</span> – the command failed <br />  <br />+EMQSUBCLOSE:(0-7),&lt;status&gt; </td> </tr> </tbody> </table>
