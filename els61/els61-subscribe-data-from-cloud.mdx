<div> <h3>Before you begin</h3> <p>Ensure your thing can send information to the cloud. </p> <p>For more information, see <a href="/els61/els61-publish-data-to-cloud">Sending data from your thing to the cloud</a> .</p> </div>

<p>To test that AWS can publish information to your thing:</p>

<ol> <li> Create two subscribe topics in the Cinterion® ELS61 Wireless Module. <ol> <li> Using a terminal emulator, type:  <br /><span>at+emqsubopen=0,"SubscribeFromCloud0"</span>&lt;return&gt; <br /> <br /><span>at+emqsubopen=1,"SubscribeFromCloud1"</span>&lt;return&gt; <br /><img src="/images/resources/images/emqsubopen.png" alt="Emqsubopen" />  </li> <li> Check that the first two index numbers are assigned a topic each. Type: <br /><span>at+emqsubopen?</span>&lt;return&gt; <br />A list of index numbers and their assigned topics appears. <br /><img src="/images/resources/images/emqsubopen2.png" alt="Emqsubopen2" />  </li> </ol> </li> <li> Use AWS to publish a message to each topic. <ol> <li> Using the AWS IoT <span>MQTT test client</span>, select the <span>Publish to a topic</span> tab. <br /><img src="/images/resources/images/aws_publish1.png" alt="Aws Publish1" />  </li> <li> In the <span>Topic name</span> box, type: <br /><span>SubscribeFromCloud0/&lt;<i>AWS thingName</i>&gt;</span>  </li> <li> In the coding window, replace  <br /><span>Hello from AWS IoT console</span> with  <br /><span>Turn heating on</span>  </li> <li> Select <span>Publish</span>. <br /><img src="/images/resources/images/aws_publish2.png" alt="Aws Publish2" />  </li> <li> In the <span>Publish</span> box, type: <br /><span>SubscribeFromCloud1/<i>AWS thingName</i></span>  </li> <li> In the coding window, replace  <br /><span>Turn heating on</span> with  <br /><span>Heat for 1 hour</span>  </li> <li> Select <span>Publish</span>. <br />View the AWS messages in the terminal emulator, in the following format: <br />+EMQ: &lt;<i>indexnumber</i>&gt;,&lt;<i>messagelength</i>&gt; <br />&#123; <br />"message": "&lt;<i>messagetext</i>&gt;" <br />&#125; <br /><img src="/images/resources/images/messagesreceived.png" alt="Messagesreceived" />  </li> </ol> </li> </ol>

<Tip>If you can see the messages in the terminal emulator, then AWS can successfully publish data to your thing through the Cinterion® ELS61 Wireless Module.</Tip>
