---

<p>Before you can send and receive data between your thing and the cloud, you need to: </p>

<ol> <li> Prepare the cloud.  <br />For more information, see<a href="/iris/getting-started">Getting Started: preparing the cloud</a>. </li> <li> Install AnyNet IRIS. <br />For more information, see<a href="/iris/installing-iris">Installing AnyNet IRIS</a>. </li> <li> Create your thing in AWS IoT Console.  <br />For more information, see<a href="/iris/creating-thing-in-aws#_Toc25599552">Creating a thing in AWS IoT Core</a>. </li> <li>  Connect the Cinterion® ELS61 Wireless Module to a cellular network.  <br />For more information, see <a href="/els61/els61-establishing-cellular-connection#Activati">Establishing a cellular connection on the LTE Cat.1-xx click</a> . </li> <li> Provision the Cinterion® ELS61 Wireless Module. <br />For more information, see <a href="/els61/els61-provisioning#Provisio">Where to next?</a> . </li> <li> Use a terminal emulator to send commands to the Cinterion® ELS61 Wireless Module.  <br />For information about setting up the terminal emulator, see <a href="/els61/els61-rs232-serial-connection">Connecting to the LTE Cat.1-xx click</a>. </li> <li> Test that you can send data from your thing to the cloud.  <br />For more information, see <a href="/els61/els61-publish-data-to-cloud">Sending data from your thing to the cloud</a> . </li> <li> Test that you can send data from the cloud to your thing. <br />For more information, see <a href="/els61/els61-subscribe-data-from-cloud">Sending data from the cloud to your thing</a> . </li> <li> Your system is now ready. Use AT commands to configure how data is sent and received between the cloud and the modem. <br />For more information, see <a href="/els61/at-commands/els61-at-commands">General AT Commands for the Cinterion® ELS61 Wireless Module</a> . </li> </ol>
